### エレベーターピッチ

記憶を失った少年が冒険者ギルドの養成学校「クロノス・アカデミー」に入学する。
クラスメイトとの学園生活、ダンジョン実習、そして徐々に明らかになる自分の正体——
彼の中には「魔王の器」としての力が封じられていた。

その事実を知った優等生の先輩は、世界を守るためにレイを殺そうとする。
幼馴染たちはレイを守るために立ちはだかる。
そして信頼していた担任教師は、魔王の力を我が物にしようとする悪の組織の幹部だった。

**「誰を信じ、何を選ぶか」——あなたの選択が7つの結末を決める。**
マーダーミステリー的な情報収集と選択が、取り返しのつかない結末へ導く学園ダークファンタジーRPG。

**Unity MCP × Claude Code × 生成AIをフル活用し、
2〜3人 × 3週間で「これ本当にこの期間で作ったの？」と驚かせるクオリティを目指す。**

## 4. 世界観設定

### 4.1 世界「ミッドガルディア」

魔物が蔓延る世界。人類は「冒険者ギルド」を組織し、魔物と戦いながら生活圏を守っている。
冒険者はこの世界で最も尊敬される職業であり、各地にギルド支部と養成学校が存在する。

かつてこの世界には「魔王」が君臨し、人類を滅亡の淵に追い込んだ。
300年前の「聖戦」で英雄たちが魔王を封印し、平和が訪れた。
しかし、魔王は完全に滅びたわけではない。
魔王復活を目論む闇の組織「黒蝕の教団（エクリプス）」が暗躍している。

### 4.2 クロノス・アカデミー

物語の舞台となる冒険者ギルド最高峰の養成学校。王都の外れの丘の上に建ち、街には歩いて行ける距離。

**表の顔:** 優秀な冒険者を輩出する名門校。実技（ダンジョン実習）と座学（魔法理論、魔物学、戦術論）のカリキュラム。卒業すればAランク冒険者としてギルドに入団できる。

**裏の顔:** 実はクロノス・アカデミーは300年前の聖戦で建てられた「魔王の封印装置」。
学園の地下深くに魔王の封印核があり、在校生の魔力が封印を維持するエネルギー源として
機能している。学園に在籍する生徒が多いほど封印は安定する。
歴代の学園長だけがこの秘密を知り、封印を守り続けてきた。

### 4.3 主要ロケーション

| エリア | 説明 | ゲーム進行 |
|--------|------|------------|
| **アカデミー本館** | 教室、食堂、図書館、訓練場。学園生活の中心 | 全編：拠点・日常パート |
| **学生寮** | キャラとの交流イベントが発生する。夜の探索も可能 | 全編：ソウルリンクイベント |
| **実習ダンジョン「翠緑の迷宮」** | 1年生向けの訓練用ダンジョン。安全なはずだが… | 序盤：チュートリアル |
| **禁域「深層回廊」** | 学園地下に広がる立入禁止区域。封印核に繋がる | 中盤〜終盤：核心ダンジョン |
| **封印の間** | 魔王の封印核がある最深部。最終決戦の地 | 終盤：ラストダンジョン |

### 5.2 あらすじ

**第1章「入学」— 1年前期（1〜12週）**

記憶を失った少年レイが、クロノス・アカデミーに入学する。
幼馴染のユナとミオが同じクラスにいて安心するレイ。
しかし実技試験で同級生のカイトに完敗し、自分の力のなさを痛感する。
2年生の先輩シオンがレイの訓練に付き合い、パーティに加入。
ミオの友人メル、訓練場で助けたリン、シオンに憧れるセツナ、情報屋のジン、
格闘家のレンジが次々と仲間に加わる。
初めてのダンジョン実習「翠緑の迷宮」で、レイの中に眠る謎の力が一瞬だけ覚醒。
黒い炎が敵を一掃する。カルロスはそれを目撃し、意味深な笑みを浮かべる。

**キャラ加入タイムライン（1章）:**
- 1週: レイ・ユナ・ミオ（初期メンバー）
- 2週: シオン（先輩として指導）
- 3週: メル（ミオの友人紹介）
- 5週: リン（訓練場で助ける）
- 6週: セツナ（シオンへの恩）
- 8週: ジン（裏通りで情報を持ちかける）
- 11週: レンジ（カイトとの決闘後）
- ※カイトは1章では加入せず（ライバルポジション）。ヴェインは2章5週で加入

**★選択ポイント:** 覚醒後、カルロスに相談するか / ユナに相談するか / 隠すか
→ 信頼ポイント・情報フラグに影響

**第2章「異変」— 1年後期（13〜24週）**

学園生活に馴染んでいくレイ。カイトとも切磋琢磨する仲に。
しかし学園では「行方不明の生徒」の噂が広がっていた。
禁域への立入禁止令が強化される中、レイは夜中に無意識で禁域に引き寄せられる。
シオンはレイの異常を察知し、独自に調査を開始。
2章後半でカルロスの正体が発覚し、代理教師に交代。
シオンは薬と封印の二段構えで操られ、闇落ちする（「テストに来なかった」で異変に気づく）。
2章11週でシオンがパーティから離脱。

**★選択ポイント:** 夜の禁域探索で誰を連れていくか / 何を調べるか
→ 得られる情報が変わり、終盤の選択肢に影響

**冬休み（3週間）:** 帰省+自由行動+強制ストーリー

**第3章「断罪」— 2年前期（25〜36週）**

シオンが独自調査の結果、レイが「魔王の器」であるという真実に辿り着く。
同時に学園が封印装置であること、カルロスの正体が判明する。
シオンは苦悩の末、世界のためにレイを殺す決意をする。

3章12週: シオン戦（第1形態 Lv55 光属性 → 第2形態 Lv75 闇属性の連戦、回復なし）。
END3ルートのみ第3形態（Lv90、壊れた裁定者）が存在。

**第4章「最終決戦」— 2年後期（37〜48週）**

3章のシオン戦の結果に応じて6ルートに分岐。
全ルート共通でカルロス（Lv100）との最終決戦がある。
END5ルートではシオンを救出し、5人パーティでカルロスに挑む。

裏では担任カルロスが全てを見ていた。彼は黒蝕の教団（エクリプス）の教主モルドレッドの部下であり、
レイの中の魔王の力を覚醒させ、それを奪い取ることが真の目的。
シオンとレイを戦わせることすら、カルロスの計画の一部だった。

### 5.3 ストーリーの構造 — 三重構造

```
プレイヤーの認識の変化:

序盤:  学園長 → 怪しい     担任カルロス → 頼れる味方    シオン → 優しい先輩
       ｜                  ｜                          ｜
中盤:  学園長 → 黒幕!?     担任カルロス → 助けてくれる   シオン → 何か隠してる
       ｜                  ｜                          ｜
転換1: 学園長 → 実は守護者  担任カルロス → まだ味方…?    シオン → レイを殺そうと!?
       ｜                  ｜                          ｜
転換2: 学園長 → 真の味方    担任カルロス → 教団の幹部!!   シオン → 正義ゆえの敵
```

この構造により、プレイヤーは3回騙される。
シオンの敵対は「悪」ではなく「正義」であり、プレイヤーの感情を大きく揺さぶる。

#### 各エンディング詳細

**END1: 裏ハッピー「平穏な日常」**

- **条件:** シオン第1形態戦でシオンが勝利（レイHP=0 かつ幼馴染3人全員気絶）

- **テーマ:** シオンの判断は正しかったのか

**END3: 壊れた正義「シオンの崩壊」**
- **条件:** シオン第2形態戦でシオンが勝利（パーティ全滅、侵蝕90%未満）
- **内容:** カルロスに操られた闇の力で4人を殺害したシオン。操りが解けた後、精神崩壊
- **4章遷移:** Nightmare以外ではグレイヴ介入+ユナの聖魔法で蘇生→4章でシオン第3形態（Lv90）戦→カルロス戦
- **テーマ:** 正義が暴走した時、最も壊れるのは正義を信じた者自身

**END4: ノーマルエンド / メリーバッドエンド「Pyrrhic Victory」**
- **条件:** シオン第2形態戦でレイたちが勝利。救出条件未達かつEND4.5の隠し条件も未達
- **内容:** 操られたシオンを倒すが、カルロスがシオンの心臓を潰して殺害。4章でカルロスを追い詰めて倒すが、シオンの命は戻らない

**END4.5: 最後に届いた光「グレイヴの約束」**

- **テーマ:** 届かなかった手、それでも伸ばし続けた光

**END5: 表ハッピーエンド「夜明けの学園」**
- **条件:** 情報フラグ8/8 + ユナ/ミオ/カイト全員信頼80+ + シオンSL+50以上 + 第2形態戦で「シオンの名を呼ぶ」使用（シオンHP50%以下で解放）→ シオン救出 → 5人でカルロス撃破

- **テーマ:** 最も困難だが、全員が生き残る唯一の道

**END6: 全滅エンド「深淵の勝者」**
- **条件:** シオン第2形態戦で全滅 / 侵蝕90%以上 / 4章カルロス戦で全滅（全難易度共通でボス全滅→END6）

- **テーマ:** 漁夫の利。全員が争っている間に、真の悪が勝つ

### 5.5 マダミス風分岐システム

#### 学園生活パートでの情報収集（蓄積フェーズ）

プレイヤーの行動が「情報フラグ」と「信頼ポイント」として蓄積される。

```

│  【情報フラグ】（知っているか知らないか）             │

│  □ 学園が封印装置であることを知っている              │
│  □ カルロスの正体に気づいている                      │
│  □ シオンがレイを調査していることを知っている         │
│  □ 黒蝕の教団の存在を知っている                      │
│  □ ユナがレイの過去を隠していることを知っている       │
│  □ 行方不明の生徒の真相を知っている                  │
│  □ 封印核の弱点を知っている                          │
│                                                    │
│  【信頼ポイント】（キャラごとに 0〜100）              │
│  ユナ:  ██████████░░ 80                            │
│  ミオ:  ████████░░░░ 65                            │
│  カイト: ███████░░░░░ 55                            │
│  シオン: █████░░░░░░░ 40                            │
│                                                    │
│  【侵蝕レベル】（0〜100）                            │
│  レイ:  ██░░░░░░░░░░ 15                            │
│                                                    │
│  → これらの値が最終決戦の選択肢を決定する            │
└──────────────────────────────────────────────────┘
```

**情報の集め方（マダミス要素）:**
- **図書館で調査:** 古文書から学園の歴史・封印の情報を得る
- **NPC会話:** 他の生徒・教師から噂・断片的情報を収集
- **夜の探索:** 禁域に侵入し、直接的な証拠を発見（リスクあり：侵蝕上昇）
- **カルロスとの会話:** 信頼すると情報をくれるが、実は誘導されている
- **シオンの行動を追う:** シオンの調査を観察すると相互理解が深まる

**信頼の上げ方:**
- 戦闘中の連携（かばう、回復する）
- 会話での選択肢（各キャラの価値観に合った答え）
- 個別イベント（悩み相談、秘密の共有）
- ★ シオンの信頼を上げるのが最も難しい（彼はレイを殺そうとしているので距離を取る）

#### 最終決戦での選択（決断フェーズ）

蓄積したフラグ・信頼度に応じて、決戦中に選べるコマンドが変わる。

```
【シオン第1形態戦中の選択肢】
├── [常時] 「全力で戦う」        → 通常戦闘
├── [常時] 「防御に徹する」      → 被ダメ軽減、シオンの攻撃を凌ぐ
├── [信頼50+] 「シオン、話を聞け！」→ シオンの攻撃力低下
└── [情報6+] 「黒幕はカルロスだ！」 → シオンが一瞬動揺（大チャンス）

【シオン第2形態戦中の選択肢】（第1形態でシオンが敗北した場合のみ）
├── [常時] 「倒すしかない」      → 通常戦闘（END4 or END6へ）
├── [シオン信頼80+] 「シオン先輩の名前を呼ぶ」→ 闇の制御を一時解除
├── [全員信頼70+] 「みんなの力を合わせる」    → 合体浄化技解放
└── [情報8+ & 全信頼80+] 「完全浄化」         → シオン救出（END5へ）
```


**基本ルール:**
- ATB方式。各キャラクターのATBゲージが100%になるとコマンド入力可能
- ゲージ蓄積速度 = AGI x 0.1 / フレーム
- バトル開始時は全員ゲージ0

**ステータス:**
HP / MP / ATK / DEF / INT / RES / AGI / DEX / LUK / Lv / EXP

**コマンド:** 攻撃 / スキル / アイテム / 防御 / 逃走 / デュアルアーツ / チェンジ / 入れ替え

**ダメージ計算:**
- 物理: (ATK x 2 - DEF) x スキル倍率 x 属性相性 x クリティカル x ブレイク x 乱数
- 魔法: (INT x 2 - RES) x スキル倍率 x 属性相性 x クリティカル x ブレイク x 乱数
- クリティカル率 = DEX / 200。クリティカル倍率 = 1.5倍

**ブレイクシステム:**
- 敵にはHPとは別に「ブレイクゲージ」がある（MAX100）
- 弱点攻撃: +25 / 通常攻撃: +5
- ゲージMAXで「BREAK状態」2ターン（被ダメージ2倍、状態異常耐性半減）

**属性相性（7属性循環型）:**
- 火→氷→風→地→火（循環弱点、1.5倍）
- 光⇔闇（相互弱点、1.5倍）
- 同属性耐性 0.5倍
- ボスは自属性攻撃を吸収

**隊列システム:**
- 前列/後列。コマンド「チェンジ」でパーティ全体の前後列入れ替え
- 後列: 物理与ダメ-20%、物理被ダメ-20%。魔法は隊列無視

**リンクゲージ + デュアルアーツ:**
- パーティ共有ゲージ（0〜100%）。与ダメ/被ダメ/バフ/ブレイク達成等で蓄積
- デュアルアーツ発動で-50%消費。全55種の2人協力技

**バトル速度:** 通常(1.0x) / 高速(1.5x) / 超高速(2.0x) + 自動戦闘

**状態異常:** 毒 / 暗闇 / 沈黙 / スロウ / ストップ / ヘイスト / リジェネ等。ブレイク中は状態異常耐性半減

**入れ替え（FF10式）:** バトル中にメンバーと控えを1対1交代。交代後ATBゲージ0から

**EXP配分:** 出場ターン比率で配分。控えメンバーは総EXPの50%。ブレイク達成+10%、デュアルアーツ使用+5%、全員生存クリア+20%

**バトルメンバー:** 最大4人（レイ固定）+ 控え最大10人

**ボス戦特殊コマンド「選択」:**
- シオン戦・カルロス戦で出現する特殊コマンド
- 蓄積した情報フラグ・信頼ポイントに応じて選択肢が増減
- 戦闘の流れやエンディングに直結する重要な要素

> 詳細は `docs/specs/battle-system.md` 参照

### 6.3 ソウルリンクシステム

クラスメイトとの絆が戦闘に直結する学園RPGならではのシステム。
マルチエンディングの分岐条件にもなる重要システム。

**絆レベルの上げ方:**
- 学園生活パートでの会話選択肢
- 戦闘中の連携行動
- キャンプイベント（ダンジョン内の休憩ポイント）
- ストーリーイベントの選択
- ★ シオンとの絆は第2章まで限定。第3章で敵対後は上がらない

**絆レベルによる効果:**
| レベル | 効果 |
|--------|------|
| Lv1 | 連携攻撃「デュアルアーツ」が使用可能 |
| Lv2 | 戦闘中にかばう・回復を自動で行うことがある |
| Lv3 | 固有の「ソウルリンク必殺技」が解放 |
| Lv MAX | 専用ストーリーイベント解放 + 最強連携技 + ★分岐条件解放 |

**デュアルアーツ（全55種）の例:**
- レイ × ユナ：「深淵浄化斬」（物理+聖魔法の複合大ダメージ + 侵蝕ゲージ軽減）
- レイ × カイト：「双剣乱舞」（連続物理攻撃 + 高ブレイクダメージ）
- レイ × ミオ：「混沌の炎」（レイの闇 + ミオの暴走魔法 = ランダムだが超火力）
- ユナ × ミオ：「癒しの爆風」（味方全体回復。ただしミオの暴走で回復量にムラ）
- ★ レイ × シオン：「星影双斬」→ END5限定で「約束の光」に進化（カルロス戦フェーズ3で使用可能）
- 全10キャラの組み合わせ（10C2 = 45種 + レイ固有10種 = 55種）

### 6.4 侵蝕システム

レイの固有能力「深淵覚醒」を使うと一時的に超強化されるが、「侵蝕ゲージ」が蓄積。
魔王の力に飲まれるか、制御するかのリスク管理がゲームの核。
侵蝕レベルはエンディング分岐にも影響する。

- 侵蝕ゲージ 0〜30%: 通常状態
- 侵蝕ゲージ 30〜60%: NPCとの会話が変化。暗い選択肢が増える。攻撃力UP
- 侵蝕ゲージ 60〜90%: 深淵覚醒がより強力に。しかし暴走して味方を攻撃するリスク
- 侵蝕ゲージ 90%〜: END6（全滅エンド）のフラグが立ちやすくなる

**侵蝕の回復手段:**
- ユナの「浄化の光」（絆レベルに依存。Lv3で大幅回復）
- 聖水・星の雫等のアイテム
- 学園生活パートでの仲間との交流（自然回復）

### 6.5 成長システム

**スキルツリー（SP制）:**

- 隠しスキル12個: 特定条件で解放されるツリー外の特殊スキル
- 専用武器スキル12個: 専用武器入手で自動習得（SP不要）

### 6.9 セーブ・難易度・NG+

**セーブシステム（22枠）:**

- オートセーブ: 時間帯切替・ポータル到達・章開始・重要イベント前

**4段階の難易度:** Easy / Normal / Hard / Nightmare

- **リセット:** レベル・通常装備・消耗品・ゴールド・ストーリーフラグ・侵蝕値・SL・授業ノート・教師信頼度

**プレイ時間目安:**

|--------|----------|

| Normal | 35〜50時間 |
| Hard | 50〜70時間 |

```
│                    ↕ Events                        │
│  ┌───────────────────────────────────────────────┐ │
│  │              Game Logic Layer                  │ │
│  │  (Pure C# — MonoBehaviour非依存)              │ │
│  │                                               │ │
│  │  ATBSystem / DamageCalculator / BreakSystem   │ │
│  │  SoulLinkManager / ErosionSystem              │ │

│  │  ★ BranchManager / FlagSystem / TrustSystem  │ │
│  └───────────────────────┬───────────────────────┘ │
│                          ↕                         │
│  ┌───────────────────────────────────────────────┐ │
│  │              Data Layer                        │ │

│  └───────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────┘
```

**設計原則:**
- **ロジックとUnity依存を分離**: ATB計算、ダメージ計算、分岐判定等はPure C#で実装 → **Edit Modeテストで高速テスト可能**
- **データ駆動**: キャラ・敵・スキル・分岐条件等はScriptableObjectまたはJSONで管理
- **イベント駆動**: システム間の通信はC# eventで疎結合に
- **インターフェース活用**: テスト時にモック差し替え可能

### 11.4 分岐テストの例

```csharp
// EndingConditionTests.cs の想定
[Test]
public void END1_WhenPlayerRefusesAcademy_ReturnsHiddenHappy()
{
    var flags = new FlagSystem();
    flags.Set("academy_refused", true);
    var result = EndingResolver.Resolve(flags, new TrustSystem());
    Assert.AreEqual(EndingType.HiddenHappy, result);
}

[Test]
public void END5_RequiresAllTrustMaxAndFullInfo()
{
    var flags = new FlagSystem();
    flags.SetAllInfoFlags(true); // 8/8 info flags
    var trust = new TrustSystem();
    trust.SetAll(80); // All trust >= 80
    var result = EndingResolver.Resolve(flags, trust);
    Assert.Contains(EndingType.TrueHappy, result.AvailableEndings);
}

[Test]
public void END6_WhenAllPartyDefeated_ReturnsTotalDefeat()
{
    var battleState = new BattleState();
    battleState.SetAllPartyHP(0);
    var result = EndingResolver.ResolveBattle(battleState);
    Assert.AreEqual(EndingType.TotalDefeat, result);
}
```
