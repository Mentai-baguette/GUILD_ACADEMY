### エレベーターピッチ

記憶を失った少年が冒険者ギルドの養成学校「クロノス・アカデミー」に入学する。
クラスメイトとの学園生活、ダンジョン実習、そして徐々に明らかになる自分の正体——
彼の中には「魔王の器」としての力が封じられていた。

その事実を知った優等生の先輩は、世界を守るためにレイを殺そうとする。
幼馴染たちはレイを守るために立ちはだかる。
そして信頼していた担任教師は、魔王の力を我が物にしようとする悪の組織の幹部だった。

**「誰を信じ、何を選ぶか」——あなたの選択が6つの結末を決める。**
マーダーミステリー的な情報収集と選択が、取り返しのつかない結末へ導く学園ダークファンタジーRPG。

**Unity MCP × Claude Code × 生成AIをフル活用し、
2〜3人 × 3週間で「これ本当にこの期間で作ったの？」と驚かせるクオリティを目指す。**

## 4. 世界観設定

### 4.1 世界「ミッドガルディア」

魔物が蔓延る世界。人類は「冒険者ギルド」を組織し、魔物と戦いながら生活圏を守っている。
冒険者はこの世界で最も尊敬される職業であり、各地にギルド支部と養成学校が存在する。

かつてこの世界には「魔王」が君臨し、人類を滅亡の淵に追い込んだ。
300年前の「聖戦」で英雄たちが魔王を封印し、平和が訪れた。
しかし、魔王は完全に滅びたわけではない。
魔王復活を目論む闇の組織「黒蝕の教団（エクリプス）」が暗躍している。

### 4.2 クロノス・アカデミー

物語の舞台となる冒険者ギルド最高峰の養成学校。

**表の顔:** 優秀な冒険者を輩出する名門校。実技（ダンジョン実習）と座学（魔法理論、魔物学、戦術論）のカリキュラム。卒業すればAランク冒険者としてギルドに入団できる。

**裏の顔:** 実はクロノス・アカデミーは300年前の聖戦で建てられた「魔王の封印装置」。
学園の地下深くに魔王の封印核があり、在校生の魔力が封印を維持するエネルギー源として
機能している。学園に在籍する生徒が多いほど封印は安定する。
歴代の学園長だけがこの秘密を知り、封印を守り続けてきた。

### 4.3 主要ロケーション

## 5. ストーリー

### 5.1 あらすじ

**第1章「入学」（ハッカソン版 必須）**

記憶を失った少年レイが、身元引受人のギルド職員に連れられクロノス・アカデミーに入学する。
新入生クラスに配属され、担任のカルロス先生に温かく迎えられる。
幼馴染のユナとミオが同じクラスにいて安心するレイ。
しかし実技試験で同級生のカイトに完敗し、自分の力のなさを痛感する。
初めてのダンジョン実習「翠緑の迷宮」で、レイの中に眠る謎の力が一瞬だけ覚醒。
黒い炎が敵を一掃する。カルロスはそれを目撃し、意味深な笑みを浮かべる。

**★選択ポイント:** 覚醒後、カルロスに相談するか / ユナに相談するか / 隠すか
→ 信頼ポイント・情報フラグに影響

**第2章「異変」（ハッカソン版 可能なら）**

学園生活に馴染んでいくレイ。カイトとも切磋琢磨する仲に。
優しい先輩シオンがレイたちの面倒を見てくれるようになり、5人での実習が始まる。
しかし学園では「行方不明の生徒」の噂が広がっていた。
禁域への立入禁止令が強化される中、レイは夜中に無意識で禁域に引き寄せられる。

シオンはレイの異常を察知し、独自に調査を開始。
学園長グレイヴが不穏な動きをしていることにも気づく。

**★選択ポイント:** 夜の禁域探索で誰を連れていくか / 何を調べるか
→ 得られる情報が変わり、終盤の選択肢に影響

**第3章「断罪」（完全版）**

シオンが独自調査の結果、レイが「魔王の器」であるという真実に辿り着く。
同時に学園が封印装置であること、レイの存在が封印を不安定にしていることを知る。

シオンは苦悩する。レイは仲間だ。しかし魔王が復活すれば世界は滅ぶ。
「レイを殺せば、魔王の復活は永遠に阻止できる」——シオンはその結論に至る。

シオンがレイに宣告する。
*「お前は……魔王の器だ。お前が生きている限り、世界は危険に晒される。」*
*「俺はお前を殺さなければならない。」*

幼馴染のユナ・ミオ・カイトはレイを守ろうとする。

**★選択ポイント:** シオンとの対話（説得する / 受け入れる / 逃げる）
→ 第一形態戦の展開・エンディング分岐に直結

**第4章「深淵」（完全版）**

封印の間での最終決戦。ここでの戦闘結果と、これまでの選択の蓄積が6つの結末を分ける。

裏では担任カルロスが全てを見ていた。彼は黒蝕の教団（エクリプス）の幹部。
レイの中の魔王の力を覚醒させ、それを奪い取ることが真の目的。
シオンとレイを戦わせることすら、カルロスの計画の一部だった。

### 5.2 ストーリーの構造 — 三重構造

```
プレイヤーの認識の変化:

序盤:  学園長 → 怪しい     担任カルロス → 頼れる味方    シオン → 優しい先輩
       ｜                  ｜                          ｜
中盤:  学園長 → 黒幕!?     担任カルロス → 助けてくれる   シオン → 何か隠してる
       ｜                  ｜                          ｜
転換1: 学園長 → 実は守護者  担任カルロス → まだ味方…?    シオン → レイを殺そうと!?
       ｜                  ｜                          ｜
転換2: 学園長 → 真の味方    担任カルロス → 教団の幹部!!   シオン → 正義ゆえの敵
```

この構造により、プレイヤーは3回騙される。
シオンの敵対は「悪」ではなく「正義」であり、プレイヤーの感情を大きく揺さぶる。

- **内容:** レイは記憶のないまま、ユナ・ミオと平穏な田舎暮らしを続ける。魔王の力が覚醒することもなく、シオンと出会うこともなく、誰も死なない。しかしエンドロール後、カルロスが別の「器」を探し始める不穏な一枚絵が映る
- **テーマ:** 無知は幸福か？平和だが何も解決していない

**END 2: トゥルーエンド「残された意識」**
- **条件:** シオン第一形態戦でシオンが勝利（レイが死亡、幼馴染3人は気絶）
- **内容:** シオンの光の力がレイの肉体を貫く。魔王の器は破壊され、封印は完全に安定する。カルロスの計画は頓挫。しかしレイの意識は消えず——封印核の中に、レイの魂が「声」として残る。学園を守る見えない守護者となったレイ。エンドロール後、ユナだけがレイの声を聞く
- **テーマ:** シオンの判断は正しかった。だがレイは完全には消えなかった。最も切なく、最も美しい結末

**END 3: シオンルート「壊れた正義」**
- **条件:** シオン第二形態戦でシオンが勝利（4人を殺害）
- **内容:** カルロスに操られた闇の力でレイと幼馴染3人を殺してしまったシオン。操りが解けた後、自分がやったことを理解し、精神が崩壊する。正義のために戦ったはずが、全てを失った。カルロスは目的を達せず去るが、シオンの心は二度と戻らない
- **テーマ:** 正義が暴走した時、最も壊れるのは正義を信じた者自身

**END 4: ノーマルエンド / メリーバッドエンド「Pyrrhic Victory」**
- **条件:** シオン第二形態戦でレイたちが勝利。ただしシオン救出条件を満たしていない
- **内容:** 操られたシオンを倒すが、カルロスがシオンの心臓を潰して殺す。「用済みだ」と。カルロスは撤退し、レイたちは生き残る。しかしシオンは死に、レイの侵蝕は進行し、カルロスは逃亡。勝ったはずなのに、不穏な空気だけが残る
- **テーマ:** 勝利したのに何も解決していない。真のメリーバッドエンド

**END 5: 表ハッピーエンド「夜明けの学園」**
- **条件:** 学園生活で十分な情報収集 + 全員の信頼レベルMAX + 第二形態戦で特殊コマンド「シオンの名を呼ぶ」解放 → シオンを闇から引きずり出す → カルロスと直接対決し撃破
- **内容:** 仲間との絆がシオンの闇を浄化。5人でカルロスに立ち向かい、撃破。カルロスの教団の情報をギルド本部に通報。封印は学園長グレイヴが再強化。レイの侵蝕は仲間の支えで制御可能に。全員で学園生活に戻る
- **テーマ:** 最も困難だが、全員が生き残る唯一の道

**END 6: 全滅エンド「深淵の勝者」**
- **条件:** 第二形態シオン戦で全員のHPが0になる / 特定の選択ミスの蓄積
- **内容:** 全員が倒れた隙にカルロスがレイから魔王の力を抽出。カルロス自身が魔王の力を取り込み、封印核を破壊。世界崩壊が始まる。最後の画面はカルロスが魔王の力に歪んだ笑みを浮かべるカットで暗転
- **テーマ:** 漁夫の利。全員が争っている間に、真の悪が勝つ

**情報の集め方（マダミス要素）:**
- **図書館で調査:** 古文書から学園の歴史・封印の情報を得る
- **NPC会話:** 他の生徒・教師から噂・断片的情報を収集
- **夜の探索:** 禁域に侵入し、直接的な証拠を発見（リスクあり：侵蝕上昇）
- **カルロスとの会話:** 信頼すると情報をくれるが、実は誘導されている
- **シオンの行動を追う:** シオンの調査を観察すると相互理解が深まる

**信頼の上げ方:**
- 戦闘中の連携（かばう、回復する）
- 会話での選択肢（各キャラの価値観に合った答え）
- 個別イベント（悩み相談、秘密の共有）
- ★ シオンの信頼を上げるのが最も難しい（彼はレイを殺そうとしているので距離を取る）

#### 最終決戦での選択（決断フェーズ）

蓄積したフラグ・信頼度に応じて、決戦中に選べるコマンドが変わる。

```
【シオン第一形態戦中の選択肢】
├── [常時] 「全力で戦う」        → 通常戦闘
├── [常時] 「防御に徹する」      → 被ダメ軽減、シオンの攻撃を凌ぐ
├── [信頼50+] 「シオン、話を聞け！」→ シオンの攻撃力低下
└── [情報6+] 「黒幕はカルロスだ！」 → シオンが一瞬動揺（大チャンス）

【シオン第二形態戦中の選択肢】（第一形態でシオンが敗北した場合のみ）
├── [常時] 「倒すしかない」      → 通常戦闘（END4 or END6へ）
├── [シオン信頼80+] 「シオン先輩の名前を呼ぶ」→ 闇の制御を一時解除
├── [全員信頼70+] 「みんなの力を合わせる」    → 合体浄化技解放
└── [情報8+ & 全信頼80+] 「完全浄化」         → シオン救出（END5へ）
```

### 5.5 主要キャラクター

- **年齢:** 16歳
- **役割:** 記憶を失った新入生。正体は「魔王の器」

- **戦闘タイプ:** 物理アタッカー（高火力・高速）
- **固有能力:** 深淵覚醒（魔王の力を一時的に解放。攻撃力・素早さが大幅UP + 闇属性追加ダメージ。ただし「侵蝕ゲージ」が蓄積）
- **ストーリーアーク:** 自分が何者かわからない不安 → 魔王の器だと知る恐怖 → シオンに殺されかける → 仲間の力で闇を受け入れ、自分の力にする（エンドによる）

#### ユナ（幼馴染ヒロイン）

- **役割:** レイとユナの幼馴染。ムードメーカー。レイを守る側

- **戦闘タイプ:** 魔法アタッカー（高火力・命中率にムラあり）

- **年齢:** 16歳
- **役割:** 実技トップの天才剣士。最初はレイを見下すが、徐々に認め合う。レイを守る側
- **武器:** 大剣（騎士家門の家宝）

- **戦闘タイプ:** 物理アタッカー / ブレイク特化

#### シオン（優しい先輩 → 正義の敵対者）
- **年齢:** 18歳
- **役割:** 2年生の優等生。レイたちのメンター。レイが魔王の器だと知り、世界のためにレイを殺す決意をする
- **武器:** 双剣

#### カルロス（担任教師 → 教団の幹部）
- **年齢:** 32歳
- **役割:** 新入生クラスの担任。穏やかで生徒想いに見える。実は「黒蝕の教団（エクリプス）」の幹部

- **戦闘タイプ:** ボス敵（END5のラスボス。高い知性で戦う策略型ボス）

- **ボス戦パターン:**
  - 第一フェーズ: 禁術魔法（広範囲攻撃 + デバフ + 回復阻害）
  - 第二フェーズ: シオンを操り人形にして2対5の戦い

- **役割:** アカデミーの学園長。冷徹に見えるが、300年封印を守り続けてきた守護者

### 6.3 ソウルリンクシステム

クラスメイトとの絆が戦闘に直結する学園RPGならではのシステム。
マルチエンディングの分岐条件にもなる重要システム。

**絆レベルの上げ方:**
- 学園生活パートでの会話選択肢
- 戦闘中の連携行動
- キャンプイベント（ダンジョン内の休憩ポイント）
- ストーリーイベントの選択
- ★ シオンとの絆は第2章まで限定。第3章で敵対後は上がらない

**絆レベルによる効果:**
| レベル | 効果 |
|--------|------|
| Lv1 | 連携攻撃「デュアルアーツ」が使用可能 |
| Lv2 | 戦闘中にかばう・回復を自動で行うことがある |
| Lv3 | 固有の「ソウルリンク必殺技」が解放 |
| Lv MAX | 専用ストーリーイベント解放 + 最強連携技 + ★分岐条件解放 |

**デュアルアーツの例:**
- レイ × ユナ：「深淵浄化斬」（物理+聖魔法の複合大ダメージ + 侵蝕ゲージ軽減）
- レイ × カイト：「双剣乱舞」（連続物理攻撃 + 高ブレイクダメージ）
- レイ × ミオ：「混沌の炎」（レイの闇 + ミオの暴走魔法 = ランダムだが超火力）
- ユナ × ミオ：「癒しの爆風」（味方全体回復。ただしミオの暴走で回復量にムラ）
- ★ レイ × シオン：「星影双斬」（光と闇の複合技。第2章限定。END5で再度使用可能）

### 6.4 侵蝕システム

レイの固有能力「深淵覚醒」を使うと一時的に超強化されるが、「侵蝕ゲージ」が蓄積。
魔王の力に飲まれるか、制御するかのリスク管理がゲームの核。
侵蝕レベルはエンディング分岐にも影響する。

- 侵蝕ゲージ 0〜30%: 通常状態
- 侵蝕ゲージ 30〜60%: NPCとの会話が変化。暗い選択肢が増える。攻撃力UP
- 侵蝕ゲージ 60〜90%: 深淵覚醒がより強力に。しかし暴走して味方を攻撃するリスク
- 侵蝕ゲージ 90%〜: END6（全滅エンド）のフラグが立ちやすくなる

**侵蝕の回復手段:**
- ユナの「浄化の光」（絆レベルに依存。Lv3で大幅回復）
- 聖水・星の雫等のアイテム
- 学園生活パートでの仲間との交流（自然回復）

### 6.5 成長システム

**レベルアップ:**
- 経験値で自動レベルアップ。基本ステータスが上昇
- レベルアップ時に「アカデミーポイント（AP）」を獲得

**スキルボード:**
```
            [剣術科]
                │
        ┌───────┼───────┐
        │       │       │
   [力+10]  [連続斬り] [カウンター]
        │       │       │
   [豪剣]  [疾風斬]  [鉄壁]
                │
          [奥義・深淵斬]
```
- APを消費してスキルやステータスを取得
- キャラごとに固有のボードを持つ（「学科」というフレーバー）

### 6.6 探索・フィールド

- **トップダウンビュー**のマップ探索
- 学園パート: 校舎内を自由移動。NPCとの会話、イベント発生
- ダンジョンパート: シンボルエンカウント、宝箱、ギミック
- ミニマップ表示あり
- ★夜の探索: 禁域に侵入可能（情報フラグ取得チャンスだが侵蝕リスク）

### 11.4 分岐テストの例

```csharp
// EndingConditionTests.cs の想定
[Test]
public void END1_WhenPlayerRefusesAcademy_ReturnsHiddenHappy()
{
    var flags = new FlagSystem();
    flags.Set("academy_refused", true);
    var result = EndingResolver.Resolve(flags, new TrustSystem());
    Assert.AreEqual(EndingType.HiddenHappy, result);
}

[Test]
public void END5_RequiresAllTrustMaxAndFullInfo()
{
    var flags = new FlagSystem();
    flags.SetAllInfoFlags(true); // 8/8 info flags
    var trust = new TrustSystem();
    trust.SetAll(80); // All trust >= 80
    var result = EndingResolver.Resolve(flags, trust);
    Assert.Contains(EndingType.TrueHappy, result.AvailableEndings);
}

[Test]
public void END6_WhenAllPartyDefeated_ReturnsTotalDefeat()
{
    var battleState = new BattleState();
    battleState.SetAllPartyHP(0);
    var result = EndingResolver.ResolveBattle(battleState);
    Assert.AreEqual(EndingType.TotalDefeat, result);
}
```
