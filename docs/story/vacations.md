# 長期休暇イベント詳細

## 概要

本作には3つの長期休暇がある。いずれもストーリー進行・SL深化・レベル上げの重要な期間。
各休暇には強制ストーリーイベントと自由行動パートが混在する。

---

## 夏休み1（1年夏 / 2週間）

**時期:** 1章（12週）と2章（13週）の間
**場所:** 学園 + 遠征先ダンジョン

### 1週目: 遠征ダンジョン解放

**強制イベント: 夏の遠征**
- カルロス引率で遠征先ダンジョン「灼熱の谷」が解放
- 灼熱の谷: 火属性モンスターが多い中級ダンジョン
- ミオ「火のモンスターだ！ わたしの出番！ ...あれ、わたし火属性だけど相性悪い？」
- レンジ「燃えるぜ！ ――って、ほんとに暑いな」

**自由行動パート:**
- 灼熱の谷でのレベル上げ・素材集め
- 遠征先の宿でSL深化チャンス

### 2週目: 仲間との夏イベント

**自由行動パート + SL深化チャンス**

**イベント一覧:**

1. **海辺の特訓**
   - カイトとレンジが海で特訓。波を斬る/殴る
   - カイト「波に勝てなきゃモンスターにも勝てねえ」
   - レンジ「負けねえ！」→ 二人とも波に飲まれる
   - カイト信頼+3 / レンジSLイベント

2. **ユナの浄化練習**
   - ユナがレイの侵蝕浄化を練習する
   - ユナ「わたしの力で、少しでもレイの負担を減らしたい」
   - 浄化の光がレイの腕の刻印に触れ、一瞬だけ温かくなる
   - ユナ信頼+5（選択肢次第）

3. **ミオとメルの料理対決**
   - ミオ「今度こそおいしいの作る！」/ メル「メルも参戦～！」
   - 結果: ミオの料理は壊滅的。メルは意外とまとも
   - 全員で味見→ 表情が面白い一枚絵
   - ミオSL / メルSL

4. **ジンの夜の情報収集**
   - ジンが遠征先の街で情報を集める
   - ジン「ここでも教団の噂がある。...規模がデカいな」
   - flag_trust_betrayal に関する伏線情報
   - ジンSL

5. **シオンとの夕暮れ**
   - シオンが一人で海を見ている。レイが声をかける
   - シオン「海は広いな。...こんな場所なら、リオンも喜んだだろう」
   - シオン信頼+5
   - ※ このシーンが後の闇落ちをより痛切にする

6. **セツナの砂の城**
   - セツナが黙々と砂の城を作っている。氷の魔法で細工を加え、異常に精巧
   - セツナ「...昔、お父さんと遺跡で見た城を...再現してみた」
   - セツナSL

7. **リンの漁**
   - リンが素手で魚を捕まえようとする。猫獣人の本能
   - リン「獲れたにゃ！ ...あっ、逃げたにゃ！」
   - リンSL

8. **ヴェインの夜読書**
   - ヴェインが宿で一人、教団の資料を読んでいる
   - レイが差し入れを持っていく→ ヴェイン「...勝手に入るな。...だが、茶は貰う」
   - ヴェインSL

## 冬休み（1年冬 / 3週間）

**時期:** 2章（24週）と3章（25週）の間
**場所:** レイの故郷 → 学園 → 学園周辺

### 1週目: 帰省

**強制イベント: レイの故郷へ帰還**
- レイ・ユナ・ミオの三人で故郷の村に帰る
- 身元引受人のギルド職員が出迎える「おかえり。大きくなったな」
- レイ「...ただいま」

**イベント一覧:**

1. **ユナの家を訪問**
   - ユナの母（治癒師）がレイたちを歓迎
   - ユナ母「あらあら、レイくん、立派になって。ユナをよろしくね」
   - ユナ（赤面）「お母さんっ！」
   - 幼少期の思い出の場所を巡るイベント
   - ユナ信頼+3

2. **ミオの家を訪問**
   - ミオの母が手料理でもてなす。ミオの料理下手は母譲りではなかった（母は上手い）
   - ミオ父は不在（「遠くに仕事」）。ミオが少し寂しそうにする
   - ミオ母「ミオ、お友達いっぱいできたのね。よかった」
   - ミオ信頼+3

3. **裏山の思い出**
   - 10歳の時に闇の力が暴発した裏山を訪れる
   - レイ「あの日、ここで...」
   - ユナ「覚えてる。わたし、あの日レイの手を握ったの」
   - ミオ「わたしは気絶してたから知らないんだけど...二人とも、あの時何があったの？」
   - ★選択: ミオに真実を話すか
     - 話す → ミオ信頼+5「...そっか。レイ、ずっと一人で怖かったんだね」
     - 話さない → ミオ「...分かった。話したくなったら聞くね」

4. **シオンのことを話す**
   - 夜、三人で縁側に座ってシオンの話をする
   - ユナ「シオン先輩、どうしてるかな...」
   - ミオ「先輩、絶対助けるんだから！」
   - レイ「...ああ。必ず」
   - シオン信頼+3（不在でも上がる。思い出の力）

5. **村のNPCとの会話**
   - 村人A「レイか？ 立派になったなぁ。昔は黙りっきりだったのに」
   - 村人B「ユナちゃんがいつもレイの面倒見てたっけ」
   - 幼少期の補足エピソード

### 2週目: 学園で自由行動

**自由行動パート**
- 学園に戻っての自由行動。SL深化の集中期間

**イベント一覧:**

1. **カイトの兄の墓参り**
   - カイトが一人で兄ハヤトの墓に行く。偶然レイが居合わせる
   - カイト「...見たのか。まあいい。兄さんの墓だ」
   - カイト信頼+5（このイベントでカイトがパーティ加入可能になる伏線）

2. **シオンの部屋**
   - シオンの部屋に入ろうとすると鍵がかかっている
   - セツナ「先輩の部屋...入れない...」
   - ※ シオンの日記は3章で別の方法で入手する

3. **メルの冬のコンサート**
   - メルが中庭で小さなコンサートを開く
   - 雪の中で歌うメルの声に、残っている生徒たちが集まる
   - メル「冬の歌は、温かさを思い出させてくれるの」
   - メルSL

4. **レンジとリンの修行**
   - レンジが雪の中で拳の修行。リンが巻き込まれる
   - リン「寒いにゃー！ でも負けないにゃ！」
   - レンジ「お前も根性あるな！」
   - レンジSL / リンSL

5. **ヴェインの図書館**
   - ヴェインが図書館で教団の歴史を調べている
   - 手伝うと教団エクリプスの成り立ちについて情報が得られる
   - ヴェインSL

6. **ジンの冬の情報**
   - ジンが冬休み中も情報収集を続けている
   - ジン「冬は人が少ない分、秘密が漏れやすい。稼ぎ時だ」
   - 教団の冬の動きについての情報
   - ジンSL

### 3週目: 強制ストーリー / カルロスの暗躍本格化

**強制イベント: 闇モンスターの街への出現**
- 冬休み最終週、学園の最寄りの街に闇属性のモンスターが出現
- 通常はダンジョンにしかいないモンスターが街に溢れる異常事態
- 住民がパニック。レイたちが防衛に駆り出される
- グレイヴ「封印が...加速的に弱まっている」

**ボス戦: 闇の群体（中ボス）**
- 街の広場での防衛戦
- 侵蝕ゲージに注意しながらの戦闘

**カルロスの暗躍:**
- カルロスが封印の弱体化を本格化させている
- 表向きは「街の防衛に全力を」と生徒を鼓舞
- 内心「いい機会だ。封印の崩壊を加速させよう」

**強制イベント後:**
- グレイヴが緊急会議を招集
- 学園の警備体制を最大レベルに引き上げ
- 3章（2年前期）へ移行

## 夏休み2（2年夏 / 2週間）

**時期:** 3章（36週）と4章（37週）の間
**場所:** 学園周辺

**※ 注意: この休暇は3章12週のシオン戦の前に発生する場合と、
3章の展開によっては存在しない場合がある。
シオン戦が3章12週に発生する場合、夏休み2は3章11週後に挿入される。**

### 1週目: 隠しダンジョン解放 / 最後のレベル上げ

**メインイベント: 隠しダンジョン「深淵の回廊」解放**
- 封印の間に近い地下に隠しダンジョンが出現
- 高難度エリア。シオン戦に向けた最後の修行場所
- レアドロップ・最強装備の素材がここで入手可能

**イベント一覧:**

1. **グレイヴの秘密の修行**
   - グレイヴがレイに個別で闇の力の制御を教える
   - グレイヴ「エルトにもこうして教えた。あいつは...上手くやった。最後まで」
   - 侵蝕制御スキルの強化イベント

2. **全員集合の決意表明**
   - パーティ全員が集まり、シオン救出の作戦会議
   - ジン「作戦はこうだ。シオン先輩の攻撃パターンは...」
   - ヴェイン「封印の力を利用した攻撃には、こう対処する」
   - 全員の絆を確認する重要シーン

3. **セツナの決意**
   - セツナが一人で訓練場にいる。氷の魔法を限界まで鍛える
   - セツナ「先輩...必ず連れ戻す。この力で」
   - セツナ信頼イベント

4. **カイトのパーティ加入（まだの場合）**
   - カイト「俺も行く。黙ってられねえ」
   - カイトがパーティに正式加入

5. **ミオの覚醒**
   - ミオが天才的な魔法制御を見せる瞬間
   - ソフィア先生「ミオ...あなた、ついに制御を掴んだのね」
   - ミオ「えっ、わたしそんなにすごい？」

### 2週目: 装備整理 / 最後の自由行動

**自由行動パート:**
- 最後の装備整理。ショップで最強装備を購入
- SLイベントの最終チャンス
- 各キャラとの「最後の日常」イベント

**イベント一覧:**

1. **レイとユナの二人きり**
   - 夜の屋上で二人が話す
   - ユナ「明日から...大変なことになるね」
   - レイ「...ユナ。ありがとう。いつも、そばにいてくれて」
   - ユナ「当たり前だよ。わたしがいるから」
   - ユナ信頼+5

2. **ジンとヴェインの情報共有**
   - 最終的な情報を整理。教団の拠点の見取り図を完成させる
   - ジン「これで全部だ。後は...お前らに任せた」
   - ヴェイン「...余計な世話だ。だが...助かった」

3. **メルの歌**
   - メルが全員のために歌を歌う。戦いの前の静寂
   - メル「明日のために。みんなのために。わたしの一番大切な歌を」
   - パーティ全体のSL微増

4. **リンの手紙**
   - リンが実家（翠嶺の里）に手紙を書いている
   - リン「お母さん。リンは元気にゃ。友達もいっぱいできたにゃ」
   - リンSL

5. **レンジの誓い**
   - レンジが拳甲を磨きながら父に語りかける
   - レンジ「親父、見てろよ。俺は強くなった」
   - レンジSL

## 休暇の設計意図

1. **ペース配分**: 章と章の間に息抜きを入れ、プレイヤーの緊張を調整
2. **SL深化の集中期間**: 通常の週では取りきれないSLイベントを消化する場
3. **キャラクター掘り下げ**: 戦闘のない場面でキャラの日常面を見せる
4. **レベル上げのタイミング**: 新ダンジョン解放で次章に向けた準備期間を提供
5. **伏線の仕込み**: 休暇中の何気ない会話が、後の展開で意味を持つ
