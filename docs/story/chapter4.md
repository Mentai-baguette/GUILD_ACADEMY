# 4章「最終決戦」 -- 2年後期（37~48週）

## 概要

3章12週のシオン戦の結果に応じて分岐する最終章。
6つのルートが並列に存在し、それぞれ異なる結末に向かう。
全ルート共通でカルロス（Lv100）との最終決戦がある（END4を除く）。

---

## ルート分岐の全体図

```
3章12週のシオン戦結果
    │
    ├── END2敗北 → [END2→4章ルート] → シオン再戦 → カルロス戦
    ├── END3敗北 → [END3→4章ルート] → シオン第3形態 → カルロス戦
    ├── 勝利(条件未達/信頼低) → [END4ルート] → カルロス追跡 → 撃破
    ├── 勝利(条件未達/信頼中) → [END4.5ルート] → カルロス撃破 → シオン回復
    ├── 勝利(条件達成) → [END5ルート] → 全員でカルロス撃破
    └── END6全滅 → [END6→4章ルート] → 侵蝕Critical → カルロス撃破 → 最終選択
```

※ Nightmare難易度ではEND2/END3/END6敗北時に4章に進行せず、そのENDで確定する。

## END2→4章ルート「残された意識の先」

### 前提条件
- シオン第1形態戦でシオン勝利（レイHP=0、幼馴染3人気絶）
- Nightmare以外の難易度

### 37週目: 蘇生

**イベント: グレイヴの介入**
- レイたちが倒れた後、グレイヴが駆けつける
- グレイヴの秘蔵アイテム「灰鋼の雫」でレイを蘇生
- グレイヴ「...まだだ。まだ終わらせん」
- レイは意識が朦朧とした状態で復帰。「俺は...死んだはずじゃ...」
- ユナの治癒でミオ・カイトも回復（カイトはこのルートでパーティ加入）

**状態:**
- パーティのレベルはシオン第1形態戦時のまま（Lv55前後）
- 装備・アイテムも維持
- 侵蝕ゲージがやや上昇した状態

### 38週目: 準備

**イベント: 再起**
- パーティが態勢を立て直す
- ジン「シオン先輩の居場所を掴んだ。教団の拠点だ」
- 装備の補強、レベル上げの最終チャンス
- セツナ「先輩を...今度こそ...」

### 39~44週目: 教団拠点攻略

**ダンジョン: 教団拠点「深淵の祭壇」**
- 教団エクリプスの拠点。複数のフロアで構成
- 道中で教団の構成員や魔王の眷属と戦闘

### 45週目: シオン再戦

**ボス: シオン第2形態「深淵の裁定者」（Lv75）**
- 前回と同じ第2形態だが、レイたちは成長していない（Lv55のまま）
- 高難度の戦闘。ギリギリの勝負になる設計
- 勝利してもシオンの意識は完全には戻らない
- シオン「レイ...すまな...」（一瞬だけ正気に戻るが、再び意識を失う）
- **シオンとの完全和解はできない**。このルートではシオンとの関係は不完全なまま

### 46~47週目: カルロス追跡

**イベント: カルロスとの対峙**
- シオンを倒した後、カルロスが現れる
- カルロス「ほう、まだ生きていたか。面白い」
- カルロス戦への道が開く

### 48週目: カルロス戦

**ラスボス: カルロス（Lv100）**
- 勝利 → END2最終演出（レイの魂残存。ユナだけが声を聞く）
- 敗北 → ゲームオーバー（リトライ可）

## END3→4章ルート「壊れた正義の先」

### 前提条件
- シオン第2形態戦でシオン勝利（4人殺害判定）
- Nightmare以外の難易度

### 37週目: 蘇生

**イベント: グレイヴの蘇生**
- グレイヴが「灰鋼の雫」と自身の闘気を使い、レイたちを蘇生
- グレイヴ「命を削ってでも...この子たちを死なせるわけにはいかん」
- 蘇生の代償でグレイヴの体に深刻なダメージ

### 38週目: 準備

**イベント: シオンの暴走情報**
- シオンは4人を殺した罪悪感で精神崩壊。薬と封印の力が制御不能に
- 第3形態「壊れた裁定者」として暴走状態
- ジン「シオン先輩が学園の周辺で暴れている。手がつけられない」

### 39~44週目: 準備・レベル上げ

- シオン第3形態はLv90。レイたちはLv75前後から成長が必要
- 鬼気の洞窟・翠緑の森の高難度エリアで修行

### 45週目: シオン第3形態戦

**ボス: シオン第3形態「壊れた裁定者」（Lv90）**
- 理性が砕け散った暴走状態。攻撃パターンがランダムかつ超高火力
- 双刃剣を二本に分離し、狂ったように振り回す
- 光と闇が同時に暴走。フィールド全体に破壊をまき散らす
- 戦闘中、シオンの声が断片的に聞こえる
  - 「リオン...ごめん...俺...みんなを...殺した...」
  - 「壊せ...全部壊せ...俺ごと...」
- **勝利してもシオンの精神は戻らない**。肉体的に制圧するのみ

### 46~47週目: カルロス追跡

- シオンを制圧した後、カルロスが姿を現す
- カルロス「壊れた駒はもう用済みだ。お前たちも同じだ」

### 48週目: カルロス戦

**ラスボス: カルロス（Lv100）**
- 勝利 → END3最終演出（シオンの精神は戻らない）
- 敗北 → ゲームオーバー（リトライ可）

### 前提条件
- シオン第2形態戦でプレイヤー勝利
- 救出条件達成（情報8/8 + 全員信頼80+ + flag_salvation_path ON）

- シオン「俺は...みんなに取り返しのつかないことを...」

- ユナ「先輩が何と言おうと、レイはレイです。そして先輩は先輩です」

**最強パーティでの教団拠点攻略**
- シオンを含む全メンバーで教団拠点に突入
- 道中でシオンと各キャラの和解イベント
  - セツナ「先輩...よかった...」（泣く）

  - シオン「ああ...楽しみにしている」（少しだけ笑う）

### 45~47週目: 教団拠点最深部

- カルロスの研究室でリオンの真実が判明

### 48週目: カルロス戦

  - 「退屈じゃなかったよ。お前たちは...本当に面白かった」
  - 最後に一瞬だけ、本物の笑みを見せる。「...悪くなかった」
  - 消滅

**END5最終演出:**
- 卒業式。全員の名前が呼ばれる
- シオン「俺はみんなに取り返しのつかないことをしかけた。許してもらえるとは思わない。でも...ありがとう。俺を止めてくれて」
- 全員で卒業写真を撮る
- レイ「...ありがとう。俺に居場所をくれて」
- 完全ハッピーエンド

## END6→4章ルート「深淵からの帰還」

### 前提条件
- シオン第2形態戦で全員HP=0
- Nightmare以外の難易度

### 37週目: 魔王暴走蘇生

**イベント: 魔王の力による蘇生**
- 全員が倒れた後、レイの体内の魔王の力が暴走
- レイの意識がなくなり、魔王の力が勝手にレイの体を動かして蘇生
- レイ「...俺は何を...」
- **侵蝕ゲージがCriticalに固定される。以降、下がらない**
- ユナの浄化でも抑えきれない。レイの体に黒い亀裂が走り始める

**パーティの状態:**
- 仲間はグレイヴの緊急治療で回復
- しかしレイの侵蝕が限界に近い。時間との戦い
- レイ「...俺の体、もう長くない。分かってる」
- ユナ「そんなこと言わないで...！ わたしが...！」

### 38~44週目: カルロス追跡（時間制限の緊迫感）

- 侵蝕がCritical固定のため、ダンジョン内での行動に制限
- 一定ターン以上戦闘を続けると侵蝕ゲージがMaxに近づく
- レイの闇スキルが強化されているが、使うたびに侵蝕が進む
- 仲間たちが「急ごう」「時間がない」と焦りを見せる

### 45~47週目: 教団拠点攻略

- カルロスへの道を切り開く
- シオンは操られた状態のまま教団側にいるが、戦闘にはならない
  - カルロスにとって「もう必要のない駒」扱い

### 48週目: カルロス戦 + 最終選択

**ラスボス: カルロス（Lv100）**
- 侵蝕Criticalのレイは闇スキルが大幅に強化されている
- しかし戦闘が長引くと侵蝕がMaxに到達し、暴走のリスク
- 勝利後、カルロス消滅

**★最終選択: レイの運命**
- カルロスを倒した後、レイの侵蝕が限界を超える
- グレイヴ「二つの道がある。封印の中に入るか...或いは...」

**選択肢A: 封印に入る**
- レイが自ら封印の中に入り、魔王の力と共に永遠に封じられる
- エルトと同じ運命を選ぶ
- レイ「先輩たちを...頼みます」
- 仲間との別れのシーン

**選択肢B: 仲間に殺してもらう**
- レイが仲間に自分を殺すよう頼む
- レイ「俺を...止めてくれ」
- ユナ「やだ...嫌だよ、レイ...！」
- カイト「...ふざけるな。お前を倒すのは俺だって決めてんだ」
- 最後にレイが目を閉じ、光の中に消える

詳細な演出は endings.md を参照。

## カルロス戦（Lv100）概要

### 共通仕様

**カルロスの戦闘スタイル:**
- 禁術を駆使した全属性攻撃。全属性を同時に扱える異常な能力
- 3フェーズ制
  - フェーズ1（HP100~60%）: 禁術の詠唱攻撃。高火力だが詠唱に隙がある
  - フェーズ2（HP60~30%）: 禁術全開放。フィールド全体に闇の領域を展開
  - フェーズ3（HP30~0%）: 最終形態。カルロスが魔王の力の一部を取り込む
- ブレイク耐性が極めて高い。長期戦になりがち
- 「退屈だな」「もっと楽しませてくれ」と挑発する台詞

**ルートごとの差異:**

| ルート | パーティ状態 | 難易度感 | 特記 |
|--------|------------|---------|------|
| END2→4章 | Lv55~60。シオン不在 | 最高難度 | レベル差が大きい |
| END3→4章 | Lv75~80。シオン不在 | 高難度 | 第3形態戦後で消耗 |
| END4 | Lv75~80。シオン不在 | 標準 | 怒りバフあり |
| END4.5 | Lv75~80。シオン不在 | 標準 | グレイヴの支援あり |
| END5 | Lv75~80。シオン復帰 | 最適 | 光と闇の連携が可能 |
| END6→4章 | Lv75~80。侵蝕Critical | 特殊 | 闇スキル強化+時間制限 |

**END5限定: シオンとの連携技**
- 「約束の光」: レイの闇とシオンの光が融合する最強合体技
- カルロスのフェーズ3でのみ使用可能
- 発動条件: シオンがパーティにいる + SL最大

## 4章の設計意図

1. **選択の重み**: 3章の結果が4章の難易度と結末を大きく変える
2. **周回のモチベーション**: 各ルートで異なる体験を提供
3. **カルロス戦の一貫性**: どのルートでもラスボスはカルロス（END4含む）
4. **感情のクライマックス**: 各ルートで異なる感情体験（悲壮・希望・決意）
5. **END5の達成感**: 最も困難な条件を達成したプレイヤーへの最大の報酬
