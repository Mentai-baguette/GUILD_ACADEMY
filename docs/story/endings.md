# 全7エンディング演出詳細

## 概要

本作には7つのエンディングが存在する（END1~END6 + END4.5）。
各ENDは条件・演出・後日談が異なり、全て見ることで物語の全体像が完成する。

---

## END1「知らなければ幸せだった」

### 条件
- ゲーム冒頭で「学園に行かない」を選択（隠し操作: ベッドに戻る / 門と反対に5回歩く）

### 演出シーン

**シーン1: 平穏な日常**
- レイ・ユナ・ミオの三人が故郷の村で穏やかに暮らす日常が流れる
- 朝、ユナが「レイくん、おはよう！ 今日もいい天気だね！」
- ミオが「あーっ、また寝坊した！ 待ってよー！」
- 三人で村の畑を手伝い、川で遊び、夕焼けの中を帰る
- BGM: 穏やかなアコースティックギター

**シーン2: 知らなかった物語**
- テキスト「レイは学園に行かなかった。闇の力は覚醒せず、魔王の器としての運命は眠り続けた」
- テキスト「ユナは村の治癒師を継ぎ、ミオは父の跡を追って旅に出た」
- テキスト「レイは村で静かに暮らした。記憶のない不安も、いつか薄れていった」

**シーン3: エンドロール**
- スタッフロールが流れる。BGMは子守歌調
- エンドロール後、暗転

**シーン4: 一枚絵（エンドロール後）**
- 暗い部屋。カルロスが机に向かっている
- カルロス「器が来なかったか。...まあいい。別の器を探すだけだ」
- 世界地図を広げ、別の場所に赤いピンを刺す
- テキスト「知らなければ幸せだった。だが、世界は止まらない」
- 画面暗転。END1完了

### 台詞の方向性
- 全体的に温かく穏やか。しかし最後の一枚絵で全てがひっくり返る
- プレイヤーに「この平和はレイだけの平和で、世界は救われていない」と気づかせる

## END2「残された意識」

### 条件
- シオン第1形態戦（Lv55）でシオン勝利（レイHP=0、幼馴染3人気絶）

### 演出シーン

**シーン1: 敗北**
- レイが倒れる。ユナが「レイ！！」と叫ぶが、気を失う
- ミオとカイトも気絶
- シオン「...終わった。これで...正しいことを...」（声は震えている）

**シーン2: 残された意識**
- 暗闇の中、レイの意識だけが浮遊している
- レイ「...俺は死んだのか？ でも...声が聞こえる」
- ユナの声が微かに聞こえる「レイ...レイ...起きて...」
- テキスト「レイの肉体は死んだ。しかし魔王の器としての魂は、封印の中に残った」

**シーン3: ユナだけが聞く声**
- 数日後。ユナがレイの墓の前に座っている
- 風が吹く。ユナの耳に、レイの声が微かに聞こえる
- レイ「...ユナ。泣くな」
- ユナ「レイ...？ レイなの...？ 聞こえる...レイの声が...」
- 周りの仲間には聞こえない。ユナだけが声を聞ける
- ミオ「ユナちゃん...？ 誰と話してるの...？」
- ユナ「レイが...レイがまだいるの...！」

**Nightmare以外の場合: → 4章へ**
- グレイヴが蘇生アイテムでレイを復活させる
- 4章ルートで復活したレイがシオンと再戦するが、シオンとの完全和解はできない
- カルロス撃破後のエピローグ:
  - レイ「先輩とはもう...前みたいにはなれないのかもしれない」
  - シオンは自分の罪の重さに押しつぶされたまま
  - ユナ「それでも、生きてる。それだけで十分だよ」

**Nightmare難易度の場合: END2確定**
- 蘇生なし。レイの魂は封印の中に永遠に残る
- ユナだけが声を聞き続ける
- テキスト「ユナは村に帰り、治癒師を継いだ。時折、風の中にレイの声を聞く」
- テキスト「それが本物か幻か、ユナには分からない。でも聞こえる限り、笑っていようと思った」
- 最後の一枚絵: ユナが窓辺で風に耳を傾けている。手にはレイのお守り

## END3「壊れた正義」

### 条件
- シオン第2形態戦（Lv75）でシオン勝利（4人殺害判定）

### 演出シーン

**シーン1: 4人の死**
- シオンが操られた状態でレイ・ユナ・ミオ・カイトを殺害
- シオンの体が勝手に動く。一人ずつ斬り伏せる
- カイト「仲間に...手を出す...な...」（レイの前に立ち、大剣を構えたまま倒れる）
- ミオ「いや...やだよ...」（泣きながら倒れる）
- ユナ「レイ...逃げて...」（レイを庇って倒れる）
- レイ「先輩...目を覚ませ...先...」（最後に倒れる）

**シーン2: シオンの精神崩壊**
- 操りが解ける。シオンが自分の手を見る。血に染まっている
- シオン「...何を...俺は...何をした...」
- リオンのピアスが血に濡れている
- シオン「レイ...ユナ...ミオ...カイト...」
- 叫び。精神が崩壊する。「あああああッ...！！」
- シオンの体から光と闇が同時に暴走。周囲が破壊される

**シーン3: 壊れた裁定者**
- テキスト「シオンの正義は4人の命を奪った。その代償は、シオン自身の心だった」
- シオンが膝をついて動かなくなる。目から光が消えている
- 口元だけが「ごめん...ごめん...」と動き続ける

**Nightmare以外の場合: → 4章へ**
- グレイヴの蘇生でレイたちが復活
- シオン第3形態（Lv90）との戦闘
- 勝利してもシオンの精神は戻らない
- カルロス撃破後のエピローグ:
  - シオンは施設で治療を受けるが、誰の声にも反応しない
  - セツナが毎日通い続ける「先輩...わたしは待っています。いつまでも」
  - テキスト「シオンの目に光が戻る日は来るのか。誰にも分からない」

**Nightmare難易度の場合: END3確定**
- 蘇生なし。4人は死亡。シオンは精神崩壊
- カルロスが現れ「使えない駒だ」とシオンを放置して去る
- テキスト「学園に残ったのは、壊れた正義の残骸だけだった」

- ピアスを握りしめる。「リオン...先に...行く...」

## 設計意図

1. **END1**: 隠しエンド。2周目で気づく驚き。「何もしない」選択にも結果がある
2. **END2**: レイの死を正面から描く。魂だけが残る切なさ
3. **END3**: 最も救いのないエンド。シオンの正義の脆さを突きつける
4. **END4**: メリーバッドエンド。勝ったのに救えなかったやるせなさ
5. **END4.5**: 不完全だが希望がある。セツナの想いが報われる
6. **END5**: 完全ハッピーエンド。最高難度の条件を達成した報酬
7. **END6**: 最終選択をプレイヤーに委ねる。どちらを選んでも涙が出る設計
