# ハッカソン版シナリオ（1章2週目まで）

## スコープ
- プレイ時間: 30分〜1時間
- 範囲: ゲーム開始〜1章2週目終了
- パーティ: レイ/ユナ/ミオ/カイト/シオン（5人）

---

## 1. 導入（暗転中）

### 前夜の職員の声
```

職員「クロノス・アカデミー...お前にとっていい場所になるといいが」
```

### 夢（断片的）
```
（暗闇の中に炎の映像がちらつく）
（何かの声）「...受け入れろ...」
（レイの声）「...誰だ...」
```

### 目覚め
```
レイ「っ...！」
（目が覚める。朝の光が差し込む部屋）
レイ「...夢か」
```

## 2. レイの部屋（操作チュートリアル）

### 世界設定
- 学園「クロノス・アカデミー」は王都の外れの丘の上にある。街に歩いて行ける距離
- レイは辺境の村から馬車で王都に来る

### レイの家
- 辺境の村。ギルド職員の家の2階
- 質素だが清潔。物が少ない（記憶がないから思い出の品がない）

### 調べられるオブジェクト

| 対象 | テキスト | 伝わること |
|------|---------|-----------|
| ベッド | 「もう少し寝ていたい...」 | END1隠し操作①の起点 |
| 机 | 「入学案内の手紙が置いてある。クロノス・アカデミー...」 | 世界観導入 |
| 窓 | 「朝日が差し込んでいる。今日はいい天気だ」 | 時間帯が朝 |
| 棚 | 「何もない。...俺の思い出って、何かあったっけ」 | 記憶喪失の暗示 |
| 鏡 | 「...」（レイの外見が映る） | 主人公の見た目 |
| ドア | 1階に降りる → ストーリー進行 |

### 操作チュートリアル
- 移動（方向キー）
- 調べる（決定ボタン）
- メニューはまだ使えない（学園到着後に解放）

### END1隠し操作①
ベッドに何度も戻ると内部フラグが立つ → END1へ

## 5. 学園に向かう道中（END1隠し操作②）

3人で道を歩いている。プレイヤーは前に進めば学園、後ろに歩けば村に戻る方向。

### 通常ルート
前に進む → 学園到着

### END1隠し操作②（反対方向に5回歩く）

**1回目:**
```
ユナ「レイ、学園こっちだよ？」
```
→ 戻される

**2回目:**
```
ミオ「寝ぼけてるの？ こっちこっち〜」
```
→ 戻される

**3回目:**
```
ユナ「レイ...？ どうしたの？」
ミオ「ねえ、なんか変だよ...」
```
→ 戻される

**4回目:**
```
ユナ「レイ、大丈夫？ 顔色悪いよ」
ミオ「無理しなくていいんだよ？ でも...」
```
→ 戻される

**5回目:**
```
ユナ「...レイ。本当に、行きたくないの？」
（沈黙）
ユナ「わたしは...レイがどうしたいか、それが一番大事だと思う」
ミオ「...ユナ」
ユナ「でも、もし怖いだけなら...わたしがいるから。大丈夫だから」
```

→ それでも反対に歩く場合:
```
レイ「...ごめん」
（レイが一人で去っていく）
（ユナとミオが見送る。ユナの目に涙）
```
→ END1エンディング演出へ

## 8. 初バトル（カイト戦・負けイベント）

### バトル設定
| 項目 | 内容 |
|------|------|
| 味方 | レイ（1人） |
| 敵 | カイト（勝てない設定） |
| 使えるコマンド | 攻撃/防御のみ（スキル/アイテムは未解放） |
| 結果 | 一定ターンで強制終了 |
| 教えること | ATBゲージの溜まり方、攻撃コマンド、防御コマンド |

### バトル後の分岐

**通常（数ターンで倒れる）:**
```
カイト「...それだけか」（背を向けて去る）
レイ「完敗だ...」
ユナ「レイ、大丈夫？」
ミオ「あいつ感じ悪〜い！」
```

**粘った場合（5ターン以上耐える）:**
```
カイト「...少しはやるな」（背を向けて去る）
レイ「...負けた」
ユナ「レイ、すごく頑張ってたよ！」
ミオ「あいつもちょっと認めてたっぽいよ！」
```

### バトル後
```
カルロス「落ち込むのは後だ。まだ始まったばかりだろう」
カルロス「今日はここまでだ。明日から本格的に授業が始まる」
```

## ハッカソン版で必要な機能まとめ

### 必須
- 移動（フィールド歩行）
- 会話（テキスト表示+キャラ名+立ち絵）
- 基本バトル（ATB+攻撃+防御+スキル）
- パーティ（5人：レイ/ユナ/ミオ/カイト/シオン）
- タイトル画面（NEW GAME/CONTINUE）
- 基本UI（HP/MP/ATBゲージ、コマンド選択）
- セーブ/ロード（基本機能）
- 授業スキップ演出
- シーン遷移（家→道→学園→寮）

### あると良い
- END1隠し操作（①ベッド、②反対方向5回）
- BGM（タイトル/学園日常/バトル/寮の4曲だけでも）
- 会話背景（部屋/道/教室/訓練場/寮の5枚だけでも）
- 立ち絵（レイ/ユナ/ミオ/カイト/シオン/カルロスの6人）

### 不要（完成版で）
- ダンジョン
- ブレイク/侵蝕/ソウルリンク
- スキルツリー
- 装備/ショップ
- 隊列/入れ替え
- リンクゲージ/デュアルアーツ
- 状態異常
- メニュー画面の大半のタブ
