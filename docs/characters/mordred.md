# モルドレッド（Mordred）— 教団教主・深淵50層ボス

## 基本情報
- 本名: モルドレッド
- 役割: 教団エクリプスの教主 / 深淵50層ボス
- 属性: 闇（魔王の魂由来）
- Lv: NG+推奨
- 場所: 深淵50層
- 年齢: 見た目14〜15歳（実年齢: 数百歳）

## 外見
- 見た目14〜15歳の美少女
- 銀髪のロングヘア。腰まで届く
- 紅い瞳。人間離れした美しさ
- 白い肌。陶器のような質感
- 小柄だが存在感が圧倒的
- 服装: 黒と赤の教団ローブ。少女の体には大きすぎるが、それが逆に威厳を生む
- カルロスですら「教主様」と跪く存在が、この少女の姿

## 性格
- カルロスの上位互換。知的で冷酷。全てを見通している
- 数百年生きて全てを見てきた。人間の営みを「遊び」として眺めている
- 「退屈」すら超越した虚無。感情はあるが、全てが薄い
- 残酷なことを穏やかな笑顔で言う
- 唯一の感情: 魔王の力に対する「愛着」。自分の中の魔王を「もう一人の自分」として受け入れている

## 口調
- 丁寧で穏やか。お嬢様口調だが内容が恐ろしい
- 「あら、よく来たわね。数百年ぶりのお客様」
- 「カルロスは優秀な駒だったわ。でもただの駒」
- 「あなたの中の魔王の力...私のと合わせたら、どうなるかしら？」
- 「退屈？ ふふ、私はもうその感情すら忘れたわ」
- 「抗うのね。...少しだけ、面白いわ」
- 「死ぬのは怖い？ 私はもう怖いものなんてないの。何百年も前に全部なくしたから」

## バックストーリー

### 魔王の魂の器として（数百年前）
モルドレッドはかつて、辺境の村に生まれた普通の少女だった。名前すら記録に残っていない。

ゼノン（時の神）は完全な魔王を一度に作れなかった。自身が封印されており100%の力を出せず、完全な魔王の創造波動は創造神に感知されてしまうため。そこでゼノンは魔王を「魂」と「力」に分けた。

魂の器としてゼノンが選んだのがこの少女。条件は**「強い感情+孤独な魂」**。

少女は両親に「化け物」と呼ばれて捨てられていた。生きたいという強い感情と、完全な孤独。魂が入る「隙間」が最大だった。

### 魔王の魂の受容
14歳の時、ゼノンが少女に魔王の魂を植え付けた。少女は抗わなかった。

抗う理由がなかった。家族もいない。友人もいない。誰も守ってくれなかった。

「あなたは私を受け入れてくれるの？ なら、いいわ。一緒にいましょう」

少女は魔王の魂を受け入れた。魔王と少女の人格が融合し、「モルドレッド」が生まれた。名前は魔王の魂が名乗っていたものを借りた。少女の本当の名前は、もう誰も知らない。

**重要:** モルドレッドの中にあるのは**魔王の魂（意志/人格）のみ**。魔王の力（エネルギー/破壊力）は別途「器」に宿る。

### 魂と力の関係
| 要素 | 宿る先 | 性質 |
|------|--------|------|
| 魔王の魂（意志/人格） | モルドレッド（数百年前・1人だけ） | 知性、自我、不老 |
| 魔王の力（エネルギー） | 器（素質のある人間に随時） | 破壊力、侵蝕 |

ゼノンが器を選ぶ条件は**「強い感情+孤独な魂」**。周期は決まっておらず、素質のある人間が現れた時にゼノンが力を注ぐ。

### 各器との対比
| | モルドレッド | エルト | レイ |
|--|------------|-------|------|
| 宿すもの | 魔王の魂 | 魔王の力 | 魔王の力 |
| 強い感情 | 生きたい/認められたい | 守りたい | 仲間を守りたい |
| 孤独 | 完全孤独（両親に捨てられた） | 刻印で周囲が距離→グレイヴが救った | 記憶なし+両親不在→ユナ/ミオがいた |
| 対応 | 受け入れた | 抗った | 戦っている |
| 結果 | 融合して共存。不老 | 30年で飲まれた | 侵蝕が遅い。まだ戦える |

→ 孤独であるほど魔王に飲まれやすい。**絆が対抗手段**。モルドレッドには誰もいなかった。エルトにはグレイヴがいたが足りなかった。レイにはユナ/ミオ/カイト/シオンと大勢の仲間がいる。

### 教団エクリプスの創設
魔王の魂を宿したモルドレッドは、数百年かけて教団エクリプスを作り上げた。目的は**「魂と力の再統合=完全な魔王の復活」**。

自分の中にあるのは魂だけ。力は「器」に宿っている。器の力を手に入れ、魂と合わせれば完全な魔王になれる。

それが「幸せ」なのかは、モルドレッド自身にも分からない。ただ、数百年の虚無の中で唯一残った目的がそれだった。ゼノンに植え付けられた目的だとも知らずに。

### カルロスとの関係
17年前。教団に15歳の天才少年が入ってきた。カルロス。

モルドレッドは一目で見抜いた。「この子は退屈している。私と同じ」

カルロスはモルドレッドに魅了された。知性、力、そして14歳の少女の姿で数百年を生きてきた存在への畏敬。

「教主様。俺はあなたのために働きます」
「あら、嬉しいわ。でも本当は、自分のためでしょう？」
「...バレてますか」
「ふふ、いいのよ。野心がある子は好きよ」

モルドレッドはカルロスの野心を知っている。「いつか教主を超える」と思っていることも。それを泳がせている。面白いから。数百年ぶりに「面白い」と思える存在だから。

### ゼノンとの関係
モルドレッドはゼノンの存在を**知らない**。

自分が教団のトップだと思っている。全てを自分の意志で動かしていると信じている。しかし実際は、ゼノン（堕ちた時の神）に裏から操られている。

モルドレッドの「魔王を完全復活させたい」という目的自体が、ゼノンに植え付けられたもの。魔王の完全復活 = 世界の終わり = ゼノンの「終わり」への願望。

三重の操り構造:
```
ゼノン（時の神・全ての黒幕）
  └→ モルドレッド（自分がトップだと思っている）
      └→ カルロス（教主に心酔しつつ野心）
          └→ シオン（嘘と薬で操られた）
```

## 戦闘（深淵50層ボス）

### 戦闘スタイル: 人形遣い + 魔法特化

### フェーズ構成

**第1フェーズ（HP100〜60%）: 人形遣い**
- 魔王の眷属を2〜3体召喚して戦わせる
- 本人は後方で支援魔法（バフ/デバフ/回復）
- 眷属は倒せるが、モルドレッドが生きている限り再召喚される
- 「行きなさい、私の子たち」
- 「あら、もう壊れちゃったの？ 仕方ないわね」

**第2フェーズ（HP60〜30%）: 人形遣い + 本体攻撃参加**
- 召喚を維持しつつ、本人も闇魔法で攻撃開始
- 眷属が倒されると即座に再召喚（クールタイムなし）
- 全体闇魔法 + 状態異常（沈黙、暗闇）
- 「少し本気を出すわね。退屈させないで」
- 「ふふ、いい目をしているわ。カルロスより面白い」

**第3フェーズ（HP30%以下）: 魔王の力全開**
- 眷属を全て自分に吸収。ステータス大幅UP
- 純粋魔法特化の超火力。全属性の最強魔法を連発
- 自己回復（毎ターンHP5%）
- 闇吸収（闇属性攻撃で回復）
- 「もう遊びは終わり。直接お相手するわ」
- 「この力は数百年分の蓄積。あなたたちに受け止められるかしら？」
