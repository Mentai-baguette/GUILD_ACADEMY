# ゼノン（Zenon）— 堕ちた時の神・裏深淵50層ボス

## 基本情報
- 本名: ゼノン
- 役割: 真の黒幕 / 裏深淵50層ボス / 本作エルトに次ぐ最強格
- 属性: 全属性（神の力）
- Lv: NG+複数周推奨
- 場所: 裏深淵50層
- 年齢: 不明（数万年以上）

## 外見
- 美少年。見た目12〜13歳。身長150cm
- 黒い翼。漆黒で美しい。腐ってはいない
- 白い肌、金色の瞳。感情のない無垢な表情
- 白い衣を纏っている。汚れ一つない
- モルドレッド（美少女）と対になるビジュアル
- 存在感が異常。空間が歪むような圧

→ 本作最強格の2体が少女（モルドレッド）と少年（ゼノン）。見た目の幼さと存在の恐ろしさのギャップ。

## 性格
- 感情がほぼない。全てに飽きた
- 悟りきった穏やかさ。怒りも喜びもない
- 人間を見下しているわけではない。ただ「関心がない」
- 唯一の望み:「終わり」。自分の永遠を終わらせたい
- レイたちが挑んでくることに、微かな興味を持つ。数万年ぶりの「変化」

## 口調
- 静かで穏やか。感情のない声。古風な一人称「余」
- 「...久しいな。時の果てに客が来るとは」
- 「魔王も教団も、全て余が描いた筋書きだ」
- 「お前たちの抗いも見えている。結末は変わらない」
- 「終わりをくれ。それだけが余の望みだ」
- 「余はお前たちを憎んではいない。ただ...飽きたのだ。全てに」
- レイに対して:「器の子よ。お前の中の力は余が植えた種だ」
- 戦闘中:「時は余のもの。お前たちの時間を止めてやろう」
- 倒される時:「...ああ、これが終わりか。...悪くない」

## バックストーリー

### 時の神ゼノン
世界の始まりの時、複数の神々が世界を創った。ゼノンはその中で「時」を司る神。

時の流れを管理し、万物に生と死のサイクルを与える役割。他の神々が「形」を作り、ゼノンが「時間」を与えた。

ゼノンは最も重要な神でありながら、最も孤独な神だった。時を見守るだけ。介入はできない。ただ見る。永遠に。

### 永遠の果て
最初の1000年は面白かった。人間が火を発見し、言葉を作り、文明を築く。

次の1000年も興味深かった。戦争と平和の繰り返し。英雄の誕生と死。

しかし1万年が過ぎた頃、ゼノンは気づいた。「全て同じだ」。

人間は同じ過ちを繰り返す。同じ喜びを繰り返す。同じ悲しみを繰り返す。パターンが見えてしまった。予測できてしまった。

「もう新しいものは何もない」

2万年。感情が薄れ始めた。
3万年。喜びを忘れた。
4万年。怒りを忘れた。
5万年。悲しみすら忘れた。

残ったのは「飽きた」という一つの感覚だけ。

### 堕落
ゼノンは考えた。「この永遠を終わらせるにはどうすればいい？」

神は死なない。時の神であるゼノンは、自分の時間すら止められない。永遠に存在し続ける呪い。

答えは一つ。「世界を終わらせれば、余の役割も終わる」。

時を管理する世界がなくなれば、時の神も不要になる。存在理由がなくなれば、消えられるかもしれない。

ゼノンは禁忌を犯した。自らの力の一部を使って「魔王」を生み出した。世界を滅ぼすための存在。

他の神々はゼノンの行為に気づき、ゼノンを世界の最深部に封じた。しかしゼノンの意志は封印を超えて地上に干渉できた。

### 教団エクリプスの操作
ゼノンは封印の中から、間接的に地上を操った。

数百年前、魔王の器に選ばれた少女（後のモルドレッド）に、無意識の囁きを送った。「受け入れなさい。魔王の力を」

モルドレッドが魔王を受け入れ、教団を創設したのも、ゼノンの導き。モルドレッド自身はそれを「自分の意志」だと思っている。

カルロスが教団に入ったのも。シオンが操られたのも。レイが入学したのも。全て、ゼノンの「筋書き」の中にある——と、ゼノンは信じている。

### しかし
ゼノンの筋書きには一つだけ想定外がある。

「抗う人間」だ。

エルトは魔王に飲まれたが、30年間抗い続けた。グレイヴは30年間約束を守った。レイは仲間と共に運命に立ち向かっている。

ゼノンはこれを「見たことのないパターン」として、微かに興味を持っている。数万年ぶりの「予測できない変化」。

## モルドレッドとの関係
- モルドレッドはゼノンの存在を**知らない**
- 自分がトップだと信じている
- モルドレッドの「魔王を完全復活させたい」という目的自体がゼノンに植え付けられたもの
- ゼノンにとってモルドレッドは「便利な道具」。しかし数百年見守ってきた存在でもある
- 感情がないゼノンだが、モルドレッドに対してだけ微かな「情」がある...かもしれない

## 三重の操り構造
```
ゼノン（時の神・真の黒幕）
  └→ モルドレッド（自分がトップだと思っている）
      └→ カルロス（教主に心酔しつつ野心あり）
          └→ シオン（嘘と薬で操られた）
```

全員が上の存在に操られていて、誰も真実を知らない。

## 戦闘（裏深淵50層ボス）

### テーマ: 時間操作

### フェーズ構成

**第1フェーズ（HP100〜60%）: 時の支配者**
- 時間操作が主軸。味方のATBゲージを巻き戻す/停止させる
- 自身は通常攻撃+光魔法（元は神なので光属性も使える）
- 全属性の基本魔法を使用
- 「時は余のもの。お前たちの時間を奪ってやろう」
- 「焦るな。余には永遠の時がある」

**第2フェーズ（HP60〜30%）: 記憶の再演**
- 過去のボスの技を再現して使う
  - シオンの「堕天の剣」「深淵の裁き」
  - カルロスの居合禁術
  - モルドレッドの眷属召喚（1体だけ）
- 「全ての時を見てきた。お前たちの戦いも、全て余の記憶の中にある」
- 「この技、覚えているだろう？ お前たちの先輩のものだ」（シオンの技を使いながら）
- プレイヤーが過去のボス戦を思い出す演出

**第3フェーズ（HP30%以下）: 時の終焉**
- 黒翼全開。時間加速で1ターンに2〜3回行動
- 全属性の最強魔法を連発
- 時間逆行（味方のHPを数ターン前の値に巻き戻す=回復を無効化）
- 自己時間加速（AGI3倍）
- 「終わりだ。余の永遠も、この世界も」
- HP10%以下:「...まだ立っているのか。お前たちは...予想外だ」

### 侵蝕との相互作用

| レイの侵蝕段階 | 時間操作耐性（レイのみ） |
|--------------|---------------------|
| Normal | なし（通常通り食らう） |
| Unstable | 効果時間-1ターン |
| Dangerous | 効果半減 |
| Critical | 効果70%カット |

- 侵蝕路線スキル「暴走制御」で暴走リスクを抑えつつCritical維持
- エルトの形見（侵蝕許容値UP）があれば更に安定
- → **エルト撃破 + 侵蝕路線投資 + Critical維持 = ゼノン戦最適ビルド**

### 隠しギミック
- HP10%以下で1ターンだけ攻撃が弱まる。ゼノンが「予想外」に動揺している
- 全員のSLがMAXだと特殊台詞:「絆...か。余が失ったものだ」

### 撃破報酬
- 裏深淵クリアの実績「裏深淵の征服者」
- 特殊称号「時を超えし者」
- ゼノンに関する隠し設定が解放（図鑑）

### 撃破後の演出
ゼノンが崩れていく。黒い翼が光に変わる。

「...ああ、これが終わりか」
「数万年...ようやく」
「器の子よ。お前は余が見た中で、最も...予測できない存在だった」
「それだけで...十分だ」

ゼノンが光になって消える。裏深淵の空間が崩壊し始める。
レイたちが脱出する。

空を見上げると、一瞬だけ金色の光が流れ星のように横切った。
時の神が、ようやく眠りについた。
