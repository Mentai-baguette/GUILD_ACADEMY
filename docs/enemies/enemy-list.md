# 敵一覧（Enemy List）

本ドキュメントは全ダンジョンに出現する敵のマスターデータ一覧。ステータスは目安値であり、バランス調整で変更可能。

## ステータス凡例

| 略称 | 意味 |
|------|------|
| Lv | レベル |
| HP | ヒットポイント |
| ATK | 物理攻撃力 |
| DEF | 物理防御力 |
| INT | 魔法攻撃力 |
| RES | 魔法防御力 |
| AGI | 素早さ（ATBゲージ蓄積速度） |
| DEX | 器用さ（クリティカル率） |
| G | ドロップゴールド |
| 弱 | 弱点（1.5倍） |
| 耐 | 耐性（0.5倍） |
| 無 | 無効（0倍） |
| 吸 | 吸収（ダメージ分回復） |
