# BGM・SE 完全リスト

## 概要

BGM全36曲 + SE全20種。各トラックの使用場面、雰囲気、使用楽器、ループ/クロスフェード仕様を記載する。

## 1. シーン系BGM（17曲）

### bgm_title — タイトル画面

| 項目 | 内容 |
|------|------|
| ID | bgm_title |
| 雰囲気 | 荘厳かつ不穏。物語の壮大さと闇の予感を同時に伝える |
| 使用楽器 | 低音オーケストラ（チェロ、コントラバス）+ ピアノ。後半に微かなコーラス |

## 5. BGM再生ルール

### 5-1. クロスフェード規則

| 遷移パターン | フェード時間 | 方式 |
|-------------|------------|------|
| エリア間（同カテゴリ） | 2秒 | クロスフェード |
| エリア間（異カテゴリ） | 1.5秒 | フェードアウト→フェードイン |
| フィールド → バトル | 0.5秒 | 即時切替（バトルBGMのイントロ重視） |
| バトル → リザルト | 0秒 | 即時切替 |
| リザルト → フィールド | 1秒 | フェードアウト→フィールドBGM復帰 |
| イベント開始 | 1秒 | クロスフェード |
| ボスフェーズ切替 | 0秒 | 同一トラック内のレイヤー切替（シームレス） |

### 5-2. BGM優先度

```
ボス戦BGM > イベントBGM > 通常バトルBGM > ダンジョンBGM > フィールドBGM
```

上位のBGMが終了すると、下位のBGMに自動復帰する。

### 5-3. バトル速度とBGM

- バトル速度1.0x: BGMはそのまま再生
- バトル速度1.5x/2.0x: BGMの再生速度は変更しない（テンポは固定）
- 自動戦闘: BGMはそのまま継続

## 6. SE再生ルール

### 6-1. 同時再生制限

- SE同時再生上限: 8チャンネル
- 同一SEの重複再生: 0.05秒以内の再トリガーは無視（連打防止）
- 優先度: バトルSE > UI SE（バトル中のUI操作音は抑制）

### 6-2. 音量バランス

| カテゴリ | デフォルト音量 | 調整可能範囲 |
|---------|-------------|-------------|
| BGM | 70% | 0〜100% |
| SE | 80% | 0〜100% |
| ボイス（将来対応） | 100% | 0〜100% |
| 環境音 | 50% | 0〜100% |
