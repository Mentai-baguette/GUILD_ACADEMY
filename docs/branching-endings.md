# 分岐・エンディングシステム詳細

## システム概要

マダミス（マーダーミステリー）風の分岐システム。プレイヤーの「情報収集」「信頼構築」「戦闘判断」の3要素が複合的にエンディングを決定する。

## 情報フラグシステム（FlagSystem）

8種類の情報フラグ。学園探索パートでNPCとの会話や特定場所の調査でON/OFFが切り替わる。

```
flag_shion_past       : シオンの過去を知る
flag_carlos_plan      : カルロスの計画を知る
flag_vessel_truth     : 魔王の器の真実を知る
flag_academy_secret   : 学園の秘密を知る
flag_seal_method      : 封印の方法を知る
flag_dark_power_risk  : 闇の力のリスクを知る
flag_trust_betrayal   : 裏切りの情報を知る
flag_salvation_path   : 救済の道を知る
```

## 信頼ポイントシステム（TrustSystem）

**10キャラ分**の信頼値（0〜100）。会話選択肢・戦闘中の判断で増減。

### メインパーティ（4人）
- **ユナ信頼**: 守り・連携系の選択で上昇
- **ミオ信頼**: 知識・分析系の選択で上昇
- **カイト信頼**: 勇気・行動系の選択で上昇
- **シオン信頼**: 理解・共感系の選択で上昇（隠しパラメータ）

### 新キャラクター（6人）
- **リン信頼**: 信頼値あり
- **ヴェイン信頼**: 信頼値あり
- **メル信頼**: 信頼値あり
- **ジン信頼**: 信頼値あり
- **セツナ信頼**: 信頼値あり
- **レンジ信頼**: 信頼値あり

## BranchManager

FlagSystem + TrustSystem + EndingResolver を統合する中央管理クラス。

```
情報フラグ8種 → 信頼ポイント10人分 → エンディング条件判定
        ↓              ↓                    ↓
   NPC会話で取得    選択肢で増減     EndingResolverで判定
```

## エンディング条件一覧（全7END）

### END1: 裏ハッピーエンド
- **条件**: ゲーム冒頭で「学園に行かない」を選択
- **結果**: 平穏な日常が続く
- **判定タイミング**: ゲーム開始直後

### END2: トゥルーエンド
- **条件**: シオン第1形態戦でシオン勝利（レイHP=0、幼馴染3人気絶）
- **結果**: レイは死亡するが意識が残る
- **判定タイミング**: シオン第1形態戦の勝敗

### END3: シオンルート
- **条件**: シオン第2形態戦でシオン勝利
- **結果**: 4人を殺してしまった罪悪感でシオンが精神崩壊
- **判定タイミング**: シオン第2形態戦の勝敗

### END4: ノーマル（メリーバッドエンド）「Pyrrhic Victory」
- **条件**: シオン第2形態戦でレイたちが勝利、ただしシオン救出条件を満たしていない
- **結果**: シオンを倒すが、カルロスがシオンの心臓を潰して殺害。カルロスは撤退
- **判定タイミング**: シオン第2形態戦後 + 救出条件（信頼値 + フラグ）チェック

### END4.5:（新規追加エンド）
- **条件**: END4とEND5の中間条件を満たした場合
- **結果**: END4より良いがEND5には届かない中間的結末
- **判定タイミング**: シオン第2形態戦後 + 救出条件の部分達成

### END5: 表ハッピーエンド「夜明けの学園」
- **条件**: 情報フラグ8/8 + 全員信頼80+ + 第2形態戦で特殊コマンド「シオンの名を呼ぶ」解放 → シオンを闇から引きずり出す → カルロス戦で撃破
- **結果**: シオン救出、5人でカルロスを倒す。全員が生き残る唯一の道
- **判定タイミング**: シオン第2形態戦後 + 信頼値 + フラグチェック → カルロス戦勝利

### END6: 全滅エンド「深淵の勝者」
- **条件**: シオン第2形態戦で全員のHPが0になる / 特定の選択ミスの蓄積
- **結果**: 全員が倒れた隙にカルロスがレイから魔王の力を抽出。世界崩壊が始まる
- **判定タイミング**: シオン第2形態戦の勝敗 + 選択肢蓄積

```
[タイトル]

    | YES

    |
[第2章: 異変]（情報フラグ収集 + 信頼ポイント蓄積）
    |
[第3章: 断罪]（シオンが真実を知る）
    |
[シオン第1形態戦（光）]
    | シオン勝利 --> END2（トゥルー）
    | プレイヤー勝利
    |
[第4章: 深淵]（カルロス介入）
    |
[シオン第2形態戦（闇・操られ）]
    | シオン勝利 --> END3（シオンルート）
    | プレイヤー勝利
    |
[信頼値 + フラグ判定]
    | シオン信頼低い --> END4（ノーマル/メリバ）

    | 勝利 --> END5（表ハッピー）
```

## 魔王の構造

魔王は「魂（モルドレッド）」と「力（器）」に分離されている。

- **魂（モルドレッド）**: 魔王の意識・人格。封印の奥で意志を持ち続ける
- **力（器）**: レイの肉体に封じられた魔王の力。侵蝕システムの根源

カルロスの真の目的は、器から力を抽出し自分が魔王の力を手に入れること。教団の「魔王復活」とは方向性が異なる（教団はモルドレッド復活、カルロスは力の奪取）。
