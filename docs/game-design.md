# ゲームデザイン決定事項

## 基本情報

- **タイトル**: GUILD ACADEMY: ABYSS CODE
- **ジャンル**: 2D 学園ダークファンタジーRPG（マルチエンディング）
- **エンジン**: Unity 6.4 + C#
- **対応プラットフォーム**: macOS（開発・テスト・プレイすべてMac対応）
- **ハッカソン**: 千葉工業大学「システム開発における生成AI活用ワークショップ」（3/20〜4/10）
- **成果発表会**: 2026年4月11日（土）

## 世界観・ストーリー

魔法学園「ギルドアカデミー」を舞台にしたダークファンタジー。主人公レイは「魔王の器」として覚醒し、その力をめぐって仲間・敵が交錯する。

### ストーリー構成（4章構成）

| 章 | タイトル | 概要 |
|----|---------|------|
| 第1章 | 入学 | レイがクロノス・アカデミーに入学。魔王の力が一瞬覚醒 |
| 第2章 | 異変 | 学園の異変、禁域調査。シオン先輩が独自に調査開始 |
| 第3章 | 断罪 | シオンがレイ＝魔王の器と知り、殺す決意をする |
| 第4章 | 深淵 | 封印の間での最終決戦。カルロスの正体が明かされる |

### 魔王の構造

魔王は「魂（モルドレッド）」と「力（器）」に分離されている。レイは力を封じた「器」であり、魂であるモルドレッドは別に存在する。カルロスの真の目的は、レイから力を抽出し自分が新たな魔王になること。

### カレンダー

- **ゲーム内期間**: 57週（旧: 90日）
- 学園生活パートと冒険パートが週単位で進行

## バトルシステム

- **ATB（Active Time Battle）**: ゲージが溜まったら行動可能
- **ブレイクシステム**: 敵の耐性を削って大ダメージチャンス
- **侵蝕（Erosion）システム**: レイの魔王の力。リスク＆リワード
- **ソウルリンク（Soul Link）システム**: パーティの絆レベル。戦闘とエンディングに影響
- **隊列システム**: ドラクエ式の隊列歩行。前衛・後衛の概念あり
- **リンクゲージ + デュアルアーツ**: 55種の連携技。リンクゲージが溜まると発動可能
- **装備属性**: 装備に属性が付与される

### ステータス

| ステータス | 説明 |
|-----------|------|
| HP | 体力 |
| MP | 魔力 |
| ATK | 物理攻撃力 |
| DEF | 物理防御力 |
| MAT | 魔法攻撃力 |
| MDF | 魔法防御力 |
| AGI | 敏捷性（旧: SPD） |
| INT | 知力 |
| RES | 耐性 |
| DEX | 器用さ |
| LUK | 幸運 |

### スキルシステム（SP制スキルツリー）

- レベルアップでSP（スキルポイント）を獲得
- SPをスキルツリーに投入してスキルを習得
- 旧方式（Lv解放+スキル書）から変更

## マダミス（マーダーミステリー）風分岐

- **情報フラグ**: 8種類。NPC会話・場所調査でフラグON
- **信頼ポイント**: **10人分**（レイを除くパーティメンバー+新キャラ含む）
- **分岐タイミング**: 会話中の選択肢 + バトル中の判断の両方組み合わせ
