# ゲームデザイン決定事項

## 基本情報

- **タイトル**: GUILD ACADEMY: ABYSS CODE
- **ジャンル**: 2D 学園ダークファンタジーRPG（マルチエンディング）
- **エンジン**: Unity 6 (6000.x) + C#
- **対応プラットフォーム**: macOS（開発・テスト・プレイすべてMac対応）
- **ハッカソン**: 千葉工業大学「システム開発における生成AI活用ワークショップ」（3/20〜4/10）
- **成果発表会**: 2026年4月11日（土）

## 世界観・ストーリー

魔法学園「ギルドアカデミー」を舞台にしたダークファンタジー。主人公レイは「魔王の器」として覚醒し、その力をめぐって仲間・敵が交錯する。

### シオンの2形態

- **第1形態**: 自分の意志でレイたちに立ちはだかる（**光の力**を使用）
- **第2形態**: カルロスに操られて4人に襲いかかる（**闇の力**を使用）

## バトルシステム

- **ATB（Active Time Battle）**: ゲージが溜まったら行動可能
- **ブレイクシステム**: 敵の耐性を削って大ダメージチャンス
- **侵蝕（Erosion）システム**: レイの魔王の力。リスク＆リワード
- **ソウルリンク（Soul Link）システム**: パーティの絆レベル。戦闘とエンディングに影響

## 6マルチエンディング

| END | 名称 | 条件概要 |
|-----|------|---------|
| END1 | 裏ハッピーエンド | 一番最初に学園に行かない選択 |
| END2 | トゥルーエンド | シオン第1形態戦でシオン勝利（レイ死亡、幼馴染3人気絶）。レイの意識が残る |
| END3 | シオンルート | シオン第2形態戦でシオン勝利→4人を殺した罪悪感でシオンが潰れる |
| END4 | ノーマル（メリバ）エンド | シオン第2形態戦敗北→シオンがカルロスに心臓を潰されて死亡 |
| END5 | 表ハッピーエンド | シオンを殺さずに闇から引きずり出し、カルロスを倒す |
| END6 | 全滅エンド | 特定の失敗条件 |

## マダミス（マーダーミステリー）風分岐

- **情報フラグ**: 8種類。NPC会話・場所調査でフラグON
- **信頼ポイント**: ミナト・ユイ・ハル・シオンの4人分
- **分岐タイミング**: 会話中の選択肢 + バトル中の判断の両方組み合わせ
