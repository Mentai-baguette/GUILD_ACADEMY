# UI/UX 画面遷移仕様書

## 概要

本ドキュメントは全画面の一覧、遷移フロー、各画面の要素配置、操作方法を定義する。

---

## 1. 全体画面遷移図

```
タイトル画面
├── NEW GAME → 難易度選択 → OPイベント → ゲーム本編
├── CONTINUE → セーブスロット選択 → ゲーム本編
├── GALLERY → 図鑑/実績/会話回想
├── SETTING → 各種設定
└── EXIT → アプリ終了

ゲーム本編
├── フィールド画面（学園/街/ダンジョン）
│   ├── NPC接触 → 会話画面
│   ├── ショップNPC接触 → ショップ画面
│   ├── 錬金釜接触 → 錬金画面
│   ├── エンカウント → バトル画面 → リザルト画面 → フィールド復帰
│   ├── ボス突入 → バトル画面（ボス）→ リザルト画面 → イベント/フィールド
│   └── メニューボタン → メニュー画面
├── 授業画面
│   ├── 授業演出（スキップ可能）
│   └── 章末テスト画面
├── 時間帯切替（朝→昼→放課後→夜）
└── イベントシーン → 会話画面（自動遷移）
```

---

## 2. タイトル画面

### 画面要素

```
┌─────────────────────────────────┐
│                                 │
│     GUILD ACADEMY: ABYSS CODE   │  ← タイトルロゴ（中央上部）
│                                 │
│         [NEW GAME]              │  ← メニュー項目（中央）
│         [CONTINUE]              │
│         [GALLERY]               │
│         [SETTING]               │
│         [EXIT]                  │
│                                 │
│  (C) 2026 ...            v1.0  │  ← バージョン情報（右下）
└─────────────────────────────────┘
```

### 操作

| 入力 | 動作 |
|------|------|
| 上下キー/左スティック | メニュー項目の選択移動 |
| 決定ボタン | 選択した項目へ遷移 |
| SE | se_ui_select（移動）、se_ui_confirm（決定） |

### 遷移先

| 項目 | 遷移先 |
|------|--------|
| NEW GAME | 難易度選択画面 |
| CONTINUE | セーブスロット選択画面 |
| GALLERY | ギャラリー画面（END到達後に解放） |
| SETTING | 設定画面 |
| EXIT | アプリ終了 |

## 3. 難易度選択画面

### 画面要素

```
┌─────────────────────────────────┐
│  難易度を選んでください          │
│                                 │
│  [EASY]    物語を楽しみたい方へ  │
│  [NORMAL]  バランスの取れた難度  │
│  [HARD]    歯応えのある戦闘      │
│  [NIGHTMARE] 最高難度(NG+限定)   │
│                                 │
│  ※後から変更可能（Nightmare除く）│

```

### 難易度による影響

| 難易度 | 敵ステータス | 獲得EXP/G | END制限 | 備考 |
|--------|------------|-----------|---------|------|
| EASY | x0.7 | x1.5 | なし | 戦闘敗北時にリトライ可能 |
| NORMAL | x1.0 | x1.0 | なし | 標準バランス |
| HARD | x1.3 | x0.8 | なし | 敵AIが賢くなる |
| NIGHTMARE | x1.6 | x0.6 | END2/3/6なし（4章強制突入） | NG+専用。END4/4.5/5のみ到達可能 |

## 4. フィールド画面

### 画面要素

```
┌─────────────────────────────────────────┐
│ [ミニマップ]          [時間帯表示]       │  ← 画面上部
│  ┌──────┐            朝/昼/放課後/夜     │
│  │ MAP  │                               │

```

### 操作

| 入力 | 動作 |
|------|------|
| 方向キー/左スティック | キャラクター移動 |
| 決定ボタン | NPC話しかけ/オブジェクト調査 |

- レイの侵蝕率が0%超の場合のみ表示

  - 30〜59%: 黄（注意）
  - 60〜89%: 赤（危険）
  - 90%〜: 紫+脈動エフェクト（臨界）

## 6. バトル画面

### 画面要素

```
┌──────────────────────────────────────────────┐
│ [敵パーティ]                                  │  ← 上部: 敵表示エリア
│  ┌──────┐  ┌──────┐  ┌──────┐               │
│  │敵1   │  │敵2   │  │敵3   │               │
│  │[HP]  │  │[HP]  │  │[HP]  │               │
│  │[BRK] │  │[BRK] │  │[BRK] │               │
│  └──────┘  └──────┘  └──────┘               │
│                                              │
│                [リンクゲージ]                  │  ← 中央: リンクゲージ表示
│                ████████░░ 80%                 │
│                                              │
│  [速度] [AUTO]                               │  ← 左下: バトル速度/自動戦闘ボタン
│                                              │
│ ┌────────────────────────────────────────┐   │
│ │ [味方1]  HP ███░  MP ██░  ATB ████████ │   │  ← 下部: 味方ステータス
│ │ [味方2]  HP ████  MP ███  ATB ██████░░ │   │
│ │ [味方3]  HP ██░░  MP █░░  ATB ████░░░░ │   │
│ │ [味方4]  HP ████  MP ██░  ATB ██░░░░░░ │   │
│ └────────────────────────────────────────┘   │
│                                              │
│  ┌──────────────┐                            │
│  │ ▶攻撃         │                            │  ← コマンドウィンドウ
│  │  スキル       │                            │    （ATBゲージMAX時に表示）
│  │  アイテム     │                            │
│  │  防御         │                            │
│  │  デュアルアーツ│                            │
│  │  入替         │                            │
│  │  逃走         │                            │
│  └──────────────┘                            │
└──────────────────────────────────────────────┘
```

### コマンド一覧

| コマンド | 説明 |
|---------|------|
| 攻撃 | 通常攻撃。対象選択 → 実行 |
| スキル | スキルリスト表示。MP消費 |
| アイテム | アイテムリスト表示。対象選択 |
| 防御 | ATBゲージ蓄積速度x1.5 + 被ダメージ半減 |
| デュアルアーツ | パートナー選択 → 固有合体技発動（リンクゲージ50%以上必要） |
| 入替 | 控えメンバーと交代（交代後ATBゲージ0から） |
| 逃走 | 通常戦闘のみ。ボス戦では表示されない |

### バトル速度切替

- 画面左下に速度ボタンを配置
- タップ/ショートカットキーで 1.0x → 1.5x → 2.0x を循環
- 現在速度をアイコンで表示: [>] / [>>] / [>>>]
- BGMのテンポは変更しない（SE・アニメーションの速度のみ変更）

### 自動戦闘

- 速度ボタンの右隣に [AUTO] ボタンを配置
- ON/OFFトグル。ONで通常攻撃を自動実行
- スキル/アイテム/デュアルアーツは自動では使用しない
- ボス戦では自動戦闘を無効化

### ブレイクゲージ表示

- 各敵の名前の下にブレイクゲージをバー表示
- ブレイク達成時: ゲージが割れるエフェクト + se_break再生
- ブレイク中は敵の枠が赤く点滅

### 侵蝕関連UI

- レイのステータス欄に侵蝕ゲージを追加表示
- 侵蝕率に応じてレイの立ち絵/スプライトが変化
- 侵蝕スキル選択時に「侵蝕+X%」の警告テキスト表示

## 8. メニュー画面

### 画面構成

```
┌─────────────────────────────────────────┐
│ [メニュータブ]                           │
│  ステータス | 装備 | スキルツリー | アイテム│
│  パーティ  | 履修 | ノート | 図鑑          │
│  コンフィグ | セーブ・ロード               │
│                                         │
│ ┌─────────────────────────────────────┐ │
│ │                                     │ │
│ │  [選択中タブの内容表示エリア]         │ │
│ │                                     │ │
│ └─────────────────────────────────────┘ │
│                                         │
│  プレイ時間: 12:34:56   Gold: 15000     │
└─────────────────────────────────────────┘
```

### 各タブの内容

#### ステータス

- キャラ選択 → HP/MP/ATK/DEF/MATK/MDEF/SPD/LUK 表示
- 侵蝕ゲージ（レイのみ）
- ソウルリンクレベル一覧
- 現在の状態異常/バフ表示

#### 装備

- キャラ選択 → 6スロット表示（武器/頭/上半身/下半身/アクセ1/アクセ2）
- スロット選択 → 装備可能アイテム一覧 → ステータス変化プレビュー
- 「最強装備」ボタン: ATK or DEF 基準で自動装備

#### スキルツリー

- キャラ選択 → ツリー表示（6〜7路線 x 5段階）
- 未習得スキルはグレーアウト、習得済みは点灯
- SP残量表示 + 習得時のステータス変化プレビュー

#### アイテム

- カテゴリ分け: 回復/攻撃/素材/貴重品/鍵
- ソート: 名前順/入手順/種類順
- アイテム選択 → 説明表示 + 使用可能な場合は「使用」ボタン

#### パーティ編成

- 前衛4名 + 控え枠の配置
- ドラッグ&ドロップ or 選択式で入れ替え
- 各キャラの簡易ステータス表示

#### 履修

- 現在の履修科目一覧
- 各科目の出席率表示
- 章の切り替え時のみ変更可能（表示は常時）

#### ノート

- 受講した授業の内容閲覧
- 章・科目でフィルタリング可能
- 欠席した授業は「(欠席)」表示

#### 図鑑

- 敵図鑑: 遭遇した敵の情報（HP/弱点/ドロップ）
- アイテム図鑑: 入手したことのあるアイテム
- 実績一覧: 達成状況の確認

#### コンフィグ

- BGM音量（0〜100%）
- SE音量（0〜100%）
- テキスト速度（遅い/普通/速い/瞬時）
- オートモード速度（遅い/普通/速い）
- バトル速度デフォルト（1.0x/1.5x/2.0x）
- 難易度変更（Nightmare以外は変更可能）
- 画面サイズ（ウィンドウ/フルスクリーン）
- キーコンフィグ

#### セーブ・ロード

- セーブスロット: 20枠 + オートセーブ1枠
- スロットにはプレイ時間、章、現在地、レベル、スクリーンショットを表示
- セーブ可能場所: 寮の自室のみ（フィールドセーブ不可）
- ロードは任意のタイミングで可能

## 13. 画面遷移時の共通ルール

### トランジション

| 遷移パターン | エフェクト | 時間 |
|-------------|-----------|------|
| エリア間移動 | フェードブラック | 0.5秒 |
| バトル開始 | 画面が割れるエフェクト | 0.8秒 |
| バトル終了 | フェードホワイト | 0.5秒 |
| メニュー開閉 | スライドイン/アウト（右から） | 0.3秒 |
| 会話開始/終了 | テキストウィンドウのスライドアップ/ダウン | 0.2秒 |
| イベントシーン開始 | シネマスコープ（上下黒帯） | 0.5秒 |

### 入力受付

- トランジション中は入力を受け付けない
- ローディング中はローディングアイコン表示（画面右下に小さく回転アイコン）
- 長時間ロード（2秒以上）の場合、Tips表示
