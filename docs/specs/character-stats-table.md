# キャラクター初期ステータス表

本ドキュメントは全パーティメンバー11名のLv1初期ステータス、成長率、推定ステータス、科目ブーストを定義する。

> **注意**: タスク指示は「13キャラ」だが、パーティメンバーとして定義されているのは11名。ボスキャラ（カルロス/モルドレッド等）のステータスは `docs/enemies/enemy-list.md` で管理する。

---

## 1. ステータス項目の説明

| 項目 | 略称 | 説明 | 戦闘での影響 |
|------|------|------|-------------|
| HP | HP | ヒットポイント。0になると戦闘不能 | 生存力の基盤 |
| MP | MP | マジックポイント。スキル使用時に消費 | スキル使用回数の上限 |
| ATK | ATK | 攻撃力。物理攻撃の威力に影響 | 物理ダメージ = (ATK x 2 - 敵DEF) x 各種倍率 |
| DEF | DEF | 防御力。物理攻撃の被ダメージを軽減 | 被物理ダメージ軽減 |
| INT | INT | 知力。魔法攻撃の威力・回復量に影響 | 魔法ダメージ = (INT x 2 - 敵RES) x 各種倍率 |
| RES | RES | 魔法防御。魔法攻撃の被ダメージを軽減 | 被魔法ダメージ軽減 |
| AGI | AGI | 敏捷性。ATBゲージの蓄積速度に影響 | ATBゲージ蓄積 = AGI x 0.1 / フレーム。逃走成功率にも影響 |
| DEX | DEX | 器用さ。クリティカル率・状態異常命中率に影響 | クリティカル率 = DEX / 200。状態異常命中判定の基準値 |
| LUK | LUK | 幸運。ドロップ率・レアエンカウント率に影響 | ドロップ率補正、レアアイテム出現率、一部イベント判定 |

### 設計意図
- **レイ**: ATK/INT両方そこそこ。突出した弱点がないバランス型。侵蝕システムで火力を補える設計
- **ユナ**: INT/RES最高クラスで回復・魔法防御の要。ATK最低タイで物理戦闘は不向き。LUK高め
- **ミオ**: INT最高タイの紙耐久魔法アタッカー。MP豊富でスキル連発可能。HP/DEF低い
- **カイト**: ATK最高・HP高い脳筋。MP/INT最低クラス。ブレイク特化の物理火力
- **シオン**: 全ステータス11前後で弱点なし。万能だが突出もない。2年生の先輩らしい安定感
- **リン**: HP/DEF最高のタンク。AGI最低（5）で行動は遅いがパーティの壁役
- **ヴェイン**: DEX最高（15）で状態異常命中率に優れる。LUK最低（82）がデメリット
- **メル**: LUK最高（115）でドロップ率等に貢献。RES高めでバフ/サブヒーラーとして安定
- **ジン**: AGI/DEX最高の超速キャラ。HP最低（80）/DEF/RES最低クラスの紙耐久
- **セツナ**: INT最高タイ・MP最高の純粋魔法アタッカー。ATK最低（5）・HP/DEF低い
- **レンジ**: ATK/AGI高い物理寄りハイブリッド。RES低めで魔法に弱い

### 成長率の設計意図
- 各キャラの役割を反映。アタッカーは攻撃系成長率が高く、タンクはHP/DEFが高い
- **レイ**: ATK/INT両方の成長率が高めで後半もハイブリッドが機能する
- **シオン**: 全体的に40-50%で安定成長。突出はないが穴もない
- **リン**: HP 90%・DEF 65%で終盤は鉄壁。AGI 10%で速度は伸びない
- **ジン**: AGI 70%・DEX 55%で終盤の速度は圧倒的。HP 40%で脆さは変わらず
- **セツナ**: INT 70%・MP 70%で最終的な魔法火力とリソースは全キャラ最高

## 補足

### ダメージ計算式との関係（参照: battle-system.md）
- 物理ダメージ: `(ATK x 2 - 敵DEF) x スキル倍率 x 属性相性 x クリティカル x ブレイク x 乱数`
- 魔法ダメージ: `(INT x 2 - 敵RES) x スキル倍率 x 属性相性 x クリティカル x ブレイク x 乱数`
- クリティカル率: `DEX / 200`（DEX 100 = 50%、DEX 200 = 100%上限）
- ATBゲージ蓄積速度: `AGI x 0.1 / フレーム`

### EXPカーブ
- 次レベルまでの必要EXP: `Lv x Lv x 5`
- レベル上限: 100

### 侵蝕システムとの連動（レイ専用）
- 侵蝕段階に応じてATK倍率が変動: Normal 1.0x / Unstable 1.3x / Dangerous 1.6x / Critical 2.0x
- Critical時は暴走確率あり。ステータス設計上、レイのATK/INTが中程度なのは侵蝕ブーストを前提としている
