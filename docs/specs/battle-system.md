# バトルシステム仕様書

## 概要

ATB（Active Time Battle）方式のコマンドバトル。ブレイクシステム・侵蝕システム・ソウルリンクシステムが連動し、戦略的な戦闘を実現する。

---

## 1. ATBシステム

### ゲージ仕様

| 項目 | 値 |
|------|-----|
| MAXゲージ | 100 |
| 蓄積速度 | AGI x 0.1 / フレーム |
| バトル開始時 | 全員ゲージ0 |

- ゲージがMAXに達したキャラクターから行動可能
- 同時にMAXに達した場合、AGIの高い方が優先
- AGI同値の場合、味方 > 敵の順（味方内はパーティ順）

### ゲージ増減の修正

| 条件 | 倍率 |
|------|------|
| 防御中 | x1.5（蓄積速度UP） |
| ヘイスト状態 | x1.3 |
| スロウ状態 | x0.5 |
| ストップ状態 | x0.0（完全停止） |
| バトル中入れ替え直後 | ゲージ0からスタート |

### 実装参照

既存クラス: `ATBSystem.cs`
- `GaugeRate = 10.0f`（仕様のAGI x 0.1に対応。Spdフィールドを使用）
- `MaxGauge = 100.0f`

> **注意**: 現在の`CharacterStats`にはAGIフィールドがなく`Spd`を使用している。仕様上のAGIは実装上`Spd`に対応する。INT/RES/DEX/LUKフィールドの追加が必要。

## 2. ダメージ計算式

### 物理ダメージ

```
最終ダメージ = (ATK x 2 - DEF) x スキル倍率 x 属性相性 x クリティカル x ブレイク x 乱数
```

### 魔法ダメージ

```
最終ダメージ = (INT x 2 - RES) x スキル倍率 x 属性相性 x クリティカル x ブレイク x 乱数
```

### 各係数

| 係数 | 値 | 備考 |
|------|-----|------|
| スキル倍率 | 各スキルに設定（通常攻撃 = 1.0） | SkillDataで定義 |
| 属性相性 | 弱点1.5 / 耐性0.5 / 無効0.0 / 等倍1.0 | 後述 |
| クリティカル | 1.5倍 or 1.0倍 | 発動率 = DEX / 200 |
| ブレイク | 2.0倍 or 1.0倍 | ブレイク中の敵への攻撃 |
| 乱数 | 0.9 ~ 1.1（均等分布） | |

### クリティカル

- クリティカル率 = DEX / 200（百分率）
  - DEX 100 で 50%
  - DEX 200 で 100%（上限）
- クリティカル倍率 = 1.5倍
- 回復スキルにはクリティカル適用しない

### 最低ダメージ

- 計算結果が1未満の場合、最低ダメージ = 1
- 属性無効（NullMultiplier）の場合のみダメージ0

### 隊列補正（後述の隊列セクション参照）

物理攻撃のみ隊列の影響を受ける。魔法攻撃は隊列を無視する。

### 実装との差異

現在の`DamageCalculator.cs`は簡略化された計算式を使用。仕様に合わせて以下の変更が必要:

1. `ATK x 2 - DEF`形式への変更
2. INT/RES分離（物理/魔法ダメージ区別）
3. クリティカル率をDEX依存に変更（現在は固定15%）
4. 乱数を0.9~1.1の倍率方式に変更（現在は加算方式）

## 3. コマンド一覧

| コマンド | 説明 | ターン消費 | ATBゲージ消費 |
|----------|------|-----------|--------------|
| 攻撃 | 通常攻撃（倍率1.0） | Yes | 全消費 |
| スキル | MP消費してスキル発動 | Yes | 全消費 |
| アイテム | アイテム使用 | Yes | 全消費 |
| 防御 | 被ダメ50%軽減 + ATB蓄積1.5倍 | Yes | 全消費 |
| 逃走 | バトルから逃走を試みる | Yes | 全消費 |
| デュアルアーツ | 2人協力技（詳細は`dual-arts.md`参照） | Yes | 両者全消費 |
| チェンジ | キャラ個別の前列⇔後列切替 | Yes | 全消費 |
| 入れ替え | 控えメンバーと交代（FF10式） | Yes | 全消費 |

### 防御の詳細

- 被ダメージ 50% 軽減（物理・魔法両方）
- ATBゲージ蓄積速度 1.5倍
- 次の行動まで防御状態を維持
- 防御中に追加コマンド不可

### 逃走の詳細

```
逃走成功率 = 50% + (味方平均AGI - 敵平均AGI) x 2%
```

| ルール | 詳細 |
|--------|------|
| 成功率下限 | 0% |
| 成功率上限 | 100% |
| ボス戦 | 逃走不可 |
| 煙玉 | 成功率 +50%（加算） |
| 失敗時 | ターン消費のみ（再試行可） |

### 入れ替え（控え交代）

- FF10方式: バトル中にバトルメンバーと控えを交代
- 交代したキャラのATBゲージは0からスタート
- 交代で登場したキャラはバトル参加扱い（EXP全額取得）
- 1ターンで1人のみ交代可能

## 4. ブレイクシステム

### ブレイクゲージ

| 項目 | 値 |
|------|-----|
| MAXゲージ | 100 |
| 弱点攻撃時の蓄積 | +25 |
| 通常攻撃時の蓄積 | +5 |
| ブレイク持続 | 2ターン |

### ブレイク中の効果

- 被ダメージ 2.0倍
- 状態異常耐性 半減（耐性値を50%にして判定）
- ブレイク持続2ターン経過後、ゲージは0にリセット

### 実装参照

既存クラス: `BreakSystem.cs`
- `WEAK_HIT_VALUE = 25`, `NORMAL_HIT_VALUE = 5`, `BREAK_DURATION = 2`, `DEFAULT_MAX = 100`
- 仕様と完全一致。追加変更不要。

## 5. 属性相性

### 循環弱点（物理元素）

```
火 → 氷 → 風 → 地 → 火（循環）
```

- 火は氷に弱点（1.5倍）
- 氷は風に弱点（1.5倍）
- 風は地に弱点（1.5倍）
- 地は火に弱点（1.5倍）

### 光と闇

- 光 ⇔ 闇（相互弱点、1.5倍）

### 同属性耐性

- 同じ属性の攻撃を受けた場合、耐性0.5倍

### ボス専用ルール

- ボスは自属性攻撃を吸収（ダメージ分HP回復）
- 吸収は`ElementType.Absorb`フラグで管理

### 属性相性テーブル

| 攻撃属性 \ 防御属性 | 火 | 氷 | 風 | 地 | 光 | 闇 |
|-----|-----|-----|-----|-----|-----|-----|
| 火  | 0.5 | 1.5 | 1.0 | 1.0 | 1.0 | 1.0 |
| 氷  | 1.0 | 0.5 | 1.5 | 1.0 | 1.0 | 1.0 |
| 風  | 1.0 | 1.0 | 0.5 | 1.5 | 1.0 | 1.0 |
| 地  | 1.5 | 1.0 | 1.0 | 0.5 | 1.0 | 1.0 |
| 光  | 1.0 | 1.0 | 1.0 | 1.0 | 0.5 | 1.5 |
| 闇  | 1.0 | 1.0 | 1.0 | 1.0 | 1.5 | 0.5 |

## 6. 隊列システム

### 前列 / 後列

| 位置 | 物理与ダメ | 物理被ダメ | 魔法 |
|------|-----------|-----------|------|
| 前列 | 等倍 | 等倍 | 隊列無視 |
| 後列 | -20% | -20% | 隊列無視 |

### チェンジ

- バトル中にコマンド「チェンジ」でキャラ個別の前列⇔後列切替
- ターン消費あり
- ATBゲージ全消費
- 対象キャラ1人の前後列を切り替える（個別変更）

### 初期配置

- キャラごとにデフォルト隊列を設定
  - 前列デフォルト: レイ、カイト、リン、レンジ
  - 後列デフォルト: ユナ、ミオ、ヴェイン、メル、ジン、セツナ

---

## 7. ターゲティング

### 単体攻撃

- デフォルトターゲット: 最も近い敵（前列優先）
- プレイヤーは任意の敵を選択可能

### 全体攻撃（範囲減衰あり）

| 位置 | ダメージ倍率 |
|------|-------------|
| 中心 | 100% |
| 隣接 | 85% |
| 2つ隣 | 70% |
| 3つ隣 | 60% |

- 敵の並び順は左から右（最大6体）
- 中心は術者の正面の敵

### 全体攻撃（均等フラグ）

- `isUniform = true` のスキルは全員に100%ダメージ
- 主にバフ・デバフ・一部の上位魔法で使用

### 回復ターゲティング

- 単体回復: HP最低の生存味方を自動選択（手動変更可）
- 全体回復: 生存味方全員

### 蘇生ターゲティング

- 最も近い戦闘不能者を自動選択（手動変更可）

---

## 8. リンクゲージ

### 基本仕様

- パーティ共有ゲージ: 0 ~ 100%
- バトル開始時: 前回バトル終了時の値を引き継ぐ（フィールド復帰で10%減少）

### 蓄積条件

| 行動 | 蓄積量 |
|------|--------|
| 与ダメージ（物理/魔法） | +3% |
| 被ダメージ | +2% |
| バフ/デバフ付与 | +5% |
| ブレイク達成 | +8% |
| クリティカルヒット | +2%（与ダメとは別途加算） |
| 毎ターン自然蓄積 | +1% |

### 消費

- デュアルアーツ発動: -50%

---

## 9. バトル速度

| 速度 | 倍率 | 説明 |
|------|------|------|
| 通常 | 1.0x | デフォルト |
| 高速 | 1.5x | ATB蓄積とアニメーション1.5倍速 |
| 超高速 | 2.0x | ATB蓄積とアニメーション2.0倍速 |

- 設定画面またはバトル中にトグル可能
- 速度変更はATB蓄積速度とアニメーション速度の両方に影響

### 自動戦闘

- ON時: 全キャラが通常攻撃のみを自動で行う
- 全場面でON/OFF切り替え可能（ボス戦含む）
- コマンド入力待ちをスキップ
- ターゲットは自動選択（HP最高の敵を優先）
- デュアルアーツ・スキル・アイテムは使用しない

---

## 10. EXP配分

### 基本ルール

- 戦闘勝利時にEXPをパーティに配分
- 出場ターン比率で配分

### 計算式

```
キャラEXP = 総EXP x (そのキャラの出場ターン数 / 全キャラの出場ターン合計)
```

### 控えメンバー

- 1ターンも出場していないメンバーは「控え扱い」
- 控え扱い: EXP = 総EXP x 0.5 / 控えメンバー数
- 入れ替えで1ターンでも出場すればバトル参加扱い（上記の通常計算）

### ボーナス

- ブレイク達成: EXP +10%
- デュアルアーツ使用: EXP +5%
- 全員生存クリア: EXP +20%

---

## 11. バトルメンバー構成

### パーティ上限

- バトルメンバー: 最大4人
- 控えメンバー: 最大4人
- 合計: 最大8人

### 固定メンバー

- ストーリー進行により、特定キャラの離脱・強制加入あり
- ボス戦でのメンバー固定あり（シオン戦など）

---

## 12. バトルリザルト

### 勝利時

1. EXP配分（上記参照）
2. ゴールド取得
3. ドロップアイテム判定
4. ソウルリンクポイント蓄積（一緒にバトルしたメンバー +2pt）
5. 侵蝕値の確認（闇スキル使用時のみ変動）

### 敗北時

- 雑魚戦: ゲームオーバー → タイトルに戻る
- ボス戦: エンディング分岐判定（`EndingResolver`参照）
- シオン戦敗北 → END2 or END3
- カルロス戦全滅 → END6
