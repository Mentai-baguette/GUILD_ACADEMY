# デュアルアーツ仕様書

## 概要

2人のキャラが協力して放つ強力な合体技。ソウルリンクレベルに応じて威力が向上する。全55種（レイ+仲間10種 + 仲間同士45種）。

---

## 1. 発動条件

| 条件 | 値 |
|------|-----|
| リンクゲージ | 50%以上 |
| 両者のATBゲージ | 両者ともMAX（100） |
| ソウルリンクレベル | Lv1以上（該当ペアの技が解放済み） |

### 発動手順

1. ATBゲージMAXのキャラがコマンド選択時に「デュアルアーツ」を選択
2. パートナー候補一覧が表示（ATBゲージMAXかつSL Lv1以上のキャラのみ）
3. パートナーを選択 → 技が自動決定（ペアごとに固有技）
4. 発動

---

## 2. 消費リソース

| リソース | 消費量 |
|----------|--------|
| リンクゲージ | -50% |
| 発動者ATBゲージ | 全消費（0にリセット） |
| パートナーATBゲージ | 全消費（0にリセット） |
| MP | 各キャラのMP消費（通常スキルの50%程度） |

---

## 3. SL倍率（ソウルリンクレベル補正）

| SLレベル | 倍率 | 解放条件 |
|----------|------|---------|
| Lv1 | x1.0 | SL Lv1到達（20pt） |
| Lv2 | x1.2 | SL Lv2到達（40pt） |
| Lv3 | x1.5 | SL Lv3到達（70pt） |
| MAX | x2.0 | SL MAX到達（100pt） |

### ベース威力

- デュアルアーツのベース威力 = 通常スキルの約1.5倍
- 最終威力 = ベース威力 x SL倍率 x 属性相性 x ブレイク x 乱数

## 6. 解放条件の詳細

### 解放タイプ一覧

| タイプ | 条件 | 対象技数 |
|--------|------|---------|
| SL Lv1 | ペアのSLが両方Lv1以上 | 35種 |
| バトル累計 | 指定ペアで累計N回共闘 | 10種 |
| END5限定 | END5ルートでシオン復帰後 | 10種 |

### バトル累計の定義

- 「共闘」= 同じバトルに同時に出場している状態
- 入れ替えで片方が控えに下がった場合、そのバトルはカウントしない
- 勝利・敗北問わずカウント

### END5限定技の扱い

- シオンがパーティに復帰した後のカルロス戦でのみ使用可能
- シオンとの10種は全てこのカテゴリ
- カルロス戦開始時に自動解放（SL条件不問）

## 7. 実装メモ

### データ構造

```
DualArtsData
├── Id (int)
├── Name (string)
├── Character1 (CharacterId)
├── Character2 (CharacterId)
├── Element (ElementType[]) — 複合属性対応
├── TargetType (SkillTargetType)
├── BasePower (int) — 通常スキルの約1.5倍
├── MpCost1 (int) — キャラ1のMP消費
├── MpCost2 (int) — キャラ2のMP消費
├── Effects[] — 追加効果（バフ/デバフ/状態異常等）
├── UnlockType (enum: SoulLink / BattleCount / End5)
└── UnlockValue (int) — SL Lv or バトル回数
```

### 複合属性の判定

- 複合属性攻撃は各属性ごとに相性判定
- 片方が弱点、片方が耐性の場合: 弱点倍率のみ適用（有利側優先）
- 両方弱点の場合: 1.5 x 1.5 = 2.25倍
- 両方耐性の場合: 0.5倍（最低耐性で判定）
