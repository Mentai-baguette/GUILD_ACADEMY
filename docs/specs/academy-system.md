# 学園システム 仕様書

## 1. 科目一覧と効果

9科目。各科目を履修すると対応するステータス成長率にボーナスが付く。

| # | 科目名 | 担当教師 | ステータス効果 | 備考 |
|---|--------|---------|--------------|------|
| 1 | 戦闘実技 | ガロン | ATK成長率+10% | 実技演習あり |
| 2 | 戦闘理論 | ダリオ | DEF成長率+10%, クリティカル回避+3% | 座学 |
| 3 | 魔法実技 | エレナ | MATK成長率+10% | 実技演習あり |
| 4 | 魔法理論 | カルロス※ | MDEF成長率+10%, MP成長率+5% | 座学。※2章後半〜代理教師に交代 |
| 5 | 戦術 | グレイヴ(特別講師) | SPD成長率+5%, 先制攻撃率+5% | 月2回の特別授業 |
| 6 | 錬金術 | ニコラ | 錬金成功率+5%, 素材鑑定スキル | 3章で錬金釜解放 |
| 7 | 歴史 | マルガリータ | 情報フラグ収集にボーナス | ストーリー理解に影響 |
| 8 | 医学 | セラフィーナ | 回復アイテム効果+10%, 状態異常回復速度+1ターン | 特定キャラ必須 |
| 9 | 商学 | ダリオ(兼任) | ショップ割引+5%, ドロップG+10% | 隠しメニュー解放条件 |

## 2. 履修ルール

### 2-1. 基本ルール

- **履修登録**: 各章の開始時に行う（章途中での変更不可）
- **最大履修数**: 7科目/章
- **最低履修数**: 3科目/章（強制）
- **1日のコマ数**: 最大5コマ。1コマ = 1授業時間

### 2-2. 5コマ目ルール

- **5コマ目は放課後の時間帯に食い込む**
- 5コマ目を取っている日は放課後の自由行動が不可（夜パートから開始）
- ダンジョン探索コストにも影響（後述: dungeon-system.md参照）

### 2-3. サボり

- 任意の授業をサボれる（日単位で「サボる」コマンド）
- サボった授業は出席率に反映されない
- サボった分の授業ノートは空白になる
- サボった時間は放課後扱い（自由行動可能）
- **ペナルティ**: 出席率が低いと章末テストの成績が下がる

---

## 3. 章末テスト

### 3-1. テスト仕様

- **タイミング**: 各章の最終週
- **出題**: 履修科目 × 各1問 = 最大7問
- **形式**: 4択問題（授業ノートの内容から出題）
- **制限時間**: 1問あたり30秒

### 3-2. 成績計算

```
成績スコア = (正解数 / 履修科目数) × 出席率
```

| 成績ランク | スコア | 報酬 |
|-----------|-------|------|
| S | 0.95〜1.00 | 先生信頼度+3, スキル書1冊, 習熟度ブースト+15% |
| A | 0.80〜0.94 | 先生信頼度+2, スキル書1冊, 習熟度ブースト+10% |
| B | 0.60〜0.79 | 先生信頼度+1, 習熟度ブースト+5% |
| C | 0.40〜0.59 | 先生信頼度±0 |
| D | 0.00〜0.39 | 先生信頼度-1（下限0） |

- **スキル書**: 対象科目に関連するスキルを習得できるアイテム
- **習熟度ブースト**: 次の章の間、該当科目の成長率ボーナスに上乗せ

## 4. 授業ノート

- 受けた授業の内容が自動記録される
- **メニュー → 授業ノート** から閲覧可能
- サボった授業は「(欠席)」と表示され内容が空白
- 章末テストの出題元となるため、ノートの確認がテスト対策になる
- NG+では既読の授業ノートは引き継がれない（テストの意味を保つため）

## 5. 仲間の履修

### 5-1. 基本ルール

- 仲間キャラも科目を履修する（ランダム選択、シード固定で周回同一）
- 各キャラに **履修傾向** があり、得意科目を優先的に選ぶ

### 5-2. キャラ別履修傾向

| キャラ | 優先科目 | 回避科目 | 必須科目 |
|--------|---------|---------|---------|
| ユナ | 魔法実技, 魔法理論 | 戦闘実技 | 医学(自動) |
| ミオ | 魔法実技, 錬金術, 歴史 | 戦闘実技 | 医学(自動) |
| カイト | 戦闘実技, 戦術 | 魔法理論, 歴史 | — |
| リン | 戦闘実技, 戦闘理論 | 商学 | — |
| ジン | 戦術, 商学, 歴史 | 魔法実技 | — |
| ヴェイン | 魔法理論, 歴史, 錬金術 | 戦闘実技 | — |
| メル | 魔法実技, 歴史, 医学 | 戦闘理論 | 医学(自動) |
| レンジ | 戦闘実技, 戦術 | 魔法理論, 錬金術 | — |
| セツナ | 魔法実技, 魔法理論, 錬金術 | 戦闘実技 | — |
| シオン | 全科目均等（別学年扱い） | — | — |

### 5-3. 履修の影響

- 仲間が履修した科目の **成長率に最大+5%** のボーナス
- 同じ科目を履修している仲間同士で SL ポイントに微量ボーナス (+1/週)

### 5-4. 履修相談イベント

- 各章開始時の履修登録前に「履修相談」イベントが発生
- SL Lv1以上の仲間に対して1科目だけ履修を提案できる
- 提案が受理されるかは信頼度に依存（SL Lv2以上でほぼ確定）
- 1章につき最大2人まで相談可能

## 6. 医学科目の特殊ルール

- **ユナ, ミオ, メル**: 医学を自動履修（履修枠を1消費。解除不可）
- **レイ**: 医学は履修不可（魔王の器であることに起因する設定上の制約）
- 医学履修者は回復スキルの効果量にボーナス

## 7. 授業と帯同の連動

### 7-1. 基本ルール

放課後にダンジョン探索を行う場合、**授業中のメンバーは帯同不可**。

```
平日の時間帯:
  [朝] → [1コマ目] → [2コマ目] → [3コマ目] → [4コマ目] → [5コマ目(任意)]
                                                              ↓
                                                    [放課後] (5コマ目なしの場合)
                                                              ↓
                                                    [夜]
```

### 7-2. 帯同可否の判定

| 条件 | 放課後帯同 | 夜帯同 |
|------|----------|-------|
| 4コマ以下の日 | 可 | 可 |
| 5コマ目を取っている日 | **不可** | 可 |
| 土曜日 | 可（終日自由） | 可 |
| 日曜日 | 可（特殊イベント枠だが基本自由） | 可 |
| 長期休暇 | 可 | 可 |
| サボり中 | 可（サボった時点から帯同可能） | 可 |

### 7-3. シオンの特殊ルール

- シオンは **2年生**のため、別学年の授業を受けている
- シオンの授業スケジュールはプレイヤーから見えない（固定パターン）
- **5コマ目の日**はシオンも放課後帯同不可
- 土日/長期休暇/夜パートでは全員帯同可能
- シオンの5コマ目の日は週に2回（火曜・木曜固定）

## 8. 教師NPC一覧

### 8-1. ガロン（戦闘実技）

- **性格**: 豪快で厳格な武人。生徒にも容赦なし。ただし努力する者には温かい
- **信頼度報酬**: Lv1=特殊技「烈風斬」習得書, Lv2=戦闘実技成長率さらに+5%, MAX=ユニーク防具「闘神の腕甲」
- **イベント連携**: カイトの過去に関する情報を持つ（兄ハヤトの在学時の話）

### 8-2. ダリオ（戦闘理論 / 商学 兼任）

- **性格**: 穏やかな知識人。理論派で分析好き。商才にも長ける
- **信頼度報酬**: Lv1=戦術書「防御の極意」, Lv2=ショップ割引+3%(累積), MAX=隠しメニュー解放
- **イベント連携**: 経済イベント、商学成績優秀者への特別課題

### 8-3. エレナ（魔法実技）

- **性格**: 若く情熱的な魔法使い。生徒と年齢が近く親しみやすい
- **信頼度報酬**: Lv1=魔法書「精霊の加護」, Lv2=MATK成長率さらに+5%, MAX=上位魔法「メテオ」習得書
- **イベント連携**: ミオの才能を見出すイベント、魔法暴走イベントのフォロー

### 8-4. カルロス（魔法理論）

- **性格**: 温厚で生徒思いの担任教師（表の顔）。実際は教団エクリプスの幹部
- **信頼度報酬**: Lv1=魔法理論の補講（MDEF+3%）, Lv2=「信頼の証」（ストーリーフラグ）
- **イベント連携**: 裏切りイベント（2章後半で正体発覚）。信頼MAXは到達不可
- **特殊**: 2章後半以降は代理教師が担当。カルロスの信頼度はリセットされない（裏切りフラグに使用）

### 8-5. グレイヴ（戦術 / 特別講師）

- **性格**: 寡黙で威厳がある学園長。エルトの過去を背負う
- **信頼度報酬**: Lv1=戦術書「先手必勝」, Lv2=隠しダンジョン情報, MAX=エルトの過去イベント全解放
- **イベント連携**: 隠しダンジョン解放、エルト関連イベント、END4/5分岐に関わる情報提供

### 8-6. ニコラ（錬金術）

- **性格**: 偏屈だが根は善良な老錬金術師。実験好きで爆発事故が多い
- **信頼度報酬**: Lv1=基本レシピ5種, Lv2=上級レシピ3種+成功率+10%, MAX=完全レシピ（成功率100%）解放
- **イベント連携**: 錬金術システム全体に関与。信頼度が錬金成功率に直結

### 8-7. マルガリータ（歴史）

- **性格**: 博識で穏やかな老婦人。語り口が上手く、授業は面白い
- **信頼度報酬**: Lv1=歴史書「古代の記録」（情報フラグ1個分ヒント）, Lv2=追加情報フラグヒント, MAX=flag_academy_secretに関する核心情報
- **イベント連携**: 世界観の深掘り。魔王の器の歴史、封印の真実に関する情報源

### 8-8. セラフィーナ（医学）

- **性格**: 慈愛深い修道女。学園の保健室も兼任
- **信頼度報酬**: Lv1=回復アイテム効果+5%(累積), Lv2=蘇生スキル習得書, MAX=ユナの隠しスキル「聖域」解放条件
- **イベント連携**: ユナの成長イベント。侵蝕に関する医学的知見

### 8-9. ダリオ（商学）

> ※ダリオは戦闘理論と商学を兼任。商学固有の信頼度報酬は以下の通り。

- 商学固有報酬: 商学成績S → 隠しメニュー解放の片方の条件
- 商学テストは「値段当て」「交渉シミュレーション」等の特殊形式

## 9. 実装上の注意

### 9-1. データ構造案

```csharp
[System.Serializable]
public class SubjectData
{
    public SubjectType type;          // 9科目のenum
    public string teacherName;
    public StatBonus[] statBonuses;   // 成長率ボーナス
    public int maxSlots;              // 1日の最大コマ数への寄与
}

[System.Serializable]
public class EnrollmentData
{
    public int chapter;
    public List<SubjectType> enrolledSubjects; // 最大7
    public Dictionary<SubjectType, float> attendanceRate;
    public Dictionary<SubjectType, int> testScore;
}

[System.Serializable]
public class CompanionSchedule
{
    public string characterId;
    public List<SubjectType> enrolledSubjects;
    public bool[] hasFifthPeriod; // 曜日ごと (月〜金)
}
```

### 9-2. 授業スケジュール生成

```
1. 章開始時にプレイヤーが履修登録
2. 登録科目を1週間(月〜金)に分配
   - 各科目は週1〜2回
   - 1日最大5コマ（5コマ目は任意）
3. NPCの履修はシード固定乱数で決定
4. 帯同可否フラグを日単位で生成
```
