## 2. 各ダンジョンの詳細

### 2-1. 翠風の森（15層）

- **雰囲気**: 学園から最も近い訓練用ダンジョン。緑が生い茂り、光が差し込む
- **ギミック**: 毒沼(3層〜), ツタを切って道を開く, 隠し通路(壁の模様が目印)
- **ボス(15層)**: 森の守護者グリムウルフ（Lv20）— 大型の魔狼
- **中ボス(8層)**: 大蜘蛛ウェブクイーン（Lv12）

### 2-2. 風鳴りの塔（15層）

- **雰囲気**: 廃墟の魔法塔。風の魔法で動く浮遊足場
- **ギミック**: 風の流れに乗って移動, 落下するとフロア最初から, 風向き切替スイッチ
- **ボス(15層)**: 暴風の精霊ストームバード（Lv30）
- **中ボス(8層)**: 風使いの亡霊ゲイル（Lv22）

### 2-3. 氷結洞窟（12層）

- **雰囲気**: 永久凍土の洞窟。壁面に古代の氷像が並ぶ
- **ギミック**: 氷の床で滑走(方向キーで操作不可), 氷壁を溶かす火属性攻撃, 凍結トラップ(触れると3ターン行動不能)
- **ボス(12層)**: 氷竜フロストドレイク（Lv35）
- **中ボス(6層)**: アイスゴーレム（Lv28）

### 2-4. 鬼気の洞窟（15層）

- **雰囲気**: 教団が利用していた地下通路。瘴気が充満
- **ギミック**: 瘴気エリア(毎歩HP-1%、解毒剤で一時無効化), 罠の頻度が高い, 隠しボス部屋(10層以上で条件付き)
- **ボス(15層)**: 瘴気の王ミアズマロード（Lv45）
- **中ボス(8層)**: 堕ちた騎士ダークナイト（Lv38）
- **特殊**: 10層以上到達 → 隠しダンジョン解放条件の1つ

### 2-5. 忘却遺跡（20層）

- **雰囲気**: 太古の文明の遺跡。文字が刻まれた壁、動く石像
- **ギミック**: 石版パズル(3種のシンボルを正しい順序に), 隠し部屋(壁を調べると発見), 記憶消去トラップ(マップが一時的にリセット)
- **ボス(20層)**: 記憶の番人ムネモシュネ（Lv55）
- **中ボス(10層)**: 古代兵器ガーディアン（Lv48）
- **隠し部屋(15層)**: 錬金レシピ「暗黒の呪書」素材

### 2-6. 竜の巣窟（15層）

- **雰囲気**: 活火山の内部。溶岩が流れ、熱気が揺らめく
- **ギミック**: 溶岩流(一定間隔で通路を塞ぐ), 耐火装備がないとスリップダメージ, 竜の卵を壊すと増援
- **ボス(15層)**: 炎竜イグニスドラゴン（Lv65）
- **中ボス(8層)**: 溶岩の巨人マグマゴーレム（Lv58）

### 2-7. 大聖堂地下（20層）

- **雰囲気**: 光と闇が交錯する神聖な地下空間。ステンドグラスが不気味に輝く
- **ギミック**: 光/闇スイッチ(フロアの属性を切り替え。光フロアでは闇敵弱体化、逆も然り), 聖水の泉(HP全回復ポイント、5層ごと)
- **ボス(20層)**: 堕天使ルシファリオン（Lv75）
- **中ボス(10層)**: 聖騎士の亡霊パラディンゴースト（Lv68）
- **中ボス(15層)**: 闇の司祭ダークプリースト（Lv72）

### 2-8. 封印の間（15層）

- **雰囲気**: 学園の最深部。封印の紋章が脈動し、闇が具現化する
- **ギミック**: 侵蝕加速(レイの侵蝕が通常の2倍速度で進行), 封印パズル(紋章を正しい順序で解除), 闇の分身(レイのコピーが敵として出現)
- **ボス(15層)**: **エルト（Lv200）** — 本作最強の敵
- **中ボス(8層)**: 封印の番犬ケルベロス（Lv80）
- **特殊**: エルトはシンボルエンカウント。**スルー可能、逃走可能**

### 2-9. 深淵 / 裏深淵（50層+19層）

- **解放条件**: NG+限定
- **雰囲気**: 世界の最深部。形のない暗黒空間。階層ごとに風景が変化
- **ギミック**: ランダム生成要素あり(レイアウト変化), 全ギミックの複合, セーブポイント10層ごと
- **ボス(深淵50層)**: **モルドレッド（Lv150）** — 魔王の先代の騎士
- **ボス(裏深淵19層)**: **ゼノン（Lv180）** — 教団の創始者
- **裏深淵**: 深淵50層クリア後に解放。19層構成

## 3. 探索コスト

### 3-1. 1日の探索可能層数

授業のコマ数によって放課後の探索可能層数が変動する。

| 条件 | 放課後探索可能層数 | 備考 |
|------|-----------------|------|
| 授業0コマ（土日/長期休暇） | **4層** | 最大効率 |
| 授業3コマ以下 | **3層** | サボりで実質コマ減可能 |
| 授業4コマ | **2層** | — |
| 授業5コマ | **0層（探索不可）** | 放課後なし |

### 3-2. 夜間探索

- 夜パートで追加 **+1層** 探索可能
- 夜間の敵は **1.5倍の強さ**（HP/ATK ×1.5）
- 夜間探索中は寮イベント・SLイベントが発生しない

### 3-3. 1日の最大探索層数

```
最大 = 放課後探索層数 + 夜間1層 = 4 + 1 = 5層/日
```

> 土日/休暇で4層 + 夜1層 = 最大5層。

## 4. 帰還とポータル

### 4-1. 帰還方法

| 方法 | 条件 | 効果 |
|------|------|------|
| 帰還の石（アイテム） | 所持していること | 即座にダンジョンから帰還。到達層を記録 |
| ポータル | 5層ごとに設置（5, 10, 15...） | 無料で帰還可能。到達層を記録 |
| 全滅 | パーティ全員HP0 | 学園に強制帰還。到達層は記録される |

### 4-2. 到達層からの再開

- ポータル到達層からダンジョンに再入場可能
- ポータル間の層は再探索が必要
- 帰還の石使用時は直近のポータル層から再開（例: 7層で使用 → 5層から再開）

## 5. エンカウント方式

### 5-1. ハイブリッドエンカウント

通常フロアとボス前で異なるエンカウント方式を採用。

| フロア種別 | 方式 | エンカウント率 |
|-----------|------|-------------|
| 通常フロア（浅層: 1〜5層） | ランダムエンカウント | 5%/歩 |
| 通常フロア（中層: 6〜10層） | ランダムエンカウント | 6%/歩 |
| 通常フロア（深層: 11層〜） | ランダムエンカウント | 8%/歩 |
| ボス前フロア | シンボルエンカウント | — |
| 中ボスフロア | シンボルエンカウント | — |

### 5-2. 先制・奇襲

| 状況 | 効果 |
|------|------|
| 通常遭遇 | 通常通りATBゲージ0から開始 |
| 先制攻撃 | 味方全員ATBゲージ50%から開始 |
| 敵の奇襲 | 敵全員ATBゲージ50%から開始 |
| シンボル背後接触 | 先制攻撃確定（中ボスのみ。ボスは不可） |

**重要ルール**:
- **ボスには奇襲不可**（正面からのみ接触可能）
- **中ボスは背後から先制可能**
- ランダムエンカウントの先制/奇襲判定: 先制15%、奇襲10%、通常75%（SPDで補正）

## 7. エルト戦（封印の間15層）

### 7-1. 基本情報

- **レベル**: 200（本作最強）
- **エンカウント**: シンボル（封印の間最深部に佇んでいる）
- **スルー可能**: 横を通り抜けてイベントのみ見ることも可能
- **逃走可能**: 戦闘開始後も逃走コマンドで離脱可

### 7-2. 戦闘仕様概要

- 全属性攻撃（1ターンごとに変化）
- 吸収属性がランダム変化（1ターンごと）
- 即死級全体攻撃「虚無の終焉」
- HP自己再生（毎ターン3%回復）
- 「魔蝕の共鳴」: レイの侵蝕+20
- HP50%以下で「封印崩壊」（全ステ1.5倍）
- HP20%以下で「エルトの残光」（1ターン攻撃停止）

### 7-3. 撃破報酬

- ユニークアクセサリー「エルトの形見」（侵蝕許容値+15）
- グレイヴの隠しイベント発生
- 実績「30年の約束」解放

## 9. ダンジョン内の共通ルール

### 9-1. マップ構造

- 各層は固定マップ（ランダム生成なし。深淵の一部を除く）
- 宝箱: 通常宝箱（復活なし）+ 隠し宝箱（壁調べで発見）
- ショートカット: 一度開通すれば永続（レバー・スイッチ）

### 9-2. セーブ

- ダンジョン内セーブは不可（Nightmare以外）
- ポータル地点でオートセーブが入る
- 中断セーブはダンジョン内でも使用可能（使用後消失）

### 9-3. パーティ編成

- ダンジョン内でのパーティ編成変更は**不可**
- 入場前にメニューから編成を行うこと
- バトル中の入れ替え（FF10式）は可能（party-system.md参照）

### 9-4. 敵レベルスケーリング

- 敵レベルは固定（レベルスケーリングなし）
- 各ダンジョン・各層に適正レベル帯が設定されている
- 適正レベル帯の目安:

| ダンジョン | 適正Lv帯 |
|-----------|---------|
| 翠風の森 | Lv5〜20 |
| 風鳴りの塔 | Lv15〜30 |
| 氷結洞窟 | Lv25〜35 |
| 鬼気の洞窟 | Lv30〜45 |
| 忘却遺跡 | Lv40〜55 |
| 竜の巣窟 | Lv50〜65 |
| 大聖堂地下 | Lv60〜75 |
| 封印の間 | Lv70〜200 |
| 深淵/裏深淵 | Lv80〜180 |
