# ソウルリンク（Soul Link）システム仕様書

## 概要

レイと仲間10人の絆レベルを表すシステム。信頼値（Bond Points）を貯めてレベルを上げ、デュアルアーツの解放・ストーリーイベント・エンディング分岐に影響する。

---

## 3. 信頼値の上げ方

### 会話選択（メイン）

| シチュエーション | 信頼値変動 | 備考 |
|-------------|-----------|------|
| メインストーリー選択肢（好感度UP） | +3 ~ +8 | 選択肢による |
| メインストーリー選択肢（好感度DOWN） | -2 ~ -5 | 悪い選択 |
| サブイベント選択肢 | +2 ~ +5 | |
| SLイベント中の選択肢 | +3 ~ +10 | SLイベントは信頼値大きめ |

### バトル関連

| 条件 | 信頼値変動 | 備考 |
|------|-----------|------|
| 一緒にバトル（勝利） | +2 | バトルメンバーに含まれていた場合 |
| デュアルアーツ使用 | +3 | パートナーとして使用 |
| 味方を回復/バフ | +1 | 対象キャラへ |
| 味方が戦闘不能 | -1 | そのキャラとの信頼値 |
| かばう（オートプロテクト発動） | +2 | かばった/かばわれた双方 |

### SLイベント

- 各SLレベルに到達した時にイベント発生
- イベント完了で追加信頼値を獲得（イベント内容は後述）

### サブイベント

- 各キャラに専用サブイベントが存在
- サブイベント完了: +5 ~ +10

## 4. シオンの特殊ルール

### 信頼値が上がりやすい設計

- シオンのサブイベントは他キャラより多い（8個 vs 通常4~5個）
- シオン関連の選択肢は信頼値変動が大きい（+5~+10）
- 理由: END5のシオン救出条件を達成させやすくするため

### 2章11週での凍結

- 2章11週目にシオンが闇落ち → SLシステムからシオンとのSLが**凍結**
- 凍結中:
  - 信頼値の増減なし
  - SLイベント発生なし
  - デュアルアーツ使用不可
  - UIに「凍結中」と表示

### END5での復活

- END5ルートでシオンを救出成功 → SL凍結解除
- 解除時の信頼値 = 凍結時点の値を維持
- カルロス戦でシオンがパーティに復帰
- シオンとのデュアルアーツ10種が自動解放

### END5のシオン救出条件

```
シオン信頼値 >= 50pt
かつ 情報フラグ 8/8
かつ 全メインキャラ（ユナ/ミオ/カイト）信頼値 >= 80pt
```

- シオン信頼値50pt以上 = SL Lv2（信頼）以上

## 6. SLレベルによる戦闘効果

| SLレベル | 効果 |
|----------|------|
| Lv1 | デュアルアーツ解放（該当ペア） |
| Lv2 | オートプロテクト: 20%の確率でレイへの致死攻撃をそのキャラが庇う |
| Lv3 | 連携追撃: 20%の確率で通常攻撃後に追撃（そのキャラの通常攻撃の50%ダメージ） |
| MAX | 絆ダメージボーナス: レイとそのキャラが同じバトルに参加時、両者のダメージ+10% |

## 7. NG+引き継ぎ

### 信頼値ボーナス

- NG+開始時に全キャラの信頼値を0にリセット
- **累積ボーナス: 毎周+5pt（上限20pt）**
  - 2周目: 全キャラ初期信頼値+5
  - 3周目: 全キャラ初期信頼値+10
  - 4周目: 全キャラ初期信頼値+15
  - 5周目以降: 全キャラ初期信頼値+20（上限）

### 引き継ぎ対象

| 項目 | 引き継ぎ |
|------|---------|
| 信頼値 | リセット（NG+ボーナス加算） |
| SLイベント閲覧フラグ | 引き継ぎ（回想で閲覧可能） |
| デュアルアーツ解放状態 | リセット（再度SL Lv1到達で解放） |
| バトル共闘回数 | リセット |
| シオン凍結状態 | リセット |

## 8. デュアルアーツとの連動

### 解放条件の関係

| SLレベル | デュアルアーツへの影響 |
|----------|---------------------|
| Lv1到達 | 基本デュアルアーツ解放 |
| Lv2到達 | SL倍率 x1.2 適用 |
| Lv3到達 | SL倍率 x1.5 適用 |
| MAX到達 | SL倍率 x2.0 適用 |

### パートナー間のSL

- デュアルアーツのSL倍率は**低い方のSLレベル**を参照
  - レイとユナ: レイ-ユナ間SLのレベル
  - ユナとミオ: ユナ-ミオ間SLのレベル
- 仲間同士のSLはレイとの個別SLとは別管理
- 仲間同士のSLは「バトル共闘回数」で自動上昇（専用SLイベントなし）

### 仲間同士の信頼値

| バトル共闘回数 | 仲間間SLレベル |
|-------------|-------------|
| 0~9回 | None |
| 10~24回 | Lv1 |
| 25~49回 | Lv2 |
| 50~79回 | Lv3 |
| 80回以上 | MAX |

## 9. 実装メモ

### データ構造の拡張

現在の`SoulLinkSystem.cs`から以下の拡張が必要:

```
SoulLinkSystem (拡張版)
├── RayBonds: Dictionary<CharacterId, int> — レイとの信頼値(10人)
├── PartyBonds: Dictionary<(CharacterId, CharacterId), int> — 仲間間の共闘回数
├── ShionFrozen: bool — シオン凍結フラグ
├── NgPlusBonusPerChar: int — NG+累積ボーナス
│
├── AddBondPoints(CharacterId, int) — 信頼値加算
├── AddBattleCoopCount(CharacterId, CharacterId) — 共闘回数加算
├── FreezeShion() — シオンSL凍結
├── UnfreezeShion() — END5でのシオンSL凍結解除
├── GetPartyBondLevel(CharacterId, CharacterId) → SoulLinkLevel — 仲間間のSL
├── GetDualArtsSLMultiplier(CharacterId, CharacterId) → float — DA用SL倍率
└── ApplyNgPlusBonus(int cycleCount) — NG+ボーナス適用
```

### CharacterId enumの拡張

```csharp
public enum CharacterId
{
    Ray, Yuna, Mio, Kaito, Shion,
    Rin, Vein, Mel, Jin, Setsuna, Renji  // 追加
}
```

### TrustSystemとの関係

- `TrustSystem`（信頼ポイント、4人: ユナ/ミオ/カイト/シオン）はエンディング分岐用
- `SoulLinkSystem`（ソウルリンク、10人）は戦闘・DA・SLイベント用
- 信頼値の上昇タイミングが一部重複するため、両システムを連携させる
  - 会話選択でTrust+5 → 同じキャラのSL BondPointsにも+5
  - ただしTrustは4人のみ、SLは10人対応
