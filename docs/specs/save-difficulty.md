# セーブ・難易度・NG+ 仕様書

## 1. セーブシステム

### 1-1. セーブ枠

| 種類 | 枠数 | 説明 |
|------|------|------|
| 手動セーブ | 20枠 | プレイヤーが任意のタイミングで保存 |
| オートセーブ | 1枠 | 自動保存（上書き） |
| 中断セーブ | 1枠 | 使用後に消失する一時保存 |
| **合計** | **22枠** | |

### 1-2. 手動セーブ

- メニューから「セーブ」を選択して保存
- 同じ枠に上書き可能（確認ダイアログあり）
- セーブデータにはサムネイル + 日時 + 場所 + プレイ時間 + 章情報を表示
- 難易度によりセーブ可能タイミングに制限あり（後述）

### 1-3. オートセーブ

- 以下のタイミングで自動保存:
  - 時間帯の切り替わり（朝→授業、授業→放課後、放課後→夜）
  - ダンジョンのポータル到達時
  - 章の開始時
  - 重要イベントの開始前
- 1枠のみ。最新のオートセーブで常に上書き

### 1-4. 中断セーブ

- ゲーム中断時に使用する一時セーブ
- **ロードすると消失**（1回限り）
- ダンジョン内でも使用可能
- Nightmareでは中断セーブも特殊ルールあり（後述）

## 2. 難易度システム

### 2-1. 4段階の難易度

ゲーム開始時に選択。**ゲーム中いつでも変更可能**（Nightmareのみ変更不可）。

| 難易度 | 対象プレイヤー |
|--------|-------------|
| Easy | ストーリーを楽しみたい人向け |
| Normal | 標準的なRPG経験者向け |
| Hard | やり応えを求めるプレイヤー向け |
| Nightmare | 高難度チャレンジャー向け。特殊ルールあり |

### 2-2. 難易度別パラメータ

| パラメータ | Easy | Normal | Hard | Nightmare |
|-----------|------|--------|------|-----------|
| 敵HP倍率 | ×1.0 | ×1.5 | ×2.0 | ×4.0 |
| 敵ATK倍率 | ×1.0 | ×1.5 | ×2.0 | ×4.0 |
| 獲得EXP倍率 | ×1.2 | ×1.0 | ×0.9 | ×0.75 |
| 獲得G倍率 | ×1.2 | ×1.0 | ×1.0 | ×0.8 |
| アイテムドロップ率 | +10% | ±0% | ±0% | -10% |

### 2-3. セーブ制限

| 条件 | Easy | Normal | Hard | Nightmare |
|------|------|--------|------|-----------|
| フィールドでのセーブ | いつでも | いつでも | 時間帯切替時のみ | セーブポイントのみ |
| ダンジョン内セーブ | いつでも | ポータルのみ | ポータルのみ | 不可 |
| 中断セーブ | あり | あり | あり | あり（特殊ルール） |
| オートセーブ | あり | あり | あり | あり |

### 2-4. 全滅ルール

| 条件 | Easy | Normal | Hard | Nightmare |
|------|------|--------|------|-----------|
| 通常戦闘で全滅 | HP1で復活+戦闘離脱 | 最後のセーブに戻る | 最後のセーブに戻る | **章の最初に戻る** |
| ボス戦で全滅 | 再挑戦（HP全回復） | 最後のセーブに戻る | 最後のセーブに戻る | **章の最初に戻る** |
| シオン戦で全滅 | — | — | — | **END6確定** |

### 2-5. 回復ルール

| 条件 | Easy | Normal | Hard | Nightmare |
|------|------|--------|------|-----------|
| 戦闘後HP/MP | 全回復 | HP全回復, MP据え置き | 据え置き | 据え置き |
| フィールド回復 | 宿屋/アイテム/ポータル | 宿屋/アイテム/ポータル | 宿屋/アイテムのみ | 宿屋のみ(ポータル回復なし) |
| 戦闘不能の自然回復 | 戦闘後にHP1で復活 | なし | なし | なし |

## 3. Nightmare特殊ルール

### 3-1. 全滅ペナルティ

- **通常戦闘・中ボスで全滅**: 章の最初に戻される（章内の進行がリセット）
- **ボス戦で全滅**: 章の最初に戻される
- **シオン第2形態戦で全滅**: **END6が確定**（全滅エンド直行）

### 3-2. 女神の加護

- **1回/周回**のみ使用可能な復活システム
- 全滅時に「女神の加護を使いますか？」と選択肢
- 使用すると全滅ペナルティを無効化し、戦闘開始時の状態に戻る
- 1周回で1回のみ。NG+でも1回/周回にリセット
- シオン戦での全滅には使用不可（END6確定は覆せない）

### 3-3. 中断セーブの特殊処理

- 中断セーブのロード後、**中断セーブデータが完全消失**（通常通り）
- ただしNightmareでは、**全滅して章の最初に戻った場合も中断セーブが消える**
- つまり中断セーブは「安全な中断」のためだけに使え、リトライの保険にはならない

### 3-4. 難易度変更制限

- Nightmareは**ゲーム開始後に変更不可**
- 他の難易度からNightmareへの変更も不可
- Nightmareは最初から選択した場合のみプレイ可能

## 4. エンディング分岐と難易度の関係

### 4-1. END2/3/6 の3章終了（Nightmareのみ）

- **Nightmareのみ**: END2, END3, END6 の条件を3章で満たした場合、3章終了時点でそのエンディングに直行
- 他の難易度ではこれらのENDの条件を満たしても、グレイヴの蘇生/魔王の暴走イベントで4章に進行する

| エンディング | Easy〜Hard | Nightmare |
|-------------|-----------|-----------|
| END2(トゥルー) | 3章で条件満たしても4章へ（蘇生） | 3章で終了 |
| END3(シオンルート) | 3章で条件満たしても4章へ（暴走） | 3章で終了 |
| END6(全滅) | 3章で条件満たしても4章へ | 3章で終了 |
| END1/4/5 | 通常通り4章で判定 | 通常通り4章で判定 |

### 4-2. END6到達後の共通処理

END6（全滅エンド）に到達した場合、**全難易度共通**で以下の処理が発生:

```
[END6エンディング再生]
    ↓
[暗転]
    ↓
「この結末を覆しますか？」
    ↓
  はい → NG+開始の導線（タイトル画面にNG+オプション追加）
  いいえ → タイトル画面に戻る（NG+オプションは追加される）
```

- END6以外のエンディングでもNG+は開始可能（4章終了後）
- END6はNG+への最も強い動機付けとして機能する

## 5. NG+（New Game Plus）

### 5-1. NG+開始条件

- **4章終了後**（どのエンディングでも可）
- タイトル画面に「NEW GAME+」オプションが追加される
- クリア済みセーブデータを選択してNG+を開始

### 5-2. 引き継ぎ項目

| 項目 | 引き継ぎ内容 |
|------|------------|
| ユニーク装備 | 全て引き継ぎ（SL報酬武器、汎用ユニーク、NG+限定ユニーク） |
| SP（スキルポイント） | 累積 +10SP/周回 を追加して引き継ぎ |
| 隠しスキル | 解放済みの隠しスキルは引き継ぎ（ツリーは振り直し） |
| 実績・図鑑 | 全て引き継ぎ |
| 既読フラグ | 全会話の既読情報を引き継ぎ（スキップ用） |
| 信頼値ボーナス | 前周の信頼値の結果に関わらず、全キャラ信頼値+5で開始（上限20） |
| 深淵ダンジョン | NG+で自動解放 |

### 5-3. リセット項目

| 項目 | リセット内容 |
|------|------------|
| キャラクターレベル | 全員Lv1に戻る |
| 通常装備 | 全て消失（ユニーク以外） |
| 消耗アイテム | 全て消失 |
| ゴールド | 0Gに戻る |
| ストーリーフラグ | 全リセット（情報フラグ8種含む） |
| 侵蝕値 | 0に戻る |
| ソウルリンク | Lv0に戻る |
| カレンダー | 1章1週に戻る |
| スキルツリー | リセット（SPは引き継ぎ分を含めて再振り分け） |
| 授業ノート | リセット |
| 教師信頼度 | リセット |

### 5-4. SP引き継ぎの詳細

```
NG+開始時のSP = 前周終了時の未使用SP + 10（周回ボーナス）
```

- スキルツリーはリセットされるが、SPの総量は増える
- 2周目は前周で稼いだSP + 10、3周目はさらに + 10...
- これにより周回ごとにより多くのスキルを取得可能
- 隠しスキルはツリー外で解放済みフラグが引き継がれるため、SP不要で即使用可能

### 5-5. 信頼値+5ボーナスの仕様

- 全仲間キャラの初期信頼値が+5で開始
- **上限20**: NG+を何周しても初期信頼値は最大20まで
- 4周で上限到達（0→5→10→15→20）
- これによりSLイベントの早期発生やEND5条件の達成が容易になる

## 6. NG+でのEND1

### 6-1. 隠し操作

通常のEND1は「学園に行かない」を選ぶだけだが、NG+では追加の隠し操作が可能。

- NG+でEND1を選択した場合、**OPムービーはスキップされない**
- OPの特定タイミングで隠しコマンド入力が可能（上上下下左右左右）
- 成功すると **特殊END1「もう一つの日常」** に分岐
- 内容: レイが学園に行かなかった世界線の「その後」が描かれる短編エピローグ
- 実績「見えなかった道」が解放

### 6-2. OP非スキップの理由

- NG+でも END1 選択時のOPは既読スキップの対象外
- 隠しコマンド入力の機会を保証するため
- END1以外のルートではOPスキップ可能

## 7. 周回スキップ機能

### 7-1. スキップ可能な要素

| 要素 | スキップ | 条件 |
|------|---------|------|
| 会話テキスト | 可（早送り+オートスキップ） | 既読フラグが立っている会話のみ |
| 授業演出 | 可 | NG+で既読の授業 |
| イベントシーン | 可（シーンスキップ） | 既読のイベントのみ |
| ダンジョン戦闘 | **不可** | — |
| ボス戦 | **不可** | — |
| 章末テスト | **不可**（内容は同一だが操作は必要） | — |
| END1 OP | **不可** | — |

### 7-2. スキップUI

- 既読テキスト表示中は画面右上に「SKIP」アイコンを表示
- L2ボタン（またはSキー）長押しで高速スキップ
- メニューから「既読オートスキップ」ON/OFFを設定可能
- オートスキップON時は既読テキストを自動で飛ばし、未読テキストで停止

## 8. 難易度別の推奨プレイスタイル

| 難易度 | 推奨 | プレイ時間目安 |
|--------|------|-------------|
| Easy | 初回プレイ（ストーリー重視）。戦闘は最低限 | 25〜35時間 |
| Normal | 初回〜2周目。バランスの取れた体験 | 35〜50時間 |
| Hard | 2周目以降。戦術と育成の工夫が必要 | 50〜70時間 |
| Nightmare | NG+推奨。高い準備と知識が前提 | 60〜100時間 |

## 9. 実装上の注意

### 9-1. セーブデータ構造

```csharp
[System.Serializable]
public class SaveData
{
    [Header("メタ情報")]
    public int slotIndex;           // 0-19:手動, 20:オート, 21:中断
    public string timestamp;
    public float playTime;
    public string locationName;
    public int chapter;
    public int week;

    [Header("難易度")]
    public Difficulty difficulty;
    public bool goddessBlessing;    // 女神の加護使用済みフラグ

    [Header("NG+")]
    public int ngPlusCount;         // 周回数
    public List<string> uniqueEquipIds;  // ユニーク装備ID
    public int carryOverSP;         // 引き継ぎSP
    public List<string> unlockedHiddenSkills;
    public Dictionary<string, bool> readFlags; // 既読フラグ
    public int trustBonus;          // 信頼値ボーナス（上限20）

    // ... その他のゲームステート
}
```

### 9-2. 難易度変更の処理

```csharp
public bool TryChangeDifficulty(Difficulty newDifficulty)
{
    if (currentDifficulty == Difficulty.Nightmare) return false;
    if (newDifficulty == Difficulty.Nightmare) return false;
    currentDifficulty = newDifficulty;
    ApplyDifficultyModifiers();
    return true;
}
```

### 9-3. Nightmare全滅処理

```csharp
public void OnPartyWipe()
{
    if (difficulty == Difficulty.Nightmare)
    {
        if (IsShionBattle() && !goddessBlessing)
        {
            TriggerEnding(EndingType.END6);
            return;
        }

        if (!goddessBlessing)
        {
            ResetToChapterStart();
            DeleteInterruptSave();
        }
        else
        {
            // 女神の加護を使うか確認
            ShowGoddessGracePrompt();
        }
    }
    else
    {
        LoadLastSave();
    }
}
```
