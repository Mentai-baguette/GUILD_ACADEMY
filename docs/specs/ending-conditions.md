## 2. 各END詳細仕様

### END1: 裏ハッピー「平穏な日常」

**条件:**
- ゲーム冒頭のOPイベントで「学園に行かない」を選択
- この選択肢は通常の選択UIとは異なり、**隠し操作**で出現する
  - 方法: OPイベント中に「学園に行こう」の選択肢が表示された状態で**10秒間何も操作しない**
  - 10秒後にテキストが変化し「...やっぱり行くのやめようかな」→「学園に行かない」が出現

**判定フロー:**
```
OPイベント開始
  ↓
選択肢「学園に行こう」表示
  ↓
10秒無操作検知 → 隠し選択肢出現
  ↓
「学園に行かない」選択
  ↓
END1 エンディングシーケンス再生
```

**結果:**
- レイは学園に行かず、ユナ・ミオと平穏な日常を過ごす
- 魔王の器の覚醒は起きない
- 短いエピローグ（30秒程度）の後、スタッフロール
- 実績「もう一つの選択」解放

**備考:**
- 全ENDの中で最も短い（プレイ時間1分未満）
- NG+でも選択可能
- このENDだけでは4章には進まない

### END2: 雪辱「レイの死」

**条件:**
- 3章12週のシオン第1形態戦（光属性フェーズ）でプレイヤーが**敗北**する
- 具体的判定: レイのHP=0 かつ 幼馴染3人（ユナ/ミオ/カイト）全員が気絶状態

**判定フロー:**
```
シオン第1形態戦 開始
  ↓
戦闘中にレイHP=0 + ユナ/ミオ/カイト全員気絶
  ↓
END2 イベントシーケンス開始
  ↓
[Nightmare以外] END2途中カット → 蘇生イベント → 4章へ
[Nightmare] 4章確定のためEND2到達不可（第1形態戦で敗北してもリトライ強制）
```

**結果:**
- レイは死亡するが、魔王の器としての意識が残る
- シオンは「正義を果たした」と確信するが、心にわだかまりが残る
- エンディング: レイの意識が闇の中で漂うシーン

**4章への遷移（Nightmare以外）:**
- END2のエンディングシーケンス途中で画面が歪む
- グレイヴが禁術「魂魄蘇生」を発動 → レイ蘇生
- 蘇生直後の1週（体調回復） + 準備1週 = **計2週**で4章開始
- 蘇生コスト: グレイヴのHPが最大値の半分に固定される（4章中のイベント戦闘に影響）

### END3: 壊れた正義「シオンの崩壊」

**条件:**
- シオン第1形態戦に勝利した後、シオン第2形態戦（闇属性フェーズ）でプレイヤーが**敗北**する
- 具体的判定: パーティ全員のHP=0（レイ含む4人全滅）かつ侵蝕率90%未満

**判定フロー:**
```
シオン第2形態戦 開始
  ↓
パーティ全滅（侵蝕90%未満）
  ↓
END3 イベントシーケンス開始
  ↓
[Nightmare以外] END3途中カット → 蘇生/魔王暴走イベント → 4章へ
[Nightmare] END3到達不可（第2形態戦で敗北してもリトライ強制）
```

**結果:**
- 操られたシオンがレイたち4人を殺害してしまう
- カルロスの操りが解けた後、シオンは自分がやったことを理解して精神崩壊
- エンディング: 壊れたシオンが虚空を見つめるシーン

**4章への遷移（Nightmare以外）:**
- END3シーケンス途中で魔王の力が暴走 → レイの魂が拒絶反応を起こす
- グレイヴの介入 + ユナの聖魔法で全員蘇生
- 蘇生1週 + 準備1週 = **計2週**で4章開始

### END4: Pyrrhic Victory「空虚な勝利」

**条件:**
- シオン第2形態戦に**勝利**
- かつ、シオン救出条件を**満たしていない**
- かつ、END4.5の隠し条件も**満たしていない**

**救出条件（END5に必要な全条件）:**
- 情報フラグ: 8/8（全フラグON）
- 信頼値: ユナ/ミオ/カイト/シオン全員80以上
- シオンSL: +50以上

**判定フロー:**
```
シオン第2形態戦 勝利
  ↓
救出条件チェック
  ↓
全条件未達（END5不可）
  ↓
END4.5隠し条件チェック
  ↓
隠し条件未達
  ↓
END4 イベントシーケンス
  ↓
4章（夏休み2: 2週間の準備期間）→ カルロス戦
  ↓
カルロス戦勝利 → END4エピローグ
カルロス戦敗北 → END6へ分岐
```

**結果:**
- シオンを倒すことには成功するが、闇の力からの解放はできない
- カルロスがシオンの心臓を潰して殺害し、撤退
- 4章でカルロスを追い詰めて倒すが、シオンの命は戻らない
- エンディング: シオンの墓前に立つレイたち。勝利したが大切なものを失った

**4章の流れ:**
1. 夏休み2（2週間）: 準備期間。装備強化・レベリング・情報収集
2. 4章本編（12週）: カルロスの拠点を目指す
3. カルロス戦: レイ+ユナ+ミオ+カイトの4人で挑む

### END4.5: 最後に届いた光「グレイヴの約束」

**条件:**
- シオン第2形態戦に**勝利**
- 救出条件は**未達**（END5にはならない）
- **隠し条件を満たしている**

**隠し条件（以下の全てを満たす）:**
1. シオン信頼値が **30〜49** の範囲（50以上だとEND5ルートへ行けるため該当しない）
2. 以下のいずれかを満たす:
   - **Aパターン**: グレイヴ追加信頼イベント「剣聖の約束」をクリア済み
   - **Bパターン**: セツナの信頼値が **60以上**

**ルート別の展開:**

| パターン | 条件 | 4章での展開 |
|---------|------|-----------|
| A+Bルート | グレイヴイベントクリア | カルロス戦中、グレイヴが約束通り駆けつける。レイに「エルトの代わりではなく、お前自身を守りに来た」と言う |
| A+Cルート | セツナ信頼60以上 | カルロス戦中、セツナがグレイヴを呼びに行っていた。セツナが「シオン先輩の分まで...」とグレイヴを連れてくる |

**判定フロー:**
```
シオン第2形態戦 勝利
  ↓
救出条件チェック → 未達
  ↓
END4.5隠し条件チェック
  ↓
シオン信頼30〜49 かつ (グレイヴイベント済 or セツナ信頼60+)
  ↓
END4.5フラグON
  ↓
4章（夏休み2: 2週間）→ カルロス戦
  ↓
カルロス戦中にグレイヴ参戦イベント発生
  ↓
カルロス戦勝利 → END4.5エピローグ
カルロス戦敗北 → END6へ分岐
```

**結果:**
- END4と同じくシオンは救えないが、グレイヴの参戦によりカルロス戦が有利になる
- カルロス撃破後、グレイヴがシオンの遺体を抱え「30年前と同じ過ちは繰り返さない」と誓う
- エンディング: END4より希望のある結末。シオンの魂が安らかに昇っていく描写

**グレイヴ追加信頼イベント「剣聖の約束」の発生条件:**
- 3章8週目以降
- グレイヴとの会話回数5回以上
- `flag_vessel_truth`（魔王の器の真実を知る）がON
- 教師室でグレイヴに話しかけると発生
- イベント内容: グレイヴがエルト（30年前の器）の話をし、「今度こそ守る」と約束

### END5: 夜明けの学園「全員生存」

**条件（全て必須）:**
1. 情報フラグ: **8/8**（全フラグON）
2. 信頼値: ユナ/ミオ/カイト **全員80以上**
3. シオンSL: **+50以上**
4. シオン第2形態戦に**勝利**
5. 第2形態戦中に特殊コマンド**「シオンの名を呼ぶ」**が解放され、それを**使用**

**特殊コマンド解放条件:**
- 上記1〜3を満たした状態でシオン第2形態戦に突入
- シオンのHPを50%以下にする
- その時点でコマンドリストに「シオンの名を呼ぶ」が追加される
- このコマンドはレイ専用（レイのATBゲージMAX時のみ選択可能）

**「シオンの名を呼ぶ」発動後の流れ:**
1. レイがシオンの本名を叫ぶ → カットイン演出
2. シオンの内面世界へ突入（ミニイベント）
3. 闇に囚われたシオンの意識に語りかける（選択肢なし、自動進行）
4. シオンが闇から解放される → 第2形態戦終了
5. シオンが正気に戻り、5人パーティでカルロス戦へ

**判定フロー:**
```
シオン第2形態戦 開始
  ↓
情報フラグ8/8 + 信頼全員80+ + シオンSL50+
  ↓ YES
シオンHP50%以下になった時点で「シオンの名を呼ぶ」解放
  ↓
「シオンの名を呼ぶ」使用
  ↓
シオン解放イベント → 5人でカルロス戦
  ↓
カルロス戦勝利 → END5エピローグ
カルロス戦敗北 → END6へ分岐（ただし5人パーティのため難易度は低い）
```

**結果:**
- シオンを闇から救出。5人でカルロスを撃破
- 全員が生存する唯一のエンディング
- エンディング: 夜が明け、学園に平和が戻る。全員が笑顔で校庭に立つシーン
- BGM: bgm_ending_hope
- 実績「夜明けの学園」解放

### END6: 深淵の勝者「世界崩壊」

**条件（以下のいずれか）:**
- シオン第2形態戦でパーティ全員HP=0（全滅）
- シオン第2形態戦中にレイの侵蝕率が**90%以上**に達する
- 4章カルロス戦でパーティ全員HP=0

**判定フロー:**
```
[パターンA: 第2形態戦で全滅]
シオン第2形態戦 → 全滅（侵蝕90%以上含む）
  ↓
END6 イベントシーケンス（3章版）
  ↓
[Nightmare以外] END6途中カット → 蘇生イベント → 4章へ
[Nightmare] END6到達不可（リトライ強制）

[パターンB: カルロス戦で全滅]
4章カルロス戦 → 全滅
  ↓
END6 イベントシーケンス（4章版）→ 真のEND6
```

**結果:**
- パターンA（3章版）: 全員倒れた隙にカルロスがレイから魔王の力を抽出。世界崩壊が始まる
- パターンB（4章版）: カルロスが魔王の力を完全に掌握。世界は闇に覆われる
- エンディング: カルロスが勝利宣言をする中、闇が世界を飲み込むシーン
- BGM: bgm_ending_sorrow

**4章への遷移（3章版、Nightmare以外）:**
- END6シーケンス途中でグレイヴが「まだだ...」と介入
- 時間逆行的な演出（画面が巻き戻る）
- 蘇生1週 + 準備1週 = **計2週**で4章開始
- ペナルティ: レイの最大HP-10%（4章中は回復不可）

## 3. 3章→4章の遷移パターンまとめ

### 遷移パターン一覧

| 3章の結果 | 遷移方法 | 4章開始までの期間 | 特殊条件 |
|----------|---------|-----------------|---------|
| END2到達（Nightmare以外） | グレイヴの禁術で蘇生 | 蘇生1週 + 準備1週 = 2週 | グレイヴHP半減 |
| END3到達（Nightmare以外） | グレイヴ介入+ユナの聖魔法 | 蘇生1週 + 準備1週 = 2週 | 特になし |
| END6到達（Nightmare以外） | グレイヴの時間介入 | 蘇生1週 + 準備1週 = 2週 | レイ最大HP-10% |
| END4/4.5/5ルート | 通常遷移 | 夏休み2（2週間の準備期間） | なし |

### Nightmareモードでの扱い

- Nightmareモードでは**END2/3/6に到達できない**
- シオン第1形態戦・第2形態戦で敗北した場合、リトライが**強制**される
- 侵蝕90%以上になった場合も戦闘リセット（侵蝕80%に戻してリトライ）
- つまりNightmareでは**END4/4.5/5のみ到達可能**
- これにより、Nightmareモードは「必ず4章を攻略する」前提の高難度モードとなる

### 4章の構成

```
[3章終了]
  ↓
[準備期間（2週間）]
  ├── レベリング（訓練場の裏ダンジョン解放）
  ├── 装備強化（最終装備素材収集）
  ├── 情報収集（カルロスの弱点に関するフラグ）
  └── SLイベント最終回（信頼値ブースト最後のチャンス）
  ↓
[4章本編（12週）]
  ├── 1〜4週: カルロスの拠点への道のり（ダンジョン攻略）
  ├── 5〜8週: 拠点内部探索（中ボス戦×3）
  ├── 9〜11週: 最深部への突入（パズル+ボスラッシュ）
  └── 12週: カルロス戦 → エンディング分岐
```

## 6. NG+（New Game Plus）

### NG+解放条件

- **全ルートの4章を終了**すること（END4/4.5/5のいずれか1つ + END6の4章版）
- または、END1〜6の全てを1回以上見ること

### NG+引き継ぎ要素

| 引き継ぐ | 引き継がない |
|---------|------------|
| キャラレベル | ストーリー進行 |
| 装備品（ユニーク装備含む） | 情報フラグ |
| アイテム（消耗品含む） | 信頼値 |
| 図鑑データ | SLレベル |
| 実績データ | 授業ノート |
| ゴールド | 教師信頼度 |
| SP（+10ボーナス累積） | エリア解放状況 |

### END6からのNG+開始

- END6（4章版）到達後、暗転画面で以下のテキストが表示:
  - 「この結末を覆しますか？」
  - [はい] → NG+開始
  - [いいえ] → タイトル画面へ

### Nightmareモード

- NG+でのみ選択可能な最高難度
- END2/3/6には到達不可（リトライ強制）
- END4/4.5/5のみ到達可能
- 敵ステータスx1.6、獲得EXP/Gx0.6
- 隠しボス「エルト」が4章に出現（エルト戦BGM使用）

## 7. 実装上の注意事項

### EndingResolver の判定順序

判定は以下の順序で行うこと（上から順にチェックし、最初に合致したENDを採用）:

```csharp
// 判定優先度（高い順）
1. END1: OP選択肢の判定（最優先）
2. END6: 侵蝕90%以上の判定（戦闘中リアルタイムチェック）
3. END2: 第1形態戦敗北の判定
4. END6: 第2形態戦全滅の判定
5. END3: 第2形態戦敗北（侵蝕90%未満）の判定
6. END5: 救出条件フル達成の判定
7. END4.5: 隠し条件達成の判定
8. END4: 上記全て非該当の場合のデフォルト
```

### フラグの永続化

- 情報フラグはFlagSystem（SaveData内）に保存
- 信頼値はTrustSystem（SaveData内）に保存
- SLレベルはSoulLinkSystem（SaveData内）に保存
- END到達履歴はグローバルセーブ（セーブスロットとは別に保持）
- NG+判定はグローバルセーブのEND到達履歴を参照
