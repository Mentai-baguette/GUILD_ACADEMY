# パーティシステム 仕様書

## 1. パーティ構成

### 1-1. 基本構成

- **バトルメンバー**: 最大4人（レイ固定 + 3人選択）
- **控えメンバー**: 最大10人
- **合計帯同上限**: 14人（レイ含む）

### 1-2. レイの固定枠

- レイは常にバトルメンバーの1番目に固定
- レイをバトルメンバーから外すことはできない
- レイが戦闘不能になってもパーティから外れない（戦闘不能状態でバトル継続）
- レイが侵蝕で暴走した場合は特殊処理（erosion-system参照）

### 1-3. 全員帯同

- パーティに加入した仲間は全員が「帯同」状態
- バトルメンバー4人 + 控え最大10人 = 全員一緒に移動
- 控えメンバーもフィールド上の会話・イベントに参加可能

---

## 2. 授業と帯同の連動

### 2-1. 放課後の帯同制限

平日の放課後にダンジョン探索を行う場合、**授業が終わっていないメンバーは帯同不可**。

| 状況 | 帯同可否 |
|------|---------|
| 4コマ以下で授業終了済み | 帯同可 |
| 5コマ目を履修している（当日） | **帯同不可**（放課後パートのみ不可。夜は可） |
| サボり中 | 帯同可（サボった時点から） |
| 非履修日（その日は授業なし） | 帯同可 |

### 2-2. 時間帯別帯同一覧

| 時間帯 | 帯同ルール |
|--------|----------|
| 放課後（平日） | 5コマ目の仲間は不可。それ以外は可 |
| 夜（平日） | 全員帯同可 |
| 土曜日 | 全員帯同可（終日） |
| 日曜日 | 全員帯同可 |
| 長期休暇 | 全員帯同可 |

### 2-3. シオンの帯同ルール

シオンは2年生で別学年の授業を受けているため、特殊な帯同ルールを持つ。

- シオンの5コマ目がある日: **火曜日・木曜日（固定）**
- 火・木の放課後はシオン帯同不可
- それ以外の平日放課後は帯同可
- 土日・長期休暇・夜パートは常に帯同可
- 2章11週でシオン離脱後は帯同不可（END5復帰まで）

## 3. バトル中の入れ替え

### 3-1. FF10式入れ替え

戦闘中にバトルメンバーと控えメンバーを交代できる。

**入れ替えルール**:
- 1回の入れ替えで **1ターン消費**（入れ替えを選んだキャラのターンを使う）
- 交代で入ったキャラは **ATBゲージ0から開始**
- 入れ替え対象: バトルメンバー（レイ以外の3人）と控えメンバーの1対1交換
- **レイは入れ替え不可**（常にバトルメンバー固定）
- 戦闘不能のキャラを控えに下げることも可能（控えでは自然回復しない）
- 控えから戦闘不能のキャラを出すことはできない

### 3-2. 入れ替えの戦術的意義

- 敵の弱点属性に合わせてメンバーを切り替える
- HP/MPが減ったメンバーを一時的に控えに下げる（控えでは回復しないが安全）
- ブレイク特化のカイトを一時投入してブレイク後に主力と交代、等

### 3-3. 入れ替え時のATBゲージ

```
交代で入るキャラ: ATBゲージ = 0%
交代で出るキャラ: ATBゲージはリセット（控えに戻った時点で0%に）
```

## 4. EXP配分

### 4-1. 基本ルール

- 戦闘で獲得したEXPは **出場ターン比率** に基づいて配分
- 1ターンでも出場すれば「バトル参加」扱い

### 4-2. 配分計算

```
キャラAのEXP = 総EXP × (キャラAの出場ターン数 / 全メンバーの出場ターン合計)
```

**例**: 総EXP 1000、3ターンの戦闘
- レイ: 3ターン出場 → 1000 × (3/9) = 333 EXP
- ユナ: 3ターン出場 → 1000 × (3/9) = 333 EXP
- カイト: 2ターン出場 → 1000 × (2/9) = 222 EXP
- ミオ: 1ターン出場(入れ替えで途中参加) → 1000 × (1/9) = 111 EXP

### 4-3. 控えメンバーのEXP

- **1ターンも出場しなかった控えメンバー**: 総EXPの **50%** を獲得
- この50%は各控えメンバーに均等配分ではなく、**各自が50%を個別に受け取る**

```
控えキャラのEXP = 総EXP × 0.5
```

### 4-4. 入れ替え参加の扱い

- 1ターンでも出場 → **バトル参加扱い**（出場ターン比率で計算）
- 0ターン出場 → **控え扱い**（EXP 50%）

> 1ターンだけ出場して比率が低い場合、控え扱い（50%）より少なくなる可能性がある。
> これは仕様通り。入れ替えの戦術的コストとして機能する。

## 5. 編成変更

### 5-1. 変更可能タイミング

| 場所 | 編成変更 |
|------|---------|
| フィールド（学園内、街中） | **可** — メニューからいつでも |
| ダンジョン外（入口前） | **可** — メニューから |
| ダンジョン内 | **不可** — 入場前に確定 |
| バトル中 | **入れ替えのみ**（セクション3参照） |
| イベント中 | **不可** |

### 5-2. 編成変更UI

```
[パーティ編成]
  バトルメンバー:
    1. レイ (固定)     Lv.XX  HP XXX/XXX
    2. [選択枠]        Lv.XX  HP XXX/XXX
```

## 6. シオンの離脱と復帰

### 6-1. 離脱（2章11週）

**トリガー**: 2章11週の強制イベント（シオンがレイを危険視して敵対）

**処理内容**:
1. シオンがバトルメンバー/控えから強制除外
2. シオンの装備を **全て返却**（プレイヤーのアイテム欄に戻る）
3. シオンのステータス・レベルは内部で保持（復帰時に使用）
4. パーティ画面からシオンが消える
5. シオンのSLイベントは凍結（進行不可）

**装備返却の実装**:
```csharp
void OnShionLeave()
{
    var shion = PartyManager.GetCharacter("shion");
    foreach (var slot in shion.equipmentSlots)
    {
        if (slot.equipment != null)
        {
            Inventory.Add(slot.equipment);
            slot.Unequip();
        }
    }
    PartyManager.RemoveFromParty("shion");
    shion.isAvailable = false;
}
```

### 6-2. 復帰（END5ルートのみ）

**トリガー**: END5の条件を満たし、シオンを闇から救出した後

**処理内容**:
1. シオンが帯同メンバーに復帰
2. シオンのレベルは離脱時のまま（EXPは加算されていない）
3. 装備は空の状態（プレイヤーが再装備する必要あり）
4. SLが再開可能（Lv3まで到達可能に）
5. カルロス最終戦にシオンがNPCとして参戦（5人目の特殊枠）

**復帰時のレベル差対策**:
- 復帰直後にシオンのレベルをパーティ平均Lv-5まで自動調整（下方修正はしない）
- これによりレベル差が極端に開くことを防ぐ

### 6-3. END5以外のルートでのシオン

- END2: シオンが勝利（レイ死亡）のため復帰なし
- END3: シオンが精神崩壊のため復帰なし
- END4: シオンがカルロスに殺害されるため復帰なし
- END6: 全員倒されるため復帰なし

## 7. 特殊パーティ状態

### 7-1. 強制パーティイベント

一部のストーリーイベントではパーティ編成が強制される。

| イベント | 強制メンバー |
|---------|------------|
| 1章チュートリアル | レイ + ユナ + ミオ |
| シオン共同任務（2章3週） | レイ + シオン + 選択2人 |
| カルロス最終戦（END5） | レイ + ユナ + ミオ + カイト（+シオンNPC） |

### 7-2. ソロイベント

- 一部のSLイベントではレイ+対象キャラの2人のみで戦闘が発生
- この場合、バトルメンバーは2人（空き2枠は使用不可）
- 控えメンバーも入れ替え不可

## 8. 実装上の注意

### 8-1. データ構造案

```csharp
public class PartyManager : MonoBehaviour
{
    public const int MAX_BATTLE_MEMBERS = 4;
    public const int MAX_RESERVE_MEMBERS = 10;

    private List<Character> battleMembers;  // [0]は常にレイ
    private List<Character> reserveMembers;

    public bool CanAccompany(Character c, TimeSlot currentTime)
    {
        if (!c.isAvailable) return false; // シオン離脱中等
        if (currentTime == TimeSlot.Night) return true;
        if (currentTime == TimeSlot.AfterSchool)
        {
            return !c.HasFifthPeriodToday(CalendarManager.Today);
        }
        return true;
    }

    public void SwapInBattle(int battleIndex, int reserveIndex)
    {
        if (battleIndex == 0) return; // レイは交代不可
        var temp = battleMembers[battleIndex];
        battleMembers[battleIndex] = reserveMembers[reserveIndex];
        reserveMembers[reserveIndex] = temp;
        battleMembers[battleIndex].atbGauge = 0f;
    }
}
```

### 8-2. EXP配分の計算

```csharp
public void DistributeExp(int totalExp, List<BattleRecord> records)
{
    int totalTurns = records.Sum(r => r.turnsActive);

    foreach (var record in records)
    {
        if (record.turnsActive > 0)
        {
            // バトル参加: ターン比率
            int exp = totalExp * record.turnsActive / totalTurns;
            record.character.AddExp(exp);
        }
    }

    // 控え: 50%
    foreach (var reserve in reserveMembers)
    {
        if (!records.Any(r => r.character == reserve && r.turnsActive > 0))
        {
            reserve.AddExp(totalExp / 2);
        }
    }
}
```
