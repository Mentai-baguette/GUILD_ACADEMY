# 状態異常仕様書

## 概要

バフ・デバフ・状態異常の完全仕様。エンハンスシステム（メル・ヴェイン専用の重ねがけ）を含む。

---

## 1. デバフ（状態異常）6種

### 毒（Poison）

| 項目 | 値 |
|------|-----|
| 効果 | 毎ターン終了時にHP8%ダメージ |
| 持続 | 5ターン（自然回復あり） |
| 重ねがけ | 不可。持続ターンリセットのみ |
| 治療 | 解毒薬 / 状態回復スキル / ターン経過 |
| ボス耐性 | 50%（通常耐性） |

- ダメージは最大HPの8%（端数切り捨て）
- 最低ダメージ1
- 戦闘不能にはならない（HP1で毒ダメージ停止）

### 麻痺（Paralysis）

| 項目 | 値 |
|------|-----|
| 効果 | 毎ターン30%の確率で行動不能 |
| 持続 | 3ターン |
| 重ねがけ | 不可。持続ターンリセットのみ |
| 治療 | 麻痺治療薬 / 状態回復スキル / ターン経過 |
| ボス耐性 | 70% |

- 行動不能時はATBゲージを消費し、ターンがスキップされる
- 防御コマンド中に麻痺判定は発生しない（防御は維持）

### 暗闇（Blind）

| 項目 | 値 |
|------|-----|
| 効果 | 物理攻撃の命中率 -50% |
| 持続 | 4ターン |
| 重ねがけ | 不可。持続ターンリセットのみ |
| 治療 | 目薬 / 状態回復スキル / ターン経過 |
| ボス耐性 | 60% |

- 魔法攻撃には影響なし
- 回避判定: 攻撃時に50%の確率でMISS表示
- クリティカル判定はMISSでなかった場合のみ発生

### 沈黙（Silence）

| 項目 | 値 |
|------|-----|
| 効果 | スキル使用不可（通常攻撃・アイテム・防御は可） |
| 持続 | 3ターン |
| 重ねがけ | 不可。持続ターンリセットのみ |
| 治療 | 沈黙治療薬 / 状態回復スキル / ターン経過 |
| ボス耐性 | 80% |

- デュアルアーツも使用不可（スキル扱い）
- 沈黙中にスキルを選択すると「使用できない」と表示

### スロウ（Slow）

| 項目 | 値 |
|------|-----|
| 効果 | ATBゲージ蓄積速度 -50% |
| 持続 | 4ターン |
| 重ねがけ | 不可。持続ターンリセットのみ |
| 治療 | 万能薬 / 状態回復スキル / ターン経過 |
| ボス耐性 | 60% |

- ヘイストとは相殺（両方付与された場合、後から付与された方が優先）
- 速度計算: AGI x 0.1 x 0.5

### ストップ（Stop）

| 項目 | 値 |
|------|-----|
| 効果 | ATBゲージ蓄積完全停止 |
| 持続 | 2ターン |
| 重ねがけ | 不可。持続ターンリセットのみ |
| 治療 | 万能薬 / 状態回復スキル / ターン経過 |
| ボス耐性 | **完全無効**（100%耐性） |

- ストップ中はターンカウントも進まない（他の状態異常の持続は減らない）
- ストップ中のキャラへの攻撃は確定ヒット（回避不可）
- 唯一ボスに完全無効の状態異常

|------|-----|
| 効果 | ATK x1.2 |
| 持続 | 5ターン |
| 重ねがけ | 通常不可（エンハンス持ちのみ2段階） |
| 解除 | ターン経過 / デスペル系スキル |

### DEFアップ

| 項目 | 値 |
|------|-----|
| 効果 | DEF x1.2 |
| 持続 | 5ターン |
| 重ねがけ | 通常不可（エンハンス持ちのみ2段階） |
| 解除 | ターン経過 / デスペル系スキル |

### ヘイスト（Haste）

| 項目 | 値 |
|------|-----|
| 効果 | ATBゲージ蓄積速度 x1.3 |
| 持続 | 5ターン |
| 重ねがけ | 通常不可（エンハンス持ちのみ2段階） |
| 解除 | ターン経過 / デスペル系スキル |

- スロウと相殺（後付け優先）

### リジェネ（Regen）

| 項目 | 値 |
|------|-----|
| 効果 | 毎ターン終了時にHP3%回復 |
| 持続 | 5ターン |
| 重ねがけ | 通常不可（エンハンス持ちのみ2段階） |
| 解除 | ターン経過 / デスペル系スキル |

- 回復量は最大HPの3%（端数切り上げ）
- 毒と同時に付与されている場合、両方判定（毒8%ダメ → リジェネ3%回復 = 実質5%ダメ）

## 4. エンハンスシステム

### 対象キャラ

- **メル**: バフ系エンハンス持ち
- **ヴェイン**: デバフ系エンハンス持ち

### 仕様

エンハンス持ちのキャラが**かけた側**である場合のみ、同じバフ/デバフを2段階まで重ねがけ可能。

### 2段階時の倍率

| 効果 | 1段階 | 2段階 |
|------|-------|-------|
| ATKアップ | x1.2 | x1.4 |
| DEFアップ | x1.2 | x1.4 |
| ヘイスト | x1.3 | x1.5 |
| リジェネ | HP3%/t | HP5%/t |
| ATKダウン | x0.7 | x0.5 |
| DEFダウン | x0.7 | x0.5 |

### ルール詳細

1. メルがATKアップをかける → 1段階（x1.2）
2. メルが同じ対象に再度ATKアップをかける → 2段階（x1.4）
3. メル以外のキャラがATKアップをかけても → 1段階のまま（持続ターンリセットのみ）
4. 2段階バフの持続ターンは、2段階目をかけた時点から5ターン
5. 2段階バフが切れると1段階に戻るのではなく、完全解除

### エンハンス習得

- メル: スキルツリー「エンハンスの道」路線で習得
- ヴェイン: スキルツリー「エンハンスの道」路線で習得
- エンハンス未習得の場合、通常の重ねがけルール（不可）が適用

## 5. 状態異常の成功率

### 基本計算式

```
最終成功率 = ベース成功率 x (1 - 耐性値)
```

### ベース成功率

- 各スキルに設定された固有値（30%~90%が一般的）
- 一部のスキルは確定付与（ベース成功率100%）

### 耐性値

- キャラ/エネミーごとに各状態異常に対する耐性値（0.0~1.0）を持つ
- 0.0 = 耐性なし（ベース成功率がそのまま適用）
- 1.0 = 完全無効
- ボスは全般的に高耐性。ストップは完全無効（1.0）

### ブレイク中の耐性

```
ブレイク中の耐性値 = 通常の耐性値 x 0.5
```

- 例: 通常耐性0.6のボスが毒攻撃（ベース70%）を受けた場合
  - 通常時: 70% x (1 - 0.6) = 28%
  - ブレイク中: 70% x (1 - 0.3) = 49%

### 例外

- ストップのボス完全無効はブレイク中でも変わらない
- ブレイク中でもストップ耐性1.0 → 0.5にはならない（特別ルール）

## 6. 状態異常の優先度と競合

### 同カテゴリの競合

| パターン | ルール |
|----------|--------|
| バフ + 同種デバフ | 両方有効（倍率を乗算） |
| スロウ + ヘイスト | 後付け優先（先の効果を上書き） |
| 毒 + リジェネ | 両方有効（毒ダメ後にリジェネ回復） |
| ストップ + スロウ | ストップ優先（ストップ解除後にスロウが残る場合あり） |

### 複数状態異常の同時付与

- 1つのスキルで複数の状態異常を付与する場合、それぞれ独立に成功判定
- 全て同時に付与可能（上限なし）

### 戦闘不能時

- 戦闘不能になると全ての状態異常・バフ・デバフが解除
- 蘇生後はクリーン状態

## 9. 実装メモ

### データ構造

```
StatusEffect
├── Type (enum: Poison/Paralysis/Blind/Silence/Slow/Stop/AtkUp/DefUp/Haste/Regen/AtkDown/DefDown)
├── RemainingTurns (int)
├── StackLevel (int: 1 or 2)
├── AppliedBy (CharacterId) — エンハンス判定用
└── BasePower (float) — 倍率や%

StatusEffectManager
├── ApplyEffect(target, effect, appliedBy) → bool (成功/失敗)
├── RemoveEffect(target, type)
├── TickTurnEnd(target) — ターン終了時処理（毒ダメ/リジェネ/持続減少）
├── GetMultiplier(target, statType) → float — ATK/DEFの最終倍率
├── GetATBMultiplier(target) → float — ATB速度倍率
├── IsActionBlocked(target) → bool — 麻痺/ストップ判定
├── IsSkillBlocked(target) → bool — 沈黙判定
└── GetHitRateModifier(target) → float — 暗闇判定
```
