# 装備・経済・ショップ・錬金術 仕様書

## 1. 装備スロット

キャラクターごとに6枠の装備スロットを持つ。

| スロット | 説明 |
|---------|------|
| 武器 | キャラ専用タイプのみ装備可 |
| 頭 | 兜・帽子・サークレット等 |
| 上半身 | 鎧・ローブ・コート等 |
| 下半身 | 脚甲・ブーツ・ズボン等 |
| アクセサリー1 | 自由枠（指輪・腕輪・ブローチ等） |
| アクセサリー2 | 自由枠（同上） |

**制約**: 同一アクセサリーの2枠同時装備は不可。

## 3. 個体差（ランダム補正）システム

装備品の入手方法によってステータスの振れ幅が異なる。

| 入手方法 | 補正ルール |
|---------|-----------|
| 敵ドロップ | 基本値 ×0.85〜1.15 のランダム補正（小数切り捨て） |
| ショップ購入 | 基本値 ×0.85 固定（最低値） |
| クエスト報酬（非売品） | 基本値 ×1.00 固定 |
| ユニーク装備 | 固定値（補正なし） |
| 錬金術生成 | 基本値 ×0.90〜1.10（素材品質で変動） |

**データ構造例**:
```csharp
[System.Serializable]
public class EquipmentInstance
{
    public string baseId;        // マスターデータ参照
    public float qualityMod;     // 0.85〜1.15
    public EquipmentSource source; // Drop / Shop / Quest / Unique / Alchemy
}
```

## 9. 錬金術システム

### 9-1. 解放条件

- **3章開始時**: ニコラ先生の錬金術授業イベントで解放
- 錬金釜がショップに追加される

### 9-2. 基本ルール

- **素材 + 素材 → 完成品**（2〜3素材の組み合わせ）
- **レシピが必要**: レシピを知らない組み合わせは実行不可
- **失敗あり**: 失敗時は素材が消失（完成品なし）
- 1回の錬金に所要時間なし（即時完了）

### 9-3. 成功率

基本成功率はレシピごとに設定（60%〜90%）。以下の要素で補正。

| 条件 | 補正 |
|------|------|
| ニコラ先生信頼度 Lv1 | +5% |
| ニコラ先生信頼度 Lv2 | +10% |
| ニコラ先生信頼度 Lv3 | +15% |
| ニコラ先生信頼度 MAX | +20%（失敗率0%のレシピが出現） |
| 錬金術の成績A以上 | +5% |

成功率上限は99%（MAX信頼+成績A以上でも一部レシピは100%にならない）。
ニコラ信頼MAXで解放される「完全レシピ」のみ成功率100%固定。

### 9-4. レシピ解放

| 解放方法 | レシピ数 | 備考 |
|---------|---------|------|
| 錬金術の授業を受ける | 10〜15種 | 章ごとに追加 |
| ニコラ信頼度上昇 | 5〜8種 | 信頼イベントで教えてもらう |
| ダンジョン内の書物 | 3〜5種 | 隠し部屋等 |
| NG+引き継ぎ | 全レシピ維持 | — |

### 9-5. 主要レシピ例

| 完成品 | 素材1 | 素材2 | 素材3 | 基本成功率 |
|--------|-------|-------|-------|----------|
| ハイポーション | ポーション | 薬草 | — | 90% |

## 10. 強化合成（3章〜）

- **装備 + 素材 → 強化装備**: 最大+5段階
- 段階ごとにステータスが+5%ずつ上昇
- 強化費用: 基本価格の10%(+1)〜50%(+5)
- 失敗なし（確定成功）
- +3以上は上位素材が必要
