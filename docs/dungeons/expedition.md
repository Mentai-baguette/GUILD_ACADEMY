# 遠征先ダンジョン

## 基本情報

| 項目 | 内容 |
|------|------|
| ダンジョン名 | 忘却の遺跡 |
| 英名 | Forgotten Ruins |
| 解放時期 | 夏休み1（期間限定） |
| 層数 | 7層 |
| 推奨Lv | 15〜28 |
| 属性傾向 | 地・闇 |
| 特記 | 期間限定ダンジョン。夏休み1終了後は入場不可。限定ドロップあり |

## 世界観・雰囲気

夏休みの遠征実習として訪れる、学園から馬車で半日の距離にある古代遺跡。かつて魔導文明が栄えた時代の研究施設跡で、今は廃墟と化している。壁面には解読不能な古代文字が刻まれ、奥に進むほど魔導技術の残滓が色濃く残る。

崩れかけた石柱、苔むした床、所々で光る魔導結晶。機能停止した魔導装置が散在し、一部はまだ稼働して罠として機能している。最深部には古代の守護装置がまだ動いている。

**ストーリー上の位置づけ:** 夏休みの課外実習。カルロスが引率として同行するが、実はカルロスが古代の魔導技術を調査する裏目的がある。

**雰囲気キーワード:** 古代遺跡、魔導結晶の光、崩壊した文明、砂と石

## 各層の概要

### 1層: 遺跡入口

- **概要:** 崩れかけた入口。陽光がまだ差し込む
- **敵の傾向:** 遺跡スライム、石像の欠片
- **ギミック:** 崩落地点（通れない道があり、迂回が必要）
- **宝箱:** ポーション×3（固定）、帰還の石×1（固定）

### 2層: 古代の回廊

- **概要:** 長い廊下が続く。壁面の古代文字が光る
- **敵の傾向:** 遺跡ゴーレム、遺跡スライム、ストーンバット
- **ギミック:** 文字パズル（壁面の古代文字を正しい順序で触れると扉が開く）
- **宝箱:** マジックポーション×2（固定）
- **隠し通路:** 20%で壁面の割れ目 → 古代の魔石×1

### 3層: 魔導研究室跡

- **概要:** かつての研究施設。壊れた装置が散乱
- **敵の傾向:** 遺跡ゴーレム、魔導パペット、マジックトラップ
- **ギミック:** 魔導装置の起動（正しい順序でスイッチを押すと宝箱出現）
- **宝箱:** ハイポーション×2（固定）
- **レア宝箱:** 15%で古代の魔石×2

### 4層: 崩壊区画

- **概要:** 大きく崩壊した区域。足場が不安定
- **敵の傾向:** 石像ガーディアン、魔導パペット、ストーンバット
- **ギミック:** 崩壊する床（一定時間で崩れる。素早く移動する必要あり）
- **宝箱:** 不死鳥の羽根×1（固定）、万能薬×1（固定）
- **隠し通路:** 20%で崩壊した壁の裏 → 速さの種×1
- **隠し部屋:** 10%で地下室 → 古代の防具（上半身。DEF+30）

### 5層: 封印の部屋

- **概要:** 何かを封じていた形跡がある部屋。魔力が残存
- **敵の傾向:** 遺跡ゴーレム（強化版）、アンシェントゴースト、石像ガーディアン
- **ギミック:** 魔法陣の解除（属性攻撃で対応する魔法陣を無効化。4属性すべて必要）
- **宝箱:** 英雄の薬×1（固定）
- **レア宝箱:** 15%で魔力の種×1

### 6層: 最深部前室

- **概要:** ボス前の最終準備エリア。回復ポイントあり
- **敵の傾向:** アンシェントゴースト、遺跡ゴーレム（強化版）
- **ギミック:** 回復の泉（HP/MP全回復）。セーブポイント
- **宝箱:** ハイポーション×3（固定）、マジックポーション×2（固定）
- **隠し通路:** 20%で壁のひび割れ → 不死鳥の翼×1
- **隠し部屋:** 10%（「遺跡の鍵」が必要） → 古代の書物（錬金レシピ2種解放）

### 7層: 守護者の間【ボス】

- **概要:** 遺跡の中枢。古代の守護装置が起動している
- **敵の傾向:** ボスのみ
- **ギミック:** ボス前にセーブポイント
- **宝箱:** ボス撃破報酬（古代魔導剣 or 古代の護符 + 3,000G）
- **レア宝箱:** 15%で追加報酬 → 古代の魔石×5
- **ボス:** 古代守護装置ガルディオン（Lv25 / 地属性 / HP4500）
  - 古代文明が作った自律型魔導兵器
  - フェーズ1（HP100%〜50%）: 「魔導砲」単体大ダメージ + 「防御シフト」DEF大幅上昇
  - フェーズ2（HP50%以下）: 「全方位魔導弾」全体攻撃 + 「自己修復」HP1000回復（1回のみ）
  - 地属性だが、弱点は氷。ブレイクで防御シフトを解除可能
  - 限定ボスのため、やや強めに設定
