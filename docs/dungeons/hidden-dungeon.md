# 忘却の回廊

## 基本情報

| 項目 | 内容 |
|------|------|
| ダンジョン名 | 忘却の回廊（ぼうきゃくのかいろう） |
| 英名 | Corridor of Oblivion |
| 解放時期 | 夏休み2 / 準備週 |
| 層数 | 22層 |
| 推奨Lv | 35〜65 |
| 属性傾向 | 全属性（層ごとに変化） |
| 特記 | 隠しダンジョン。やり込み用。レア装備多数 |

## 解放条件

- **基本条件:** グレイヴ学園長に話しかける（夏休み2 / 準備週で選択可能に）
- **追加条件（END2/3/6ルート）:** 鬼気の洞窟10層以上到達済みであること
- グレイヴが「かつての英雄が辿った道」として、忘却の回廊の入口を教えてくれる

## 世界観・雰囲気

学園の敷地内にある古い井戸の底から繋がる異空間。300年前の聖戦時代に英雄たちが修行した場所とされるが、その存在は学園長グレイヴだけが知っている。

時間の流れが歪んでおり、壁面には様々な時代の風景が映し出される。ある層では灼熱の砂漠、ある層では凍える雪原、ある層では星空の海。各層が異なる環境を持ち、全属性の敵が出現する。忘却の名の通り、この場所に長く留まると記憶が薄れるという伝承がある。

**ストーリー上の位置づけ:** やり込みコンテンツ。ストーリー本筋には直接関わらないが、グレイヴの過去やエルトに関する追加情報が得られる。レア装備が多く、END5攻略の準備に最適。

**雰囲気キーワード:** 時の歪み、多層世界、幻想的、記憶の断片、英雄の修行場

## 各層の概要

### 1〜3層: 記憶の入口（火属性テーマ）

- **環境:** 灼熱の遺跡。溶岩の流れる回廊
- **敵の傾向:** フレイムゴーレム、マグマスライム、火竜の幼体
- **ギミック:** 溶岩地帯（踏むとHP減少。氷魔法で冷却して通行可能）
- **宝箱:** 各層にハイポーション×3、エクスポーション×1
- **3層レア宝箱:** 15%で炎の結晶×2（錬金素材）

### 4〜6層: 凍える時代（氷属性テーマ）

- **環境:** 永久凍土の雪原。吹雪が視界を遮る
- **敵の傾向:** フロストナイト、氷の精霊、ブリザードウルフ
- **ギミック:** 吹雪エリア（AGI低下デバフ。暖炉ポイントで一時解除）
- **宝箱:** 各層にマジックポーション×3、万能薬×2
- **4層隠し部屋:** 10%で蒼氷の杖（ユナ/ミオ用杖MATK90）
- **6層レア宝箱:** 15%で氷の精霊石×1

### 7〜9層: 嵐の時代（風属性テーマ）

- **環境:** 荒れ狂う嵐の中の浮島。雷が落ちる
- **敵の傾向:** ストームイーグル、雷精霊、風の剣士（幻影）
- **ギミック:** 落雷トラップ（ランダム位置に雷。当たると大ダメージ+麻痺）
- **宝箱:** 各層に英雄の薬×1、不死鳥の羽根×2
- **7層隠し部屋:** 10%で風切りの短刀（ジン用短刀ATK95）
- **9層レア宝箱:** 15%で雷の結晶×2

### 10層: 安全地帯

- **概要:** 時の狭間の休憩所。全環境から隔絶された静かな空間
- **敵の傾向:** なし
- **ギミック:** 回復の泉 + セーブポイント + 時の商人（NPC）
- **宝箱:** フルポーション×2（固定）
- **隠し部屋:** 10%で時の商人の隠し在庫（エリクサー×1を50,000Gで販売）

### 11層: 時の番人の間【中ボス】

- **環境:** 巨大な時計台の内部。歯車が回る
- **敵の傾向:** 中ボスのみ
- **ギミック:** セーブポイント（中ボス前）
- **宝箱:** 中ボス撃破報酬（時の砂時計[アクセ。AGI+15%] + 12,000G）
- **中ボス:** 時の番人クロノス（Lv45 / 風属性 / HP12000）
  - 時間操作を使う特殊なボス
  - 「時間停止」ランダム1体のATBゲージを0にする
  - 「加速」自身のAGI2倍（3ターン）
  - 「時の逆流」直前に受けたダメージを回復
  - 地属性が弱点。素早く倒さないと長期戦でジリ貧に

### 12〜14層: 大地の時代（地属性テーマ）

- **環境:** 地下大空洞。巨大な水晶が輝く
- **敵の傾向:** クリスタルゴーレム、地竜の幼体、宝石甲虫（上位）
- **ギミック:** 結晶パズル（光の反射を利用して道を開く）
- **宝箱:** 各層にエクスポーション×2、精霊石×1
- **13層隠し部屋:** 10%で烈火の拳甲（レンジ用拳ATK90）
- **14層隠し通路:** 20%で深淵の結晶×2

### 15〜17層: 光の時代（光属性テーマ）

- **環境:** 黄金に輝く大聖堂。荘厳な雰囲気
- **敵の傾向:** 聖騎士の幻影、光の精霊、天使型魔導体
- **ギミック:** 光と影のパズル（光源を移動させて影を作り、隠し通路を発見）
- **宝箱:** 各層にフルポーション×1、聖水×5
- **15層レア宝箱:** 15%で光断の双刃（シオン用双刃ATK95）
- **16層隠し部屋:** 10%で聖騎士の全身鎧(上)（DEF+55、光耐性+10%）
- **17層隠し通路:** 20%で女神の祝福（アクセ。1回/戦闘 戦闘不能防止）

### 18〜20層: 闇の時代（闇属性テーマ）

- **環境:** 漆黒の虚空。足元だけが光る不気味な空間
- **敵の傾向:** 闇の眷属、影の剣士、虚空の魔導師
- **ギミック:** 闇の侵蝕（レイの侵蝕ゲージ+5/層。ユナの浄化で軽減）
- **宝箱:** 各層にエリクサー×1（21層手前の最後の補給）
- **18層レア宝箱:** 15%で暗黒の呪書・改（ヴェイン用呪具MATK120）
- **19層隠し部屋:** 10%で盗賊王のマント（上半身。ドロップ率+15%）
- **20層隠し通路:** 20%で深淵の結晶×3

### 21層: 最終安全地帯

- **概要:** ボス前の最終休憩
- **敵の傾向:** なし
- **ギミック:** 回復の泉 + セーブポイント
- **宝箱:** フルポーション×5（固定）、エリクサー×1（固定）
- **レア宝箱:** 15%で不死鳥の翼×2

### 22層: 回廊の終焉【ボス】

- **環境:** 全ての時代が混在する異空間。壁面に無数の時代の風景
- **敵の傾向:** ボスのみ
- **ギミック:** セーブポイント（ボス前）
- **宝箱:** ボス撃破報酬（忘却の剣[レイ用武器ATK180] + 30,000G + エリクサー×3）
- **レア宝箱:** 15%で追加報酬 → 時の指輪（アクセ。ATBゲージ蓄積速度+15%）
- **ボス:** 忘却の王エターナル（Lv60 / 全属性 / HP25000）
  - 回廊を守り続けてきた古代の存在
  - 6属性を切り替えて戦う。使用属性に対応する弱点を突くのが鍵
  - フェーズ1（HP100%〜70%）: 単一属性攻撃。3ターンごとに属性チェンジ
  - フェーズ2（HP70%〜40%）: 2属性同時使用。「忘却の風」で全員にランダムデバフ
  - フェーズ3（HP40%以下）: 「時空崩壊」全体全属性ダメージ + 「記憶消去」ランダム1体のMP全消費
  - 属性チェンジに対応できる多属性のパーティ構成が重要
