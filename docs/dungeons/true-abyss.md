# 裏深淵

## 基本情報

| 項目 | 内容 |
|------|------|
| ダンジョン名 | 裏深淵（うらしんえん） |
| 英名 | True Abyss |
| 解放時期 | 深淵（50層）クリア後 |
| 層数 | 50層 |
| 推奨Lv | 150〜200（ゼノン戦はLv200でも苦戦する設計） |
| 属性傾向 | 全属性（神聖+深淵） |
| 特記 | 本作最高難度エリア（エルトを除く） |

## 世界観・雰囲気

モルドレッドが300年かけて作り上げた「魔王復活の祭壇」。深淵のさらに奥、創世の残滓が漂う異空間に存在する。ここは世界が生まれる前の「無」に最も近い場所であり、神々の力の名残が形を成している。

空間は白と黒が混在する幻想的な風景。巨大な結晶柱が浮遊し、星雲のような光が漂う。床は透明な水晶で、足元に無限の虚空が広がる。深層に進むほど、創世の力が濃くなり、敵は「神の眷属」と呼ばれる超越的存在に変わる。

50層最深部には堕ちた神ゼノンが封じられている。かつて世界を創造した神々の一柱であり、魔王を生み出した元凶。モルドレッドはゼノンの復活こそが教団の最終目標だった。

**雰囲気キーワード:** 創世の残滓、白と黒の異空間、浮遊する結晶、神の領域、終焉と始まり

## 構造概要（10層ごとのブロック）

### 1〜10層: 創世の入口ブロック

- **環境:** 白い結晶の回廊。浮遊する光の粒子
- **敵Lv帯:** 150〜165
- **敵の傾向:** 光の断片、創世のスライム、結晶の騎士
- **ギミック:** 結晶パズル（光を正しく反射させて道を開く。深淵より複雑）
- **宝箱:** エリクサー、神木の枝、創世の欠片（最高級素材）
- **5層:** 安全地帯（回復+セーブ）
- **10層【中ボス】:** 光の審判者（Lv160 / 光属性 / HP60000）
  - 「聖なる裁き」全体光超大ダメージ
  - 「光の盾」3ターン物理+魔法ダメージ半減。ブレイクで解除
  - 「浄化の光」状態異常を全解除（自身のデバフ含む）
  - 闇属性が弱点

### 11〜20層: 堕天の回廊ブロック

- **環境:** 白と黒が入り混じる。天使の像が並ぶ回廊
- **敵Lv帯:** 165〜178
- **敵の傾向:** 堕天使の幻影、闇の大天使、創世の獣
- **ギミック:** 善悪の選択（分岐路で「光の道」「闇の道」を選ぶ。それぞれ異なる敵と報酬）
- **宝箱:** NG+最終装備の素材
- **15層:** 安全地帯（回復+セーブ）
- **20層【中ボス】:** 堕天使ルシファエル（Lv175 / 闇属性 / HP80000）
  - 光と闇の両方を使う。属性切り替え
  - 「天罰」全体光超大ダメージ + 「堕落」全体闇超大ダメージ
  - 「天地開闢」全ステバフ+HP10000回復
  - 光モード時は闇弱点、闇モード時は光弱点

### 21〜30層: 神殿ブロック

- **環境:** かつて神々が住まいした神殿の廃墟
- **敵Lv帯:** 178〜188
- **敵の傾向:** 神殿の守護者、神の兵士、石化の天使
- **ギミック:** 神殿の謎解き（古代神の言語を解読して進む。ヒントは各層の石碑）
- **宝箱:** 超レア素材、創世の結晶
- **25層:** 安全地帯（回復+セーブ）
- **30層【中ボス】:** 神殿の守護神ミカエリス（Lv185 / 光属性 / HP100000）
  - 全属性の防御スキルを持つ鉄壁のボス
  - 「不壊の盾」1属性を完全無効化（3ターン。属性はランダム）
  - 「神罰の剣」単体物理超大ダメージ+DEFダウン
  - 「聖域展開」全体回復+全体バフ
  - 闇属性が弱点だが、不壊の盾で無効化される場合あり

### 31〜40層: 創造の間ブロック

- **環境:** 世界が創られた場所。星雲と光の渦が広がる
- **敵Lv帯:** 188〜195
- **敵の傾向:** 創世の使徒、星の精霊、原初の獣
- **ギミック:** 創造パズル（素材を集めて「道」を創る。属性の組み合わせが鍵）
- **宝箱:** 虚無の腕輪の素材、最高ティアの全装備素材
- **35層:** 安全地帯（回復+セーブ）
- **40層【中ボス】:** 創世の竜王テラス（Lv192 / 全属性 / HP120000）
  - 全属性のブレスを使う最強格の竜
  - 「全属性ブレス」ランダム属性の全体超大ダメージ
  - 「竜の咆哮」全員ATBゲージ0+全デバフ
  - 「再生」毎ターンHP8000回復。ブレイクで3ターン停止
  - 弱点はターンごとに変化。パターンを読む必要あり

### 41〜49層: 神の領域ブロック

- **環境:** 純白の虚空。何もない空間にゼノンの力だけが渦巻く
- **敵Lv帯:** 195〜200
- **敵の傾向:** ゼノンの分身、虚無の騎士（神級）、終焉の天使
- **ギミック:** 虚無の侵蝕（全員HP/MP毎ステップ減少。最高難度の耐久戦）
- **宝箱:** 最終装備の完成素材
- **45層:** 安全地帯（最終回復+セーブ）

### 50層: 創造主の玉座【ボス: ゼノン】

- **環境:** 世界の始まりと終わりが交差する空間。ゼノンの玉座
- **敵の傾向:** ボスのみ
- **ギミック:** セーブポイント（ボス前）
- **宝箱:** ボス撃破報酬（虚無の腕輪[アクセ。EXP+15%, 全属性耐性+10%] + 500,000G + 真・エンディング解放鍵）
- **ボス:** 堕ちた神ゼノン（Lv200 / 全属性 / HP200000）
  - 本作の真のラスボス（エルトを除く最高難度）
  - **フェーズ1「創造主」**（HP100%〜70%）:
    - 「創世の光」全体全属性超大ダメージ
    - 「世界改変」フィールド効果変更（全属性相性がランダムに変化）
    - 「神の御手」ランダム2体を行動不能にする
    - 「創造」雑魚2体を召喚（Lv180級）
  - **フェーズ2「裁定者」**（HP70%〜30%）:
    - 「終焉の宣告」3ターン後に全体即死（3ターン以内に一定ダメージで解除）
    - 「因果律操作」味方のバフを全てデバフに変換
    - 「神罰」HP最大の味方に即死級ダメージ
    - 「神の再生」毎ターンHP15000回復
  - **フェーズ3「堕ちた神」**（HP30%以下）:
    - 「深淵解放」全能力超上昇。外見が完全に変化
    - 「世界崩壊」全体全属性99999ダメージ級（HP1耐えアクセ+防御で生存）
    - 「運命の書き換え」ランダムで味方1体を即死させる（蘇生必須）
    - 「最後の審判」HP5%以下で全体即死（これを耐えて削り切ると勝利）
  - 弱点属性なし（全等倍）。純粋なステータスと戦略で勝つ戦い
  - 全員Lv200+最強装備+完璧な戦略が求められる本作究極の挑戦
