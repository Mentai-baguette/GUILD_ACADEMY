# 学園地下訓練場

## 基本情報

| 項目 | 内容 |
|------|------|
| ダンジョン名 | 学園地下訓練場 |
| 英名 | Training Ground |
| 解放時期 | 1章（入学直後） |
| 層数 | 7層 |
| 推奨Lv | 1〜10 |
| 属性傾向 | 無属性中心 |

## 世界観・雰囲気

クロノス・アカデミー本館の地下に広がる訓練施設。学園創設以来、新入生の実戦訓練に使われてきた。壁には古い魔法陣が刻まれ、訓練用ゴーレムや擬似モンスターを生成する仕組みが稼働している。

照明は魔導灯で一定の明るさが確保されており、安全面も配慮されている。ただし下層に行くほど魔導灯の数が減り、やや薄暗くなる。7層目の最深部は「上級試験場」と呼ばれ、実力ある生徒のみが挑む場所。

**雰囲気キーワード:** 石造りの地下、魔法陣の淡い光、安心感のある訓練場

## 確率要素

| 要素 | 発生率 | 配置可能層 |
|------|--------|-----------|
| 隠し通路 | 20% | 2層、4層、6層 |
| レア宝箱 | 15% | 3層、5層、7層 |
| 隠し部屋 | 10% | 5層、6層 |
| NPC出現 | 25% | 1層、3層、5層 |

**NPC例:**
- 先輩冒険者: 戦闘のコツを教えてくれる（チュートリアルTips）
- 教官: 属性相性やブレイクシステムの説明
- 迷子の1年生: 助けるとアイテムをくれる

## 各層の概要

### 1層: 入門の間

- **概要:** チュートリアル層。基本操作（移動・調べる）を学ぶ
- **敵の傾向:** 訓練用スライム（HP低い、攻撃しない）のみ
- **ギミック:** なし（安全な移動練習）
- **宝箱:** ポーション×2（固定）
- **レイアウト:** 一本道。教官NPCが各所で操作説明

### 2層: 基礎戦闘場

- **概要:** ATBバトルの基本を学ぶ。通常攻撃・防御・アイテムの練習
- **敵の傾向:** 訓練用スライム、訓練用ゴーレムLv1
- **ギミック:** 壊せる壁（攻撃で破壊→宝箱が出現する場合あり）
- **宝箱:** ポーション×1（固定）、解毒剤×1（固定）
- **隠し通路:** 20%で北壁に通路出現 → 力の種×1

### 3層: 中間試験場【中ボス】

- **概要:** ブレイクシステムを学ぶ。中ボス「訓練用ガーディアン」が待ち受ける
- **敵の傾向:** 訓練用ゴーレムLv1、訓練用ゴーレムLv2
- **ギミック:** スイッチパズル（2つのスイッチを踏むと中ボス部屋が開く）
- **宝箱:** マジックポーション×1（固定）
- **中ボス:** 訓練用ガーディアン（Lv5 / 無属性 / HP250）
  - ブレイク値が低く設定されており、ブレイクを体験させる設計
  - 「防御姿勢」→「突進」の2ターンパターンで攻撃を教える

### 4層: 属性訓練場

- **概要:** 属性相性を学ぶ。火・氷・風・地の属性付き訓練ゴーレムが出現
- **敵の傾向:** 火訓練ゴーレム、氷訓練ゴーレム、風訓練ゴーレム、地訓練ゴーレム
- **ギミック:** 属性パネル（対応属性のスキルで起動すると道が開く）
- **宝箱:** 力の種×1（固定）、守りの種×1（固定）
- **隠し通路:** 20%で東の壁 → ポーション×3

### 5層: 応用戦闘場

- **概要:** 複数の敵が同時に出現。連携やアイテム管理を学ぶ
- **敵の傾向:** 訓練用ゴーレムLv2（2〜3体同時出現）、訓練用スライム（大）
- **ギミック:** 落とし穴（踏むと1層分戻される。周囲のヒビで判別可能）
- **宝箱:** ハイポーション×1（固定）
- **隠し部屋:** 10%で南壁 → 鉄の剣（レイ用初期武器上位）
- **NPC:** 25%で先輩冒険者が出現。デュアルアーツの説明

### 6層: 上級前夜

- **概要:** ボス前の最終準備層。強敵が出現し、装備やアイテムの確認を促す
- **敵の傾向:** 訓練用ゴーレムLv3、上級訓練スライム
- **ギミック:** 回復の泉（HP/MP全回復、1回のみ使用可能）
- **宝箱:** 不死鳥の羽根×1（固定）、ポーション×2（固定）
- **隠し通路:** 20%で西の壁 → マジックポーション×2
- **隠し部屋:** 10%で北の壁 → 鉄の盾（リン用初期装備上位）

### 7層: 上級試験場【ボス】

- **概要:** 訓練場の最終層。ボス「上級訓練ゴーレム」が守る
- **敵の傾向:** ボスのみ（道中はなし。ボス部屋直行）
- **ギミック:** ボス前にセーブポイントと回復の泉
- **宝箱:** ボス撃破後に報酬宝箱（訓練修了の証 + 500G + ポーション×5）
- **レア宝箱:** 15%で追加報酬 → 速さの種×1
- **ボス:** 上級訓練ゴーレム（Lv8 / 無属性 / HP600）
  - 1章チュートリアルの総まとめ
  - フェーズ1（HP100%〜50%）: 通常攻撃パターン
  - フェーズ2（HP50%以下）: 「自己強化」でATK上昇 → 攻撃が痛くなる
  - 弱点: ブレイクしやすい（ゲージ80）。ブレイク中に畳みかけるのが正攻法
