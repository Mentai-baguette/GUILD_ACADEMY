## 世界観・雰囲気

クロノス・アカデミーの地下深くに広がる「禁域」。学園の真の姿――300年前の聖戦で建てられた「魔王の封印装置」の中心部。

入口は学園長室の隠し階段からのみアクセス可能（2章で解放）。石壁には魔王封印のための古代魔法陣が無数に刻まれ、青白い光を放っている。空気は冷たく、封印の力が渦巻いている。呼吸するだけで魔力を感じるほどの濃密な魔力場。

深層に進むほど封印の力は不安定になり、壁面の魔法陣が明滅し始める。魔導体や亡霊が徘徊し、封印の綻びから漏れ出した魔王の力で変異した存在が蠢く。15層最深部にはかつての「器」エルトが封印されたまま変質し、最強の敵として待ち受ける。

**ストーリー上の重要性:**
- 2章でレイが夜中に無意識で引き寄せられる場所
- シオン戦（ストーリー）は封印の間の特別エリアで発生（通常探索とは別区画）
- カルロスの真の目的は封印核の破壊

**特別エリアについて:** ストーリー上のシオン戦は「封印の間・祭壇区画」という別マップで行われる。通常探索の15層とは独立しており、ストーリー進行時のみアクセス可能。

**雰囲気キーワード:** 封印の魔法陣、青白い光、不気味な静寂、冷気、魔王の残滓

## 各層の概要

### 1層: 禁域の入口

- **概要:** 学園長室から続く隠し階段の先。厳重な封印扉がある
- **敵の傾向:** 封印の残滓、魔導コウモリ
- **ギミック:** 封印扉の解錠（グレイヴの許可が必要。ストーリー進行で解放）
- **宝箱:** ハイポーション×3（固定）、聖水×2（固定）

### 2層: 封印の回廊

- **概要:** 壁面に魔法陣が光る長い回廊。静寂が支配する
- **敵の傾向:** 封印の残滓、魔導体Lv1、亡霊兵
- **ギミック:** 魔法陣パズル（光る魔法陣を正しい順序で触れると先に進める）
- **宝箱:** マジックポーション×2（固定）
- **隠し通路:** 20%で壁面の魔法陣の裏 → 守りの種×2

### 3層: 魔導体の巣

- **概要:** 封印の力で生まれた魔導体が多数生息するエリア
- **敵の傾向:** 魔導体Lv1、魔導体Lv2、封印の目
- **ギミック:** 魔導体の巡回パターン（シンボルの動きを読んで回避可能）
- **宝箱:** 万能薬×2（固定）
- **レア宝箱:** 15%で魔力の種×1

### 4層: 亡者の間

- **概要:** かつてここで命を落とした者たちの亡霊が漂う
- **敵の傾向:** 亡霊兵、亡霊魔導師、封印の残滓（強）
- **ギミック:** 亡霊との会話イベント（情報フラグ「封印の方法を知る」に関連）
- **宝箱:** 不死鳥の羽根×2（固定）
- **隠し部屋:** 10%で壁の裏 → 聖水×5 + 古い日記（ロアアイテム）

### 5層: 中間安全地帯

- **概要:** 封印が特に強い区域。敵の出現が少ない休憩層
- **敵の傾向:** 封印の残滓のみ（出現率低い）
- **ギミック:** 回復の泉 + セーブポイント
- **宝箱:** エクスポーション×1（固定）、ハイマジックポーション×1（固定）
- **隠し通路:** 20%で泉の裏 → 英雄の薬×1

### 6層: 封印核外殻

- **概要:** 封印核に向かう外殻部。封印の力が強く、侵蝕値に影響
- **敵の傾向:** 魔導体Lv2、封印の目、闇の精霊
- **ギミック:** 侵蝕エリア（レイの侵蝕ゲージが微量ずつ上昇。ユナの浄化で抑制可能）
- **宝箱:** 聖水×3（固定）
- **レア宝箱:** 15%で不死鳥の翼×1

### 7層: 魔王の記憶

- **概要:** 封印された魔王の記憶が映像として漂う不気味な空間
- **敵の傾向:** 記憶の断片（闇）、魔導体Lv3、闇の精霊
- **ギミック:** 記憶の映像イベント（300年前の聖戦のビジョン。情報フラグ関連）
- **宝箱:** 魔力の種×1（固定）、万能薬×3（固定）
- **隠し通路:** 20%で映像の裏側 → ハイポーション×5

### 8層: 崩壊回廊

- **概要:** 封印の綻びで崩壊が進んでいる区域。足場が不安定
- **敵の傾向:** 魔導体Lv3、亡霊騎士、封印崩壊体
- **ギミック:** 崩壊する床（タイミング移動）。魔法陣で足場を安定化可能
- **宝箱:** 英雄の薬×1（固定）
- **レア宝箱:** 15%で鋼のインゴット×3（錬金素材）
- **隠し部屋:** 10%で崩壊した壁の裏 → ミスリルの冠（頭防具）

### 9層: 守護者の試練

- **概要:** 封印の守護者が設置した試練の間。パズルと戦闘の複合
- **敵の傾向:** 試練の守護者（複数体）、封印崩壊体
- **ギミック:** 連戦試練（3連戦。途中回復なし。クリアで先に進める）
- **宝箱:** エクスポーション×2（固定）、ハイマジックポーション×2（固定）
- **隠し通路:** 20%で試練の間の裏 → 速さの種×2

### 10層: 深淵の入口【中ボス】

- **概要:** 封印の最深部への入口。中ボスが待ち受ける
- **敵の傾向:** 中ボスのみ
- **ギミック:** セーブポイント + 回復の泉（中ボス前）
- **宝箱:** 中ボス撃破報酬（封印の欠片 + 5,000G）
- **中ボス:** 封印の番人ゼラス（Lv35 / 光属性 / HP6000）
  - 封印を守るために作られた魔導兵器
  - 「聖光の盾」で物理ダメージ半減
  - 「裁きの光」全体光属性大ダメージ
  - 闇属性が弱点。ブレイクで聖光の盾を解除

### 11層: 封印核接近域

- **概要:** 封印核が近い。魔力が極めて濃密。レイの侵蝕が加速
- **敵の傾向:** 魔導体Lv4、深淵の影、亡霊将軍
- **ギミック:** 侵蝕強制上昇エリア（レイの侵蝕+5。ユナの浄化で+2に軽減可能）
- **宝箱:** フルポーション×1（固定）
- **レア宝箱:** 15%で精霊石×2（錬金素材）
- **隠し部屋:** 10%で隠し祭壇 → 「エルトの形見」入手のヒント

### 12層: 魔王の揺籃

- **概要:** かつて魔王の肉体があった場所。空の玉座が佇む
- **敵の傾向:** 深淵の影、魔導体Lv4、封印崩壊体（上位）
- **ギミック:** 玉座を調べると情報フラグ「魔王の器の真実を知る」に関連するイベント
- **宝箱:** 英雄の薬×2（固定）
- **隠し通路:** 20%で玉座の裏 → 深淵の結晶×1（高級錬金素材）

### 13層: 記憶の回廊

- **概要:** エルトの記憶が漂う回廊。彼の人生の断片が見える
- **敵の傾向:** 記憶の断片（強）、深淵の眷属、魔導体Lv5
- **ギミック:** エルトの記憶イベント（グレイヴとの出会い、封印の経緯が語られる）
- **宝箱:** エクスポーション×2（固定）、聖水×5（固定）

### 14層: 最深部前室

- **概要:** エルト戦前の最終準備エリア
- **敵の傾向:** 深淵の眷属、魔導体Lv5
- **ギミック:** セーブポイント + 回復の泉。エルトの幻影との会話イベント
- **宝箱:** フルポーション×2（固定）、不死鳥の翼×1（固定）
- **隠し通路:** 20%で壁面 → ハイマジックポーション×3
- **隠し部屋:** 10%で地下室 → 古代の全身鎧（上半身DEF+45）

### 15層: 封印の最深部【シンボルボス: エルト】

- **概要:** 封印核がある最深部。変質したエルトが佇んでいる
- **敵の傾向:** エルトのみ（シンボルエンカウント。スルー可能。逃走可能）
- **ギミック:** エルトに話しかけると戦闘開始（任意）。無視して宝箱だけ回収可能
- **宝箱:** 封印の間クリア報酬（封印の結晶 + 8,000G）
- **レア宝箱:** 15%で深淵の結晶×3
- **シンボルボス:** 魔蝕のエルト（Lv200 / 全属性 / HP99999）
  - **スルー可能・逃走可能。** 本作最強の敵（FFオメガ枠）
  - 撃破報酬: 「エルトの形見」（アクセサリー。侵蝕許容値+15）+ 50,000G
  - 全属性を使う。弱点なし（全属性等倍）。自属性攻撃は吸収
  - フェーズ1（HP100%〜70%）: 単属性攻撃のローテーション
  - フェーズ2（HP70%〜30%）: 複合属性攻撃 + 「魔蝕」でレイの侵蝕強制上昇
  - フェーズ3（HP30%以下）: 「深淵解放」全能力上昇 + 「封印崩壊」全体9999ダメージ級
  - **攻略のポイント:** Lv200で挑むことを想定していない超裏ボス。NG+の最強装備でようやく戦える。2章初見では絶対に勝てない設計
