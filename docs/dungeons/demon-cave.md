# 鬼気の洞窟

## 基本情報

| 項目 | 内容 |
|------|------|
| ダンジョン名 | 鬼気の洞窟（きき のどうくつ） |
| 英名 | Demon Cave |
| 解放時期 | 3章〜 |
| 層数 | 18層 |
| 推奨Lv | 30〜55 |
| 属性傾向 | 闇・氷・地 |
| 特記 | 高難度。状態異常対策が必須 |

## 世界観・雰囲気

学園から北東に位置する山岳地帯の洞窟。闇の気配が極めて強く、かつて魔王の眷属が巣食っていた場所と伝えられる。現在も瘴気が充満し、洞窟内に棲む生物は全て変異している。

入口から既に冷気と瘴気が混在し、視界が悪い。洞窟壁面には鬼火のような青白い燐光が浮かび、氷柱と瘴気の結晶が交互に現れる。深部に進むほど闇は濃くなり、状態異常を引き起こす毒霧、麻痺ガス、催眠胞子が充満する。

**ストーリー上の位置づけ:** 3章でシオンがレイの真実を知り敵対を決意するきっかけとなる調査を行った場所。シオンの同期リオンが変異した場所でもある。

**雰囲気キーワード:** 瘴気の洞窟、鬼火、氷柱、毒霧、暗闇

## 各層の概要

### 1層: 洞窟入口

- **概要:** 冷たい風が吹き出す洞窟の入口。瘴気の気配を感じる
- **敵の傾向:** 洞窟コウモリ、毒蛇
- **ギミック:** なし。ただし帰還の石を使えない特殊エリア通知
- **宝箱:** ハイポーション×3（固定）、万能薬×2（固定）、解毒剤×5（固定）

### 2層: 毒霧の通路

- **概要:** 毒霧が漂う。一定時間ごとにHP減少
- **敵の傾向:** 毒蛇、瘴気ワーム、毒キノコ（洞窟種）
- **ギミック:** 毒霧エリア（滞在でHP減少。万能薬か風魔法で霧を払える）
- **宝箱:** 解毒剤×5（固定）
- **隠し通路:** 20%で壁面のひび → 万能薬×3

### 3層: 氷結区画

- **概要:** 氷柱が立ち並ぶ寒冷地帯。滑る床あり
- **敵の傾向:** アイスバット、氷結スライム、フロストワーム
- **ギミック:** 氷の床（滑って制御不能。壁にぶつかるとダメージ+敵出現）
- **宝箱:** エクスポーション×1（固定）
- **レア宝箱:** 15%で氷の結晶×3（錬金素材）

### 4層: 麻痺ガス地帯

- **概要:** 黄色い麻痺ガスが充満。麻痺状態異常のリスク
- **敵の傾向:** 麻痺キノコ、雷蛇、洞窟スパイダー
- **ギミック:** 麻痺ガスエリア（ランダムで麻痺付与。気付け薬か雷魔法で無効化）
- **宝箱:** 気付け薬×5（固定）
- **隠し部屋:** 10%で壁の裏 → 速さの種×2

### 5層: 鬼火の回廊

- **概要:** 青白い鬼火が浮遊する不気味な回廊
- **敵の傾向:** 鬼火、幽鬼兵、洞窟コウモリ（大）
- **ギミック:** 鬼火の誘導（正しい鬼火を追うと進める。偽の鬼火に釣られると敵出現）
- **宝箱:** 聖水×3（固定）、不死鳥の羽根×2（固定）
- **隠し通路:** 20%で鬼火の裏 → 英雄の薬×1

### 6層: 中間安全地帯

- **概要:** 瘴気が薄い区域。回復ポイント
- **敵の傾向:** 出現率低い（洞窟コウモリのみ）
- **ギミック:** 回復の泉 + セーブポイント
- **宝箱:** フルポーション×1（固定）、ハイマジックポーション×2（固定）
- **レア宝箱:** 15%で不死鳥の翼×1

### 7層: 催眠の花畑

- **概要:** 洞窟内に不気味な花が咲き乱れる。催眠効果あり
- **敵の傾向:** 催眠フラワー、マンドラゴラ（洞窟種）、ダークモス
- **ギミック:** 催眠エリア（滞在で睡眠付与。気付け薬やバフで予防）
- **宝箱:** 気付け薬×5（固定）、万能薬×2（固定）
- **隠し部屋:** 10%で花畑の奥 → 鋼の兜（頭防具。状態異常耐性+5%）

### 8層: 呪いの墓所

- **概要:** かつての冒険者たちの墓。呪いの瘴気が漂う
- **敵の傾向:** 呪いの骸骨、幽鬼兵（強）、呪術師の亡霊
- **ギミック:** 呪いエリア（呪い状態異常。聖水で解除。放置するとHP回復不可）
- **宝箱:** 聖水×5（固定）
- **隠し通路:** 20%で墓石の裏 → 鉄壁の鎧（上半身DEF+40）

### 9層: 捕食者の巣【中ボス】

- **概要:** 洞窟の主が棲む広大な空間。中ボス戦
- **敵の傾向:** 中ボスのみ
- **ギミック:** セーブポイント（中ボス前）
- **宝箱:** 中ボス撃破報酬（岩盾グラナイト[リン用盾DEF110] + 8,000G）
- **レア宝箱:** 15%で魔力の種×2
- **中ボス:** 洞窟竜ヴォルガ（Lv40 / 氷属性 / HP10000）
  - 洞窟に棲みついた若い竜。氷のブレスが脅威
  - 「氷結のブレス」全体氷大ダメージ+凍結（2ターン行動不能）
  - 「尾撃」前列物理大ダメージ+吹き飛ばし
  - 「咆哮」全体ATK/DEFデバフ
  - 火属性が弱点。凍結耐性装備を推奨

### 10層: 竜の宝物庫

- **概要:** ヴォルガが溜め込んだ宝物が散乱する部屋
- **敵の傾向:** ミミック、洞窟スパイダー（大）、宝石甲虫
- **ギミック:** ミミック判定（宝箱の50%がミミック。色の違いで見分け可能）
- **宝箱:** ハイポーション×5（固定）、帰還の石×2（固定）
- **隠し部屋:** 10%で壁面の亀裂 → ミスリル鉱×2（錬金素材）

### 11層: 瘴気の深層

- **概要:** 瘴気が最も濃い区域。全ての状態異常リスクあり
- **敵の傾向:** 瘴気の魔獣、ダークゴーレム、毒の精霊
- **ギミック:** 複合状態異常エリア（毒+暗闇+麻痺のランダム付与。万能薬必須）
- **宝箱:** 万能薬×5（固定）
- **隠し通路:** 20%で瘴気の薄い壁 → エクスポーション×3

### 12層: 暗黒回廊

- **概要:** 完全な暗闇。光源が必要
- **敵の傾向:** 闇の捕食者、暗黒スライム、シャドウバット
- **ギミック:** 暗闇探索（松明か光魔法が必要。なしでは移動速度低下+先制攻撃されやすい）
- **宝箱:** 砕岩の大剣（カイト用非売品武器ATK105）
- **レア宝箱:** 15%で精霊石×1（錬金素材）

### 13層: 変異の回廊

- **概要:** 瘴気で変異した生物が闊歩するエリア
- **敵の傾向:** 変異コウモリ、変異ワーム、変異スパイダー
- **ギミック:** 変異トラップ（触れるとランダムでデバフ付与）
- **宝箱:** フルポーション×1（固定）、聖水×3（固定）
- **隠し部屋:** 10%で変異した壁 → 暗黒の呪書（ヴェイン用呪具MATK100）

### 14層: 結晶の間

- **概要:** 瘴気の結晶が壁一面に広がる美しくも恐ろしい空間
- **敵の傾向:** 結晶ゴーレム、瘴気の結晶体、闇の精霊（上位）
- **ギミック:** 結晶パズル（正しい順序で結晶を砕くと道が開く。間違うと敵出現）
- **宝箱:** 英雄の薬×2（固定）
- **隠し通路:** 20%で結晶の裏 → 深淵の結晶×1（高級錬金素材）

### 15層: 封印崩壊地帯

- **概要:** 封印の力が届かない完全な闇の領域
- **敵の傾向:** 深淵の眷属（洞窟種）、瘴気の王蛇
- **ギミック:** レイの侵蝕ゲージが上昇するエリア（+10）
- **宝箱:** 不死鳥の翼×2（固定）
- **レア宝箱:** 15%で ミスリルローブ（上半身MDEF+45）

### 16層: 鬼神の前室

- **概要:** 最終ボスへの準備エリア
- **敵の傾向:** 結晶ゴーレム（強）、深淵の眷属（洞窟種）
- **ギミック:** 回復の泉 + セーブポイント
- **宝箱:** フルポーション×3（固定）、万能薬×5（固定）
- **隠し部屋:** 10%で壁の裏 → 竜鱗のコート素材（竜の鱗×3）

### 17層: 鬼神の回廊

- **概要:** ボスへの最後の通路。圧倒的な瘴気
- **敵の傾向:** 全種の上位版がランダム出現
- **ギミック:** なし（純粋な戦闘層）
- **宝箱:** エリクサー×1（固定）
- **隠し通路:** 20%で壁のひび → 不死鳥の翼×1

### 18層: 鬼神の間【ボス】

- **概要:** 洞窟最深部。瘴気の源泉
- **敵の傾向:** ボスのみ
- **ギミック:** セーブポイント（ボス前）
- **宝箱:** ボス撃破報酬（鬼神の角[素材] + 15,000G + エクスポーション×5）
- **レア宝箱:** 15%で追加報酬 → 鬼神の腕輪（アクセ。ATK+10%、状態異常耐性+15%）
- **ボス:** 鬼神ラグナロク（Lv50 / 闇属性 / HP18000）
  - 瘴気が凝縮して生まれた鬼神。元は魔王の眷属の一体
  - フェーズ1（HP100%〜60%）: 物理攻撃メイン。「鬼気の拳」単体大ダメージ + 「瘴気の波動」全体毒+暗闇
  - フェーズ2（HP60%〜30%）: 「鬼神化」ATK/AGI大幅上昇。「百鬼夜行」全体闇大ダメージ+混乱
  - フェーズ3（HP30%以下）: 「深淵の咆哮」全員の状態異常耐性を無視して毒+呪い付与。「滅殺」単体即死級ダメージ
  - 光属性が弱点。状態異常耐性をできるだけ積んで挑むこと
  - 全状態異常を使う本作屈指の難敵
