## 世界観・雰囲気

黒蝕の教団（エクリプス）が拠点とする、学園から遠く離れた山中の要塞。表向きは廃棄された鉱山だが、地下には教団が建造した巨大な施設が広がる。

鉄と石で造られた冷たい通路。壁面には教団のシンボル「蝕まれた太陽」が刻まれ、禁術の魔法陣が赤黒い光を放つ。教団兵が巡回し、魔導兵器が配備され、変異させられた魔獣が鎖に繋がれている。

深層部はカルロスの私室と研究施設。魔王の力を抽出する装置や、禁術の実験設備が並ぶ。カルロスの知性と狂気が結実した空間。

**ストーリー上の重要性:** END5ルートの最終ダンジョン。シオンを救出した後、5人（レイ・ユナ・ミオ・カイト・シオン）でカルロスの拠点に乗り込む。

**雰囲気キーワード:** 鉄の要塞、教団のシンボル、禁術、魔導兵器、カルロスの狂気

## 各層の概要

### 1層: 要塞入口

- **概要:** 廃鉱山の入口を偽装した教団の正門。守衛を突破して侵入
- **敵の傾向:** 教団兵Lv1、教団偵察兵
- **ギミック:** 警報システム（見つかると増援が来る。ステルス移動可能）
- **宝箱:** エクスポーション×3（固定）、万能薬×3（固定）

### 2層: 兵舎区画

- **概要:** 教団兵の生活区域。食堂や武器庫がある
- **敵の傾向:** 教団兵Lv1、教団兵Lv2、教団魔導師
- **ギミック:** 武器庫の扉（ロックピック or 鍵入手で開錠 → 装備入手）
- **宝箱:** ミスリルの剣（レイ用武器ATK150）
- **隠し通路:** 20%で武器庫の裏 → 英雄の薬×2

### 3層: 魔獣飼育場

- **概要:** 教団が変異させた魔獣を飼育する区域。鎖で繋がれた魔獣が暴れる
- **敵の傾向:** 変異魔獣Lv1、教団獣使い、狂化ウルフ
- **ギミック:** 魔獣の鎖（壊すと魔獣が解放されて襲ってくるが、倒すとレアドロップ）
- **宝箱:** 不死鳥の翼×1（固定）
- **レア宝箱:** 15%で竜の鱗×3（錬金素材）

### 4層: 訓練施設

- **概要:** 教団兵の訓練場。戦闘用ゴーレムが配備
- **敵の傾向:** 教団兵Lv2、教団エリート兵、魔導兵器Lv1
- **ギミック:** 訓練用ゴーレムの制御装置（ハッキングすると味方になる）
- **宝箱:** フルポーション×2（固定）
- **隠し部屋:** 10%で教団の機密書類 → 情報フラグ関連イベント

### 5層: 中間安全地帯

- **概要:** 教団の物資倉庫。敵の巡回が少ない
- **敵の傾向:** 教団偵察兵のみ（出現率低い）
- **ギミック:** 回復ポイント（物資を使ってHP/MP回復）+ セーブポイント
- **宝箱:** エリクサー×1（固定）、ハイマジックポーション×5（固定）
- **隠し通路:** 20%で倉庫の裏 → 教団の秘宝（アクセ。DEF+10%）

### 6層: 魔導兵器工場

- **概要:** 教団が魔導兵器を製造する工場。稼働中の機械が脅威
- **敵の傾向:** 魔導兵器Lv1、魔導兵器Lv2、教団技師
- **ギミック:** 稼働中の機械（触れるとダメージ。スイッチで停止可能）
- **宝箱:** オリハルコン×1（固定・錬金素材）
- **レア宝箱:** 15%で魔導兵器のコア（売値10,000G）

### 7層: 禁術研究室

- **概要:** カルロスの禁術研究が行われた場所。不気味な実験の跡
- **敵の傾向:** 実験体α、実験体β、教団魔導師（上位）
- **ギミック:** 研究ノート（カルロスの計画の詳細がわかるイベント）
- **宝箱:** 深淵の結晶×2（固定）
- **隠し通路:** 20%で研究室の裏 → 禁術の書（ヴェイン用呪具MATK130）
- **隠し部屋:** 10%で秘密の実験室 → 実験体のデータ（ロアアイテム）

### 8層: 教団幹部の間【中ボス】

- **概要:** 教団幹部が待ち受ける大広間
- **敵の傾向:** 中ボスのみ
- **ギミック:** セーブポイント + 回復ポイント（中ボス前）
- **宝箱:** 中ボス撃破報酬（教団幹部の紋章 + 20,000G）
- **中ボス:** 教団幹部ヴァルザス（Lv65 / 闇属性 / HP18000）
  - カルロスの右腕。禁術の使い手
  - 「闇の刃」単体闇大ダメージ + 「呪縛の鎖」ランダム1体を2ターン行動不能
  - 「禁術・生命吸収」攻撃しつつHP回復
  - 「闇の結界」3ターン物理ダメージ半減
  - 光属性が弱点。呪縛対策に万能薬を用意

### 9層: 投獄区画

- **概要:** 教団に囚われた人々の牢獄。救出イベントあり
- **敵の傾向:** 教団看守、魔導兵器Lv2、変異魔獣Lv2
- **ギミック:** 牢獄の解錠（囚人を助けるとサブイベント+報酬）
- **宝箱:** フルポーション×3（固定）
- **レア宝箱:** 15%で英雄の脚甲（下半身。SPD+10%, ATK+8%）

### 10層: 教団神殿

- **概要:** 教団が魔王を崇拝する神殿。不気味な祭壇がある
- **敵の傾向:** 教団司祭、闇の祭壇兵、変異魔獣Lv2
- **ギミック:** 祭壇の破壊（教団の力の源。壊すと以降の敵が弱体化）
- **宝箱:** 聖水×10（固定）、エリクサー×1（固定）
- **隠し通路:** 20%で神殿の裏 → 神木の枝×2（最上位錬金素材）

### 11層: カルロスの書斎

- **概要:** カルロスの私的な書斎。蔵書が並ぶ
- **敵の傾向:** 教団エリート兵、教団魔導師（上位）
- **ギミック:** カルロスの日記（カルロスの過去と動機がわかるイベント）
- **宝箱:** 深淵の結晶×3（固定）
- **隠し部屋:** 10%で隠し金庫 → 50,000G

### 12層: 魔王の力抽出装置

- **概要:** カルロスが魔王の力を抽出するために作った巨大装置
- **敵の傾向:** 魔導兵器Lv3、実験体γ、教団精鋭兵
- **ギミック:** 装置の破壊（3箇所の制御装置を壊すと装置停止。経験値ボーナス）
- **宝箱:** オリハルコン×2（固定）
- **レア宝箱:** 15%でオリハルコンの脚甲（下半身DEF+50）

### 13層: 精鋭の間

- **概要:** カルロス直属の精鋭部隊が守る区域
- **敵の傾向:** 教団精鋭兵、教団暗殺者、魔導兵器Lv3
- **ギミック:** 連戦エリア（3連戦。途中回復なし）
- **宝箱:** エリクサー×2（固定）
- **隠し通路:** 20%で精鋭の間の裏 → 不死鳥の翼×2

### 14層: 最終安全地帯

- **概要:** カルロス戦前の最終準備
- **敵の傾向:** なし
- **ギミック:** 回復の泉 + セーブポイント。仲間との会話イベント（決意のシーン）
- **宝箱:** フルポーション×5（固定）、エリクサー×2（固定）、万能薬×10（固定）
- **隠し部屋:** 10%で隠し武器庫 → オリハルコンの大剣（カイト用ATK280）

### 15層: カルロスの玉座【ボス】

- **概要:** 教団の最深部。カルロスが待ち受ける
- **敵の傾向:** ボスのみ
- **ギミック:** セーブポイント（ボス前）。カルロスとの対話イベント
- **宝箱:** ボス撃破報酬（教団の残骸 + 50,000G + エリクサー×5）
- **レア宝箱:** 15%で追加報酬 → カルロスの魔導杖（素材。売却で100,000G）
- **ボス:** カルロス（Lv100 / 闇属性 / HP30000）
  - END5のラスボス。3フェーズの長期戦
  - **フェーズ1「策略家」**（HP100%〜60%）:
    - 「禁術・闇の渦」全体闇大ダメージ
    - 「回復阻害の呪い」3ターン回復量半減
    - 「デバフ連鎖」全員にATK/DEF/AGIダウン
    - 知性的に弱点を突いてくる。デバフ対策が必要
  - **フェーズ2「操り師」**（HP60%〜30%）:
    - ストーリーでは操られたシオンと2対5だが、END5ではシオン救出済み
    - 代わりに魔導兵器2体を召喚。カルロス+兵器2の3対5
    - 兵器を先に壊さないとカルロスが「修復」で兵器を復活させる
  - **フェーズ3「狂気」**（HP30%以下）:
    - 「禁術・自己強化」全ステ大幅上昇
    - 「深淵の一撃」単体即死級ダメージ（防御かHP管理で対処）
    - 「最後の手段」HP10%以下で全体即死級攻撃（1回のみ）
    - 光属性が弱点。レイ×シオンのデュアルアーツ「星影双斬」が有効
