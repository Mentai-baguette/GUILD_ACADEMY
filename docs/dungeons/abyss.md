## 世界観・雰囲気

黒蝕の教団（エクリプス）の真の本拠地。カルロスの拠点はあくまで前線基地に過ぎず、教団の本体はこの「深淵」に存在する。封印が最も薄い場所に建造された、現実と夢の境界にある異質な空間。

壁、床、天井という概念が曖昧で、空間そのものが歪んでいる。あるフロアでは逆さまの都市が広がり、あるフロアでは星空の海に浮島が漂う。現実の物理法則が通用せず、敵もまた異形の存在——人とも獣とも言えない、深淵から生まれた存在たち。

50層最深部には教団教主モルドレッドが待つ。魔王の魂の一部を宿す彼は、300年間この深淵で教団を率い、魔王完全復活の時を待っていた。

**雰囲気キーワード:** 歪んだ空間、逆さまの世界、星空の海、現実と夢の境界、異形

## 構造概要（10層ごとのブロック）

### 1〜10層: 歪みの入口ブロック

- **環境:** 逆さまの回廊。重力が不安定
- **敵Lv帯:** 80〜95
- **敵の傾向:** 深淵スライム系、歪みの兵士、虚空の魔獣
- **ギミック:** 重力反転（天井と床が入れ替わるエリア）、空間ループ（正しい出口を見つける）
- **宝箱:** エリクサー、フルポーション、深淵の結晶
- **5層:** 安全地帯（回復+セーブ）
- **10層【中ボス】:** 歪みの守護者（Lv90 / 闇属性 / HP40000）
  - 空間を歪めて攻撃を回避する。ブレイクで歪み解除
  - 「空間断裂」単体即死級ダメージ
  - 「次元圧縮」全体闇大ダメージ+DEFダウン

### 11〜20層: 夢幻の回廊ブロック

- **環境:** 夢のような幻想空間。美しいが危険
- **敵Lv帯:** 95〜110
- **敵の傾向:** 夢幻の使い魔、幻影の騎士、深淵の花
- **ギミック:** 幻覚エリア（偽の道や偽の宝箱が出現。真偽を見極める）
- **宝箱:** NG+限定素材、レア錬金レシピ
- **15層:** 安全地帯（回復+セーブ）
- **20層【中ボス】:** 夢幻の支配者（Lv105 / 闇属性 / HP55000）
  - 幻覚で味方を混乱させる。「幻影召喚」で自分のコピーを3体召喚
  - 本体を見極めないと無駄にターンを消費する
  - 「永遠の眠り」全体睡眠（万能薬で解除）

### 21〜30層: 深淵の都市ブロック

- **環境:** かつて栄えた文明の残骸。廃墟の都市
- **敵Lv帯:** 110〜125
- **敵の傾向:** 深淵の市民（変異体）、鉄屑のゴーレム、闇の学者
- **ギミック:** 都市探索（廃墟の中を探索。隠しアイテム多数）
- **宝箱:** 超レア素材、NG+限定装備
- **25層:** 安全地帯（回復+セーブ）
- **30層【中ボス】:** 深淵の将軍（Lv120 / 闇属性 / HP70000）
  - 教団の古参幹部。モルドレッドへの忠誠心が異常
  - 「教団の号令」で雑魚3体を召喚+全体バフ
  - 「滅殺の剣」単体超大ダメージ。防御かHP管理必須
  - 光属性が弱点

### 31〜40層: 教団中枢ブロック

- **環境:** 教団の真の神殿。禁術の気配が充満
- **敵Lv帯:** 125〜140
- **敵の傾向:** 教団司祭（上位）、禁術の産物、深淵の守護者
- **ギミック:** 禁術トラップ（触れると全体デバフ+HP減少。浄化で無効化）
- **宝箱:** 深淵の王冠の素材、超高額G
- **35層:** 安全地帯（回復+セーブ）
- **40層【中ボス】:** 教団副教主アザゼル（Lv135 / 闇属性 / HP85000）
  - モルドレッドに次ぐ教団No.2
  - 「禁術・魂喰い」攻撃+MP吸収
  - 「闇の審判」全体闇超大ダメージ+ランダムデバフ
  - 「不死の呪い」自身にHP回復（毎ターン5000回復。ブレイクで解除）
  - 光属性が弱点

### 41〜49層: 深淵の最深部ブロック

- **環境:** 現実が完全に崩壊した空間。虚無が広がる
- **敵Lv帯:** 140〜150
- **敵の傾向:** 深淵の化身、虚無の騎士、終焉の使者
- **ギミック:** 虚無侵蝕（全員のHPが毎ステップ微減。回復手段の管理が鍵）
- **宝箱:** 最高ティア素材、NG+限定最強装備の材料
- **45層:** 安全地帯（最終回復+セーブ）

### 50層: 教主の間【ボス: モルドレッド】

- **環境:** 深淵の最奥。巨大な魔法陣の上にモルドレッドが佇む
- **敵の傾向:** ボスのみ
- **ギミック:** セーブポイント（ボス前）
- **宝箱:** ボス撃破報酬（深淵の王冠[頭装備。全ステ+10%, 侵蝕時ATK+15%] + 100,000G）
- **ボス:** 教団教主モルドレッド（Lv150 / 闇属性 / HP120000）
  - 魔王の魂の一部を宿す教団の真の指導者。300年生き続けた異形の存在
  - **フェーズ1「教主」**（HP100%〜70%）:
    - 「禁術・暗黒連鎖」全体闇大ダメージ+全員にランダム状態異常
    - 「魂の縛鎖」1体のATBゲージを0+3ターン蓄積速度半減
    - 「闇の叡智」自身全ステバフ
  - **フェーズ2「魔王の器」**（HP70%〜30%）:
    - 「魔王覚醒」ATK/INT大幅上昇。外見が変化
    - 「深淵の津波」全体全属性ダメージ
    - 「支配の瞳」ランダム1体を3ターン操る（味方を攻撃させる）
    - 「闇の再生」毎ターンHP3000回復
  - **フェーズ3「深淵そのもの」**（HP30%以下）:
    - 「深淵解放」全能力再上昇。闇のオーラ
    - 「終焉の刻」全体即死級ダメージ（HP1で耐えるアクセが必須級）
    - 「魔王の咆哮」全員の状態異常耐性を0にして全状態異常
    - 「最後の禁術」HP5%以下で全体99999ダメージ（防御+HP管理で耐えること）
  - 光属性が弱点。全員Lv150近くまで育てて挑む設計
