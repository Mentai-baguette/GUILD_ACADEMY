## 世界観・雰囲気

クロノス・アカデミーから徒歩30分ほどの位置にある深い森。1年生のダンジョン実習として使われる実戦フィールド。

入口付近は木漏れ日が差し込み、小鳥のさえずりが聞こえる穏やかな雰囲気。しかし奥に進むにつれて木々は高く密集し、光が届かなくなっていく。6層を越えると「深層域」と呼ばれ、闇の気配が漂い始める。10層以降は瘴気が立ち込め、植物が変色し、不気味な花が咲く異常地帯。

**ストーリー上の重要性:** 1章でレイの覚醒が起きた場所。黒い炎が森の一角を焼いた痕跡が残っている（6層付近で確認可能）。

**雰囲気キーワード:**
- 1〜4層: 穏やかな森、木漏れ日、小川のせせらぎ
- 5〜8層: 薄暗い深い森、霧、巨木
- 9〜12層: 瘴気漂う異常地帯、変色した植物、不気味な光

## 各層の概要

### 1層: 森の入口

- **概要:** 実習開始地点。木漏れ日が差し込む明るい森
- **敵の傾向:** グリーンスライム、フォレストラット
- **ギミック:** なし（安全な導入）
- **宝箱:** ポーション×2（固定）、解毒剤×1（固定）

### 2層: 小川の道

- **概要:** 小川に沿って進む。水辺の敵が出現
- **敵の傾向:** グリーンスライム、フォレストラット、毒キノコ
- **ギミック:** 川の流れ（特定ポイントで渡渉。失敗するとスタート地点に戻る）
- **宝箱:** マジックポーション×1（固定）
- **隠し通路:** 20%で川の上流方向 → 薬草×3

### 3層: 大樹の広場

- **概要:** 巨大な古木がある広場。休憩ポイントあり
- **敵の傾向:** ツタの魔獣、ポイズンフラワー
- **ギミック:** 大樹に登ると周囲を見渡せる（次層のマップヒント）
- **宝箱:** 帰還の石×1（固定）
- **レア宝箱:** 15%で力の種×1

### 4層: 獣の縄張り

- **概要:** 獣系モンスターが多い区域。群れで出現する
- **敵の傾向:** フォレストウルフ、フォレストボア
- **ギミック:** 獣の足跡を辿ると宝箱の場所がわかる
- **宝箱:** ハイポーション×1（固定）
- **隠し通路:** 20%で木の根の下 → 守りの種×1

### 5層: 霧の森

- **概要:** 霧が立ち込め視界が悪い。ここから難易度上昇
- **敵の傾向:** フォレストウルフ、モスゴーレム、ウィスプ
- **ギミック:** 霧のナビゲーション（正しい方向に進まないとループする。光る苔が目印）
- **宝箱:** 万能薬×1（固定）
- **隠し部屋:** 10%で霧の奥 → 鉄の兜（頭防具）

### 6層: 覚醒の痕跡【中ボス】

- **概要:** レイの覚醒で焼けた森の跡地。中ボスが棲みつく
- **敵の傾向:** エンシェントトレント（幼体）、ポイズンフラワー
- **ギミック:** 焼け跡を調べると回想イベント（レイの覚醒シーン）
- **宝箱:** マジックポーション×2（固定）
- **レア宝箱:** 15%で不死鳥の羽根×1
- **中ボス:** エンシェントトレント（Lv12 / 地属性 / HP1200）
  - 「根の束縛」で1体を行動不能にする
  - 火属性が弱点。風属性は吸収するので注意
  - 「再生」でHPを回復する。ブレイクで再生を封じるのが鍵

### 7層: 深層入口

- **概要:** ここから深層域。空気が変わり、光が少なくなる
- **敵の傾向:** モスゴーレム、ダークウルフ、ウィスプ
- **ギミック:** 光る花を集めると暗闇エリアを照らせる（3つ必要）
- **宝箱:** 帰還の石×1（固定）、ポーション×3（固定）
- **隠し通路:** 20%で枯れ木の裏 → 速さの種×1

### 8層: 妖精の泉

- **概要:** 深層にある神秘的な泉。回復ポイント
- **敵の傾向:** ダークウルフ、フォレストスプライト、毒蔦の魔獣
- **ギミック:** 泉で休むとHP/MP全回復（1回限り）。泉の水を汲むとポーション効果
- **宝箱:** ハイポーション×2（固定）
- **レア宝箱:** 15%で魔力の種×1
- **隠し部屋:** 10%で泉の裏 → 薬草師のレシピ（錬金術用・3章以降使用可能）

### 9層: 瘴気の境界

- **概要:** 瘴気が立ち込め始める。毒・暗闇の状態異常が増える
- **敵の傾向:** 毒蔦の魔獣、ダークフラワー、フォレストスプライト（闇）
- **ギミック:** 瘴気エリア（滞在するとHP減少。解毒剤か浄化スキルで一時無効化）
- **宝箱:** 万能薬×2（固定）、聖水×1（固定）
- **隠し通路:** 20%で変色した木の幹 → ハイポーション×3

### 10層: 変異の園

- **概要:** 植物が完全に変色し、不気味な花が咲き乱れる
- **敵の傾向:** ダークフラワー、瘴気スライム、マンドラゴラ
- **ギミック:** 花のパズル（3色の花を正しい順序で触れると道が開く）
- **宝箱:** 不死鳥の羽根×1（固定）
- **レア宝箱:** 15%で黒鉄の剣（レイ用非売品武器）

### 11層: 古代の祭壇

- **概要:** 森の奥深くにある謎の祭壇。かつて何かの儀式が行われた場所
- **敵の傾向:** マンドラゴラ、エルダートレント、闇の妖精
- **ギミック:** 祭壇を調べると情報フラグのヒント（学園の秘密に関する古文書の断片）
- **宝箱:** 英雄の薬×1（固定）
- **隠し通路:** 20%で祭壇の裏 → 精霊の腕輪の欠片（素材）
- **隠し部屋:** 10%で地下への階段 → 古い魔導書（ミオ用武器上位）

### 12層: 森の心臓【ボス】

- **概要:** 翠緑の森の最深部。巨大な古代樹が根を張る空間
- **敵の傾向:** ボスのみ
- **ギミック:** ボス前にセーブポイント+回復の泉
- **宝箱:** ボス撃破報酬（翠緑の結晶 + 2,000G + ハイポーション×5）
- **レア宝箱:** 15%で追加報酬 → 鋼の脛当て（下半身防具）
- **ボス:** 翠緑の守護者（Lv18 / 地属性 / HP3000）
  - 森の意志が具現化した巨大な樹精
  - フェーズ1（HP100%〜60%）: 根の攻撃+葉の刃（物理+風）
  - フェーズ2（HP60%〜30%）: 「大地の怒り」全体地属性攻撃。「瘴気散布」で毒付与
  - フェーズ3（HP30%以下）: 「森の咆哮」ATK/INT大幅上昇。猛攻開始
  - 火属性が弱点。ブレイク中に火スキルで大ダメージ
