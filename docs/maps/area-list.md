## 2. 学園外エリア

学園外は章の進行に応じて段階的に解放される。

### 2-1. 学園門前

| 項目 | 内容 |
|------|------|
| 解放時期 | 1章2週目 |
| 主な機能 | 学園と街の中継地点 |
| BGM | bgm_academy → bgm_town へ切り替わるクロスフェードポイント |

**配置NPC:**
- 門番NPC: 外出許可の確認（1章1週目は「まだ外出は早い」と止められる）
- モブ生徒: 街から帰ってきた生徒の雑談

**雰囲気:** 大きな石造りの門。学園のシンボルが刻まれている。門の先に街並みが見える。

## 3. 移動システム

### 3-1. 徒歩移動

- 初回訪問時は必ず徒歩で移動する（イベントシーン付き）
- マップ間移動はエリア選択式（フィールドをシームレスに歩くのではなく、出口に到達→行先選択）

### 3-2. ファストトラベル

| 項目 | 仕様 |
|------|------|
| 消費アイテム | 転送の石（1回1個消費） |
| 使用可能場所 | フィールド上のみ（ダンジョン内不可） |
| 転送先 | 一度訪れたことのあるエリアの転送ポータル |
| 制限 | 4章カルロスの拠点突入後は学園への帰還不可 |

### 3-3. 転送ポータル

- 学園内: 寮入口、訓練場入口、正門前の3箇所
- ダンジョン内: 各ダンジョンの中間地点（休憩ポイント）に1〜2箇所
- 一度触れたポータルはファストトラベルの転送先として登録される
- ポータルは青い光の柱として視覚的に表示

## 4. モブNPC反応パターン

### 4-1. 章ごとの反応変化

| 時期 | モブNPCの傾向 |
|------|-------------|
| 1章序盤 | 新入生への好奇心、歓迎ムード |
| 1章中盤〜 | レイの力に対する噂が出始める |
| 2章 | シオンとの対立を受けて不安の声。レイを避ける生徒も |
| 3章 | 学園内に緊張が走る。「何かが起きる」という雰囲気 |
| 4章 | 学園が危機的状況。覚悟を決めた生徒たちの激励 |

### 4-2. 2年生モブの特殊反応

- 2年生は1年生より情報を持っている設定
- 「去年も似たようなことがあったんだ...」→ 学園の過去を示唆
- 特定のフラグを立てた後に話しかけると追加情報を教えてくれる
- 2年生の中にカルロス派（隠れ教団員）が数名いる → 3章で判明
